厉嫣然坐在副驾驶座椅上，她当然也看到豪车开出来后陆寒霆还在看着后面渐行渐远的夏夕绾，她气的咬了一下唇，“寒霆哥哥，今天我去看望一下陆奶奶吧，我学的营养师，正好根据陆奶奶的身体制订出一套餐谱。”
<br/>
<br/>        厉嫣然知道陆寒霆最爱自己的奶奶了，她打算从陆奶奶这里入手，先获得陆奶奶的喜爱。
<br/>
<br/>        厉嫣然对自己的营养师专业还是很有信心的，她一定能将陆奶奶的胃口和身体调理好的，到时她就能离陆寒霆更进一步了。
<br/>
<br/>        陆寒霆已经看不到后面的夏夕绾了，他收回目光，点了一下头，“奶奶的胃口越来越不好，我是打算请一个专业的营养师来照顾奶奶的，我先给奶奶打个电话，如果她不排斥的话我就带你过去。”
<br/>
<br/>        “好。”厉嫣然乖巧的笑道。
<br/>
<br/>        陆寒霆将电话拨了出去，悠扬的手机铃声响了一遍然后被接起了，陆老夫人年迈的声音传来，“喂，阿霆。”
<br/>
<br/>        “奶奶，你今天胃口如何，嫣然想去看看你，给你定制一套食谱，我们现在就过去。”
<br/>
<br/>        那端的陆老夫人顿了一下，突然问，“阿霆，你现在跟嫣然在一起？”
<br/>
<br/>        厉嫣然迅速开口道，“是的陆奶奶，今天放学是寒霆哥哥来接我的哦，我们现在就过去看你。”
<br/>
<br/>        那端的陆老夫人默了一下，拒绝道，“你们都不用过来了，我今天胃口很好，就是犯困，先睡了，没事的话我挂电话了。”
<br/>
<br/>        陆老夫人直接将电话给挂断了。
<br/>
<br/>        厉嫣然满腔的热情瞬间被扑灭，傻子都能听出来老夫人对她并不亲昵，她委屈的嘟了一下红唇，“寒霆哥哥，我是不是做错什么事情了啊，我总觉得陆奶奶去了海城七年之后就不太喜欢我了。”
<br/>
<br/>        陆寒霆深邃的狭眸里快速的闪过了什么，奶奶真的很反常了，奶奶刚才把电话给挂了是因为他跟厉嫣然在一起。
<br/>
<br/>        如果没有记错的话，奶奶最大的心愿就是他能娶妻生子，给她抱个大重孙子，按理说厉嫣然出现在他的身边，奶奶欢喜都来不及，怎么会是这么冷淡又推拒的态度？
<br/>
<br/>        陆寒霆目不斜视的看着前方的路，嗓音低沉磁性道，“不用多想，改日再看奶奶吧，我送你回家。”
<br/>
<br/>        送她回家？
<br/>
<br/>        难道…晚上没有安排吗？
<br/>
<br/>        “寒霆哥哥，我们一起去吃晚餐吧？”厉嫣然期待的看着他，最好是浪漫的烛光晚餐。
<br/>
<br/>        “晚上我没有时间，已经有约了。”陆寒霆给拒绝了。
<br/>
<br/>        厉嫣然好失望，“那好吧，那我们改日再约。”
<br/>
<br/>        ……
<br/>
<br/>        陆寒霆将厉嫣然送回了家，然后他带着小圆圆驾车去了帝吧，他真的有约了，顾夜瑾从海城来到了帝都。
<br/>
<br/>        豪华包厢里，陆寒霆坐在了暗红色的沙发上，他看了一眼身边的顾夜瑾，“怎么来帝都了？”
<br/>
<br/>        多日不见的顾夜瑾依然俊美如玉，脸上的金丝眼镜折射出他一双清寒冷漠的黑眸，“来这里开个会，过两天就回去，你怎么样了？”
