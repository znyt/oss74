　　“你们所有子弟都在这里了，那其他小型矿脉怎么办？难道你们慕容家就只有两座小型矿脉？”
<br/>　　萧逸疑惑地看向慕容轩。
<br/>　　慕容轩老实地回答道，“当然不是，我们也有十座小型矿脉，不过另外八座聘请了猎妖师帮忙开采，只留下一个子弟看守就行了。”
<br/>　　“原来如此。”萧逸冷声道，“这样你们就能有人手来抢夺我们萧家的小型矿脉了吧。”
<br/>　　这个世界所谓的猎妖师，在萧逸看来，性质大概跟前世地球的雇佣兵差不多，雇主只要给得起钱，就能让他们帮忙做任何事。
<br/>　　“额...那个...”慕容轩支支吾吾，生怕会惹怒萧逸。
<br/>　　恰在此时，忽然，一道人影疾速袭来，并一把抱走了慕容轩。
<br/>　　同一时间，十数道人影忽然出现，挡在了慕容家子弟前面。
<br/>　　萧逸目光一凛，大喝道，“退。”
<br/>　　声音刚落，萧壮萧子木等人也算反应快，立刻后退，来到萧逸的身后。
<br/>　　这突然出现的十数道人影也没有追击萧壮等人，只是单纯地为慕容家子弟解绑。
<br/>　　嗤、嗤、嗤...一道道剑光闪过，绳子应声断裂，慕容家子弟恢复了自由。
<br/>　　这忽然出现的十数人，个个都手持兵刃，脸色冷酷，显然非一般的武者。
<br/>　　“你们是猎妖师？”萧逸冷冷地看着这十数人。
<br/>　　“不错。”刚才出现将慕容轩抱走的人影，替慕容轩解绑后，上前一步，来到萧逸面前。
<br/>　　“萧家少家主，在下铁刀猎妖队副队长王源，多有得罪了。”那人对着萧逸拱了拱手。
<br/>　　萧逸打量了此人一眼，年龄约莫三十，脸上有一狰狞刀疤，气息沉稳，显然非泛泛之辈。
<br/>　　“你想在我眼皮子底下将他们带走？”萧逸淡淡地看着王源。
<br/>　　王源脸色一沉，沉声道，“萧家少家主，我们只是收人钱财，替人消灾罢了，请不要为难我们。”
<br/>　　如果是别人，王源肯定拔刀相向，或者直接带着慕容家子弟离开。可是，当他看着萧逸时，却明显感觉到一股压力，让他不敢造次。
<br/>　　萧逸沉声道，“打伤了我们萧家这么多子弟，你想让我就这么放他们离开？”
<br/>　　“这...”王源思索了一下，忽然一把扯过慕容轩身上的中级乾坤袋，而后扔给萧逸。
<br/>　　“萧家少家主，这当是给你们的赔礼，可行？”王源道。
<br/>　　萧逸看了眼乾坤袋里面的东西，点了点头，对萧壮等人道，“我们走。”
<br/>　　转身走了没几步，萧逸忽然停下了脚步，淡淡地回过头，道，“慕容家的，听清楚了，以后再敢抢夺萧家矿脉，杀我萧家子弟，我见一个杀一个。”
<br/>　　“还有你们，铁刀猎妖队是吧。你们护慕容家子弟一次也就罢了，再有下次，再惹到我，我会让你们直接消失。”
<br/>　　从始至终，萧逸都是语气平淡，但那平淡中，却丝毫没有掩饰浓郁的杀意，反倒更让王源等人心头一凛。
<br/>　　.....
<br/>　　“王源，你干嘛拿我的乾坤袋给那小废物。以你后天六重的实力，加上其他猎妖师，足以将他击毙，再不济也能带着我们安然离开。”
<br/>　　慕容轩见萧逸等人已经走远，这才愤怒地质问王源。
<br/>　　而王源，则久久地注视着萧家子弟离开，确切来说，是注视着萧逸的背影。
<br/>　　“我问你话呢，你发什么呆。”慕容轩愤怒地说道。
<br/>　　王源没有回答，当萧家众人已经彻底消失在眼前时，他才忽然松了口气。
<br/>　　“那个少年，很恐怖。”王源道。
<br/>　　“你什么意思？”慕容轩一愣，问道。
<br/>　　王源摇了摇头，道，“他给我很大的压力，这股压力，无关修为，而是像....”
<br/>　　“像什么？”慕容轩见王源脸色沉重，不禁问道。
<br/>　　王源沉声道，“像一个刽子手，一个杀人无数，双手沾满了鲜血的刽子手。”
<br/>　　慕容轩嗤笑道，“王源你开什么玩笑，萧逸不过是个少年，哪能杀人无数。”
<br/>　　“我的感觉不会错。”王源的脸色变得更加沉重，道，“我常年游走于陨星山脉，经历过无数危险，多次在妖兽口中、各处险地逃生，早已养成了敏锐的直觉。”
<br/>　　“总之，你听我的不会错。”王源认真地对慕容轩说道，“不要再去惹萧家的子弟，特别是萧逸，否则，我们铁刀猎妖队会立刻与你们解除雇佣关系。”
<br/>　　.....
<br/>　　另一边，萧逸带着萧壮等人已经走远。
<br/>　　萧壮愤愤不平地说道，“萧逸，刚才真不应该就这样放过慕容家的子弟，他们太过分了，我还想好好教训他们呢。”
<br/>　　萧逸淡淡道，“没办法，那支猎妖队不是善茬。而且，慕容家的百多号子弟已经被解绑，虽然他们身负伤势，但联合起来，是一股不弱的战力。”
<br/>　　萧逸见萧壮等人还是满肚子怨气，笑道，“你们知道这乾坤袋里面的东西后，估计就不会生气了。”
<br/>　　“哦？”萧壮和萧子木微微疑惑。
<br/>　　萧逸打开乾坤袋，顿时一股紫气冲天而出。
<br/>　　“是紫云铁。”萧壮一喜。
<br/>　　“嗯。”萧逸笑道，“装满了整个中级乾坤袋，估计是慕容轩他们整整一、两个月的挖矿收获了。”
<br/>　　.....
<br/>　　天色渐黑，萧逸等人回到了萧家的大型矿脉。
<br/>　　大型矿脉，比之小型矿脉巨大了百倍以上，占地也极广。
<br/>　　三大家族都分别在自家的大型矿脉上建立了大型据点，防守严密，外人休想染指。
<br/>　　小型矿脉可以说是家族子弟间的小打小闹，就算被抢了也影响不大。而大型矿脉，则是三大家族的重中之重，绝不可能被外人得到。
<br/>　　萧逸也是第一次来萧家的大型矿脉，看着那高耸入云的巨大紫色山峰，不禁对这个世界的神奇更为惊讶。
<br/>　　那密布的紫云铁，不仅将整个天空映衬得一片紫色，就连萧家营地，也如同被一块巨大的紫色宝石所包裹，美丽至极。
<br/>　　萧逸从乾坤袋中拿出了一块紫云铁，用力握了握，足足用了一成力度才捏碎。
<br/>　　以萧逸的实力，就算只有一成力，也比凡境九重强大得多。
<br/>　　可想而知紫云铁的坚固程度，难怪会被用在武器的锻造上，更是价格不菲。
<br/>　　“哟，萧逸，你怎么这么快就到了。”
<br/>　　“听大长老说，你今天早上才出发的，起码该明天才到吧。”
<br/>　　两道熟悉的声音传来，萧逸抬头望去，竟是四长老和六长老。
<br/>　　家族里，七八九长老与五长老狼狈为奸；而四、六长老则是中立，但有时也会倾向于三长老。
<br/>　　他们两人本来就是负责萧家矿脉事宜的，处理完家族的琐碎事后，早早就回到了矿脉。
<br/>　　“参见四长老、六长老。”萧壮等人纷纷行礼。
<br/>　　萧逸倒是不用行礼，他的少家主身份，与各大长老齐平。
<br/>　　萧逸道，“想早些过来，便没有骑马赶路，自己跑过来了。”
<br/>　　也幸好他没有慢慢骑马赶路，否则，今天肯定无法及时救援萧壮他们。
