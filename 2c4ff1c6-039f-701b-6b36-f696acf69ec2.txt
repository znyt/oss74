        &lt;h3&gt;第三卷第二百九十七章  行动&lt;/h3&gt;
<br/>
<br/>        郝景文和从哲、傅海生在讨论西城运营的事情，这才几天时间，西城就发生了好几起事故，他们的车无法进行正常的运营，如果再这样下去，他们的工作就无法开展。
<br/>
<br/>        郝景文：“从哲，你之前在这个行业待得久，有些人你也认识，你联系一下，看看是什么情况。”
<br/>
<br/>        “我已经联系过了，他们准备抱团抵制我们，他们觉得我们抢了他们的生意，另外过几天他们会罢运。”从哲已经将情况打听清楚了。
<br/>
<br/>        “什么时候罢运？”傅海生问。
<br/>
<br/>        “下周的周三周四周五，三天时间。”从哲手中拿着烟，轻轻的揉捏着。
<br/>
<br/>        “如果他们统一时间罢运，将会引起巨大的影响，光我们的几百俩车根本解决不了西城巨大市场出行的问题。”郝景文好看的眉头拧在一起。
<br/>
<br/>        “我是协会的成员，我可以提一些建议，我怕我阻止不了，毕竟人数太多。”从哲也有些担心，他们没有想到事情会发生这样，之前别的地方也没有这么大的反响。
<br/>
<br/>        “我的方案已经做完了，我和你一起去看看情况。郝总如果能想办法让官方出面更好。”傅海生将手中的方案发给郝景文。
<br/>
<br/>        “江总和陈子昂已经去联系人了，应该很快就会有结果。我们分头行动吧。”郝景文站起来，他需要看看傅海生的方案，还要联系一下江俞轩，看看江俞轩那里是什么情况。
<br/>
<br/>        江俞轩找到陈子乔，说明了来意，陈子乔原本对妹妹抱有歉意，江俞轩刚说完话，陈子乔就说：“我带你去见马总，让他给有关部门沟通一下，尽量不要将事情闹大了，要控制在萌芽状态。”
<br/>
<br/>        江俞轩见到马总：“啊哈哈，小江啊，没有想到你这身份还多的不行。”
<br/>
<br/>        “马总说笑了，我就是混日子的，这干什么不是还得靠马总赏口饭吃。”江俞轩毕恭毕敬的说，那个时候在一个桌子上称兄道弟，现在人家可是大佬。
<br/>
<br/>        “是有什么事情吗？”马总对陈子乔说，最近陈子乔都很少来找他，他知道陈子乔心里有气，他也没有想到夏颜手伸得那么长，但是事情已经发生了，他也不能再说什么。
<br/>
<br/>        “江总准备到西城投资，他们先期投放了一些车辆，借鉴北京的运行方案，到西城就不行了。”陈子乔将资料放在了马总的面前。
<br/>
<br/>        “我记得这个之前有报备啊，是个好事情，南方很多城市不是都已经有了吗？落家的杭州峰会不是全部用的他们的车辆？”马总看着手中的资料说。
<br/>
<br/>        “是啊，可是现在他们的车辆无法运营，还有后期不知道会出现什么问题。”陈子乔轻轻的说。
<br/>
<br/>        “你们先回去，我一会打个电话，让人去处理一下，你们的这个投资方案不错，准备什么时候实施？”马总问江俞轩。
<br/>
<br/>        “只要这边批准，我们立马实施。”江俞轩自信的说。
<br/>
<br/>        “行，明天你去找相关部门，将这个事情早点落实下来，我觉得你们来了之后会是我们西城一道靓丽的风景线。”
<br/>
<br/>        “谢谢马总的认可，我们会努力成为西城的一道靓丽的风景线。”
<br/>
<br/>        江俞轩从马总那里出来对陈子乔说：“谢谢四姐，让你为难了。”
<br/>
<br/>        陈子乔微微一愣，对江俞轩说：“子昂让你来找我的吧？”
<br/>
<br/>        江俞轩：“你们陈家人都是人精，谁想干什么你们都知道啊？”
<br/>
<br/>        “都是自己家的姐妹，谁心里咋想的能不知道？”陈子乔微微一笑，江家是有自己的门道，不去找自己的人，来找陈子乔绝对是陈子昂的注意。
<br/>
<br/>        “其实，四姐，我一直都没有明白子昂为什么要我来找你。”江俞轩不知道陈子昂的布局和想法，当然不知道陈子昂要他来找陈子乔的意义，不管怎么说，只要结果是好的就行了。
<br/>
<br/>        “她让我以后别在这个位置上干了。”陈子乔叹了口气。
<br/>
<br/>        “你和马总闹矛盾了？”江俞轩脱口而出。
<br/>
<br/>        “也不算闹矛盾吧，只是各自的工作性质不同了，人家现在又都是大佬，我没事就不会去麻烦他的了。”陈子乔苦笑了一下，以前要有多风光，以后怕是要有多凄惨了。
<br/>
<br/>        江俞轩沉默了。
<br/>
<br/>        陈子昂让他来找陈子乔，无法是要陈子乔向马总低头，缓和俩个人之间的关系，他只是提供给他们一个缓和关系的机会。
<br/>
<br/>        郝景文接到了江俞轩的电话，说有关部门已经开始协调，让他们该干什么还是干什么，另外投资方案只要被有关部门一落实，便要进行投资和施工，让郝景文做好准备。
<br/>
<br/>        而从哲和傅海生一起参加了出租业的行业协会，傅海生用他那三寸不烂之舌说服了很多人，大家已经在心里动摇起来了。
<br/>
<br/>        傅海生说：“新的事物更能促进一个行业的发展，大家在这个行业几十年如一日，想一想你们有没有寻求过改变？现在有了新鲜的血液我觉得他们给你们带了新的使命和新的机遇，你们现在是要寻求突破和改变的时候，而不是固步自封，首先大家应该想一想该怎么去改变，改变自己，改变我们这个行业的服务环境和服务态度。
<br/>
<br/>        你们如果不运营的话，你们可以去试坐一下他们的车，体验一下他们的服务态度。对比一下，寻找自己的突破口，加以改变，这样才能立于不败之地，才能不被社会所淘汰，你们抵制不了社会发展的步伐，也抵制不了信息带来的机遇和毁灭。”
<br/>
<br/>        傅海生的一番话像投入湖心的石子，激起阵阵涟漪。
<br/>
<br/>        那一些原本就不想罢运的人们心里松动下来了，无论千变万化，有一条是恒古不变的，那就是安分做事，市场太庞大了，谁也不能将整个市场垄断下来。
<br/>
<br/>        第四天，有关部门发了通告，严禁非法聚集，严禁无理由罢运。
<br/>
<br/>        一场既将爆发的行业之争悄无声息的销声匿迹了，郝景文将后期的准备工作交给了从哲，另外他实在是想留下傅海生，但是傅海生拒绝了。
<br/>
<br/>        “我在哪里工作都是工作，不一定是在西城，我在泉城也可以工作，现在的网络这么发达，你们有什么事情可以安排给我做，我一样会把工作干好。”
<br/>
<br/>        郝景文其实觉得傅海生真的是可惜了，但是他也很知道傅海生的心思，便不勉强，傅海生在西城呆了一段时间后返回了泉城，再此期间连夏颜的面都没有见。
