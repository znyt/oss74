        林苏依照洛羽的意思，让司机将连星儿送到了临海大学附近。
<br/>
<br/>        连星儿下车后，刚准备走，洛羽忽然间又叫住了她：“星儿。”
<br/>
<br/>        一阵措手不及之际，连星儿只觉自己的手臂被捉住，随着一道力度，整个身子旋了过去。
<br/>
<br/>        洛羽一把将连星儿搂入怀中，温柔地低沉道：“让我抱抱你，不知道下一次见面，又要等到什么事情后。”
<br/>
<br/>        两人就在保姆车的车门口相拥，更确切地来说，只是洛羽抱着连星儿。
<br/>
<br/>        连星儿有点儿晃神，双手垂在两腿旁，抬也不是，不抬也不是。
<br/>
<br/>        车内，林苏很满意洛羽这个行动，这样或多或少能俘获这丫头的心吧！
<br/>
<br/>        洛羽抱了一会儿便放开了连星儿，他抬起手来，轻柔地刮了刮这丫头的鼻梁。
<br/>
<br/>        “不管遇到什么事情，开心的事，不开心的事，你都可以告诉我。”洛羽微笑着说。
<br/>
<br/>        连星儿怔愣了一下，缓过神来后，连连点头。
<br/>
<br/>        “拜拜，星儿。”洛羽莞尔一笑。
<br/>
<br/>        连星儿也跟着挥了挥手。
<br/>
<br/>        不远处，看到两人亲热这一幕的明司寒，双手不禁握成了拳。
<br/>
<br/>        平生第一次，他被一个已经有男朋友的女生“捉弄”了。
<br/>
<br/>        或许是因为气不过，又或许是有了一丝恨意，又或许是有些不甘心等等，种种复杂的情绪绕上心头后，明司寒掏出手机翻到连星儿的手机号码后直接拉黑了。
<br/>
<br/>        回去后，连星儿给洛羽发了短信报平安，然后又给张蓉蓉打了通电话。
<br/>
<br/>        结果也打不通。
<br/>
<br/>        实在是丈二的和尚摸不着脑袋后，连星儿拨了明司寒的手机号码。
<br/>
<br/>        然而……
<br/>
<br/>        连星儿纳闷了，电话没打通的原因很简单，十有八九是他们三个人全都将她给拉入了黑名单。
<br/>
<br/>        顾千凝见连星儿一个人趴在书桌前，看着手机屏幕发呆，便拿了条椅子，走过去坐在了连星儿的身旁，关心道：“星儿，你怎么不开心呢？”
<br/>
<br/>        “我的两个高中同学，还有那个明司寒，他们三个同时把我的手机号码拉入了黑名单，是什么意思？”连星儿闷闷不乐地问。
<br/>
<br/>        顾千凝想了一会儿，瘪了瘪嘴，回答道：“会不会是他们三个直接发生了什么矛盾，然后怪到你头上来呢？要不，你去找他们三个一一问清楚，说清楚？”
<br/>
<br/>        连星儿咬了咬下唇，咧嘴一笑：“算了！拉黑了就拉黑了吧！反正，他们对我来说可有可无！”
<br/>
<br/>        反正，最近她的朋友缘是真的很差劲。
<br/>
<br/>        先是网络世界里的师父，现在又轮到了现实世界。
<br/>
<br/>        不过，自从明司寒把她的手机号码拉入黑名单后，连带着他本人好像一并也从她的生活圈里消失了一样。
<br/>
<br/>        往常在食堂里，还能遇到明司寒和他的那三个室友，现在能遇到的，只有他的三个室友。
<br/>
<br/>        并且，那三个室友在排队的时候遇上她们了，他们三也视而不见，好像从来就没认识过一样。
<br/>
<br/>        对于这三个男生的反常，顾千凝、赵安彤和路娅玟也闭口不谈。
<br/>
<br/>        大致都能猜到是与连星儿有关。
<br/>
<br/>        军训结束后，进入正常的上课阶段。
<br/>
<br/>        班上总共就三十个人，有三分之二是男生，但这些男生从来不敢靠近连星儿，哪怕连星儿主动找他们说一句话，接着他们就像见了瘟疫一样，对她是有多远就躲多远。
<br/>
<br/>        连星儿也无所谓，反正不跟他们说话又不会死。
<br/>
<br/>        更何况，她的异性朋友中，还有洛羽。
<br/>
<br/>        好像……
<br/>
<br/>        她就只有洛羽这么一个异性朋友。
<br/>
<br/>        经过这些天的沉淀发酵，明司寒的那三个室友总算是明白了，明司寒对连星儿是单相思，人家连星儿压根就没把他明司寒当回事。
<br/>
<br/>        虽然明司寒的三个室友不理会连星儿，但是没有不理会连星儿的室友。
<br/>
<br/>        顾千凝在校园超市里买东西的时候，遇上了赵子晋，于是主动过去跟他套近乎。
<br/>
<br/>        “最近，你们怎么都不理我们了啊？”顾千凝将赵子晋欲要拿的饮料抢先一步拿走。
<br/>
<br/>        赵子晋看向顾千凝，只是会心一笑：“想吃什么，我请客。”
<br/>
<br/>        “那我就不客气了！你吃什么零食，我就吃什么零食，你每样拿双份就好了。”顾千凝喜上眉梢地说。
<br/>
<br/>        赵子晋照做，将自己喜欢的零食每样拿了两份放购物篮里。
<br/>
<br/>        “是不是因为明司寒和我们家星儿的事情？”顾千凝走在赵子晋的身旁，有一搭没一搭地问。
<br/>
<br/>        一提起连星儿，赵子晋就莫名其妙地来气。
<br/>
<br/>        “连星儿既然有男朋友了，还去勾搭司寒。”赵子晋义愤填膺地说道。
