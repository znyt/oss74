
<br/>　　<br/>　　    渭河河道中，那两个落水的人身子翻腾在浪花中，起伏了两个来回之后，就看见他们的手臂伸出水面晃了晃，然后就一下子淹没在河水中没了影子，从落水到消失也就不到两分钟的时间，河岸上的渔船和木舟都还没有准备好离岸呢。<br/>　　<br/>　　    老船家的脸色“豁”的一下就白了，嘴唇哆嗦着就嘀咕一句话：“完了，完了，是河神不高兴了，今年靠水吃饭恐怕要遭殃了，哎，老天爷不赏饭了啊······”<br/>　　<br/>　　    王惊蛰抹了把脸上的雨水，瞅了眼旁边的姑娘为了避免继续尴尬，就顺着老船家的话问了一句：“怎么就完了，怎么就跟河神扯上关系了？”<br/>　　<br/>　　    “落水的两个人，那个我不知道是谁，另外一个却是渭河边的老相识了”老船家脸色忧愁的指着河面那边说道：“他是船工老赵，我都认识他几十年了，当初他妈生他的时候就是在渭河边洗衣服时生下来的，都没来得及去村卫生所，老赵一出生就掉在了河水里，说来也奇怪那么小的婴孩落在水里却什么事都没有，就飘在了水面上，他七八岁的时候跟我们就在河边玩耍，一个猛子扎下去半天都不会露头，十二岁就在渭河开始跑船了，一跑就是三十几年，我们这种人一辈子中有大半辈子都是飘在渭河上的，对这条河的熟悉都跟自己的身体差不多了，你想想啊就这样的老赵掉在渭河里，虽然是刮风下雨，但这么快就没影了，正常么？别看风大浪也大，可也不见得能淹死他啊······”<br/>　　<br/>　　    老赵说话的时候，旁边的船工和渔家脸色都不太好看了，有人就说道：“是河神啊，一定是河神动怒了，我们得找个好日子该祭奠一下了，杀猪宰羊祭奠河神”<br/>　　<br/>　　    生活在河边的人大部分都对河水有一种莫名的敬畏心，就像大山脚下的人一样会笃信山神，只要有灾了，九成九会跟河神什么的联系在一起。<br/>　　<br/>　　    到了现代社会笃信归笃信但情形还没那么过分，早几百年前，河边的村镇每年都是要祭一对童男童女给河神的，活生生的两个孩子直接就给沉到河底了。<br/>　　<br/>　　    说来也怪，这时候风雨也见小了，刚刚的狂风大作和瓢泼大雨瞬间就弱了下来，河面也不再翻腾不止了。<br/>　　<br/>　　    老船家突然跪在地上，紧接着他旁边的人也都跪了下来，虔诚的朝着渭河方向叩拜。<br/>　　<br/>　　    王惊蛰皱眉问道：“这边水深么？刚刚淹死的那两个，不得把尸体捞上来么？”<br/>　　<br/>　　    老船家摇头说道：“不捞了，捞也捞不上来，风浪那么大，尸体沉底之后早就被水下的暗流给冲走了，怎么捞啊？”<br/>　　<br/>　　    “天灾人······”王惊蛰刚感慨了一句，忽然有一丝很好闻的香气钻进了他的鼻子里，然后他就用力的嗅了嗅。<br/>　　<br/>　　    这股香特别好闻，比刚刚吃的那两条渭河鲤鱼还要香。<br/>　　<br/>　　    王惊蛰一转脑袋，就看见那钓鱼女子也来到这边，香味就是从她身上透过来的。<br/>　　<br/>　　    “哦，体香”王惊蛰豁然通透了。<br/>　　<br/>　　    这姑娘来了以后眼神只在王惊蛰的脸上停留了不到一秒就掠了过去，王惊蛰顺着她的眼神望向河面，他忽然发现刚刚两人沉水的地方还在泛着水花，一圈一圈的散了出去。<br/>　　<br/>　　    没过多久，可能是因为阴天的原因，天色很快就黑了，渭河河面也重新恢复了平静变得古井不波的，连流水声都变的弱不可闻起来。<br/>　　<br/>　　    这时候，河边来了不少的人，都是附近的村民，其中以船工老赵家的人最多，一个几十岁的老妇女堆坐在河边嗷嚎大哭，几个年轻的男女抽泣着蹲在河边烧着纸钱，老赵淹死在了河里，家人是最难过的。<br/>　　<br/>　　    王惊蛰一直都没走，就留在岸边等着天黑，他对淹死的这两个人有点好奇，他可是不信什么河神发怒了的说法的。<br/>　　<br/>　　    河神没那么小气，不会因为一点不高兴就祸害两条人命的，河神也是在乎因果和阴德的，除非河神脑子进水了他才会凭白无故的淹死两个人。<br/>　　<br/>　　    天下万物都有灵，就是阎王索命，也得看死的这个人该不该死。<br/>　　<br/>　　    人死的头一天，按照习俗家里人都是要守夜的，特别是头七之前死者的亡魂都有可能回到家里，或者出现在死的地方，这个时候家人就要在这守着，不要让灵魂感觉到孤苦无依。<br/>　　<br/>　　    一直等到快要午夜时分左右，河岸远处忽然走过来一道身影，手里提着一盏灯，等走近了后王惊蛰看清对方，还是那个钓鱼的女子。<br/>　　<br/>　　    这女子拎着灯走过来后就蹲在地上，轻声跟船工老赵的媳妇说道：“老人家去世尸骨无存也不太好，我倒是有个法子能把他的尸骨找出来，你们要不要试试看？”<br/>　　<br/>　　    老赵的媳妇和儿子全都诧异的说道：“人都被卷到河底去了，白天暗流那么大，这时候尸体恐怕早就不知道冲到哪里去了，还找得到么？”<br/>　　<br/>　　    “也不一定，可以试试····”<br/>　　<br/>　　    老赵媳妇连忙说道：“姑娘那就麻烦你了，还希望能把尸体给捞上来，谢谢了”<br/>　　<br/>　　    “把人的生辰八字告诉我，再找一件他的贴身衣服过来就行了”<br/>　　<br/>　　    王惊蛰在旁边一听就知道对方是什么意思了，这女子手里拎着的是一盏引魂灯，加上生辰八字是可以把死者的亡魂召过来的，老赵的魂魄一出现，自然就能知道他的尸体沉在哪里了。<br/>　　<br/>　　    老赵家人把他的生辰八字告诉这女子后，她就写在了一张符纸上，然后将引魂灯放在地上。<br/>　　<br/>　　    符纸被点燃后“噗”的一下就烧了起来，这女子挥手一扬纸灰就纷纷扬扬的飘了出去：“子女把他的衣服穿在身上，然后呼喊他的名字，连叫三声就可以了······”
