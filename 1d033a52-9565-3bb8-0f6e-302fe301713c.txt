<br/>    在曙光大君正在渡劫的同时，从古仙门之中又传出了一个消息，四大古仙门，四位老君，已经前往了一处名为登仙阶的地方。
<br/>    
<br/>        关于登仙阶，众多的修行者也是头一次听到这个名字，完全不明所以，根本不知道登仙阶蕴含的秘密。
<br/>    
<br/>        不过对于无疆来说，却是让他震惊不已，“难道说，陈明要成仙，这是真的！登仙阶，可是仙路的入口，难道说古仙门准备给陈明开仙路不成？”
<br/>    
<br/>        “能够随意将血海魔刀这等宝物给门下一位不过入道境的弟子，就是凌仙，手中也是拥有着太初宗的镇派之宝，门下大弟子，卓青瑶的手中，同样是拥有仙器，这陈明，到底是何方神圣？莫非，他与我一般，都是距离成仙只差一步的存在？”
<br/>    
<br/>        “我倒要看看，古仙门是不是真的敢重开仙路！”
<br/>    
<br/>        就在这时，李素衣走了进来，禀报道:“禀报宗主，三宗主，四宗主，五宗主，六宗主，联袂求见！”
<br/>    
<br/>        无疆坐在黑色的玄铁大座上，大殿之中是诸魔雕像，“他们应该是为了陈明的事而来的，毕竟一位仙人，对于他们来说，是不可抵挡的存在。对了，你师父，到底是一个什么样的人？”
<br/>    
<br/>        李素衣直接开口道:“大道三千，无一不通。我的刀法，是师父教的，凌仙的搏仙术，也是师父教的，大师姐的剑法，还是师父教的，四师妹的术法，功法，同样是师父教的。”
<br/>    
<br/>        无疆的心头一震，无论是血海魔刀，还是搏仙术，或者是卓青瑶的剑法，在他的眼中，都是绝代天骄施展的术法，而这些绝代天骄，竟然出自一门！
<br/>    
<br/>        难道说，陈明一直在隐藏实力，对了，攻打十万大山之时，听闻燕山只是出兵十万，而现在，燕山的兵力，足足有着数十万，陈明，绝对是在隐藏实力！
<br/>    
<br/>        难道说，陈明也发现了仙路断绝，所以隐藏自己的实力吗？
<br/>    
<br/>        无疆深思片刻，然后道:“告诉他们，陈明他不可能成仙，我就不信，古仙门会让一个外人成仙。不过同样告诉他们，对燕山的一切行动，筹谋，全部取消！就算陈明成仙失败，也不是他们可以招惹的！”
<br/>    
<br/>        待得李素衣离开，无疆暗自道:“若是如此，陈明未必不可以争取到自己的阵营来，他行事多有魔宗风范，阴晴不定，而且和我一样，因为古仙门关闭了仙路无法成仙，倒是可以拉拢一番！”
<br/>    
<br/>        陈明行走于众多修行者汇聚之地，每当他到时，头顶都有着一朵劫云，众多的修行者，越来越相信，陈明是在渡仙劫，不然，你见过持续了十几天的天劫不成？
<br/>    
<br/>        “这肯定是仙劫啊！”
<br/>    
<br/>        “已经持续了十三天了，这么长时间的天劫，必定是仙劫无疑！”
<br/>    
<br/>        “一位新的仙人将要君临天下了吗？”
<br/>    
<br/>        陈明看了一眼众人的反应，很是满意，如今整个大陆，都在传陈明正在渡仙劫的事，这些消息，已经传得沸沸扬扬。
<br/>    
<br/>        陈明要的，就是这个效果，这样一来，燕山自己的计划就成功了不少了，虽说古仙门的动作陈明看不懂，但是估计也是有着古仙门自己的算计。
<br/>    
<br/>        现在自己要做的，就是装高人。
<br/>    
<br/>        于是陈明抬头，看向天际，眸光深邃不可测，如同群星映照其中，“仙路渺茫几难寻，今日蓬莱到门前！”
<br/>    
<br/>        众多修行者听到陈明的话，顿时只觉心潮彭拜，“真的是要成仙了吗？”
<br/>    
<br/>        “曙光大君，意在成仙！”
<br/>    
<br/>        为了让自己装的更像，陈明展开背后的星图，璀璨星光落下，让他如同是仙人下界！
<br/>    
<br/>        众多修行者看到这番模样，更是惊叹地无以复加，“这是什么，难道是成仙之时的异象！”
<br/>    
<br/>        “仙师，这是仙师啊！”
<br/>    
<br/>        陈明淡淡扫了周围一眼，而后消失在茫茫大山之中，众多修行者看到陈明来无影去无踪的模样，更是如同魔怔了一般，“仙师莫走！”
<br/>    
<br/>        “我等还想在仙师座下聆听仙道呢！”
<br/>    
<br/>        陈明来到一处荒山之中，开始渡自己的天劫，准确的说，是参悟眼前的这一道星光之中蕴含的周天星辰大阵。
<br/>    
<br/>        将这座大阵参悟完毕，陈明心想，消息已经传出去了，就先消失一段时间，保持自己的神秘感。
<br/>    
<br/>        在一处荒山之上，陈明寻了一块青石，直接盘坐在上，反正也是闲来无事，干脆继续在灵剑上绘制传送阵吧。
<br/>    
<br/>        卓青瑶最近可谓是出尽了风头，她也是按照自己心中所想的，准备战尽这个天下的诸王，如今死在她手上的诸王，怎么说也有上百了，而她心中温养的剑意，也是开始逐渐凝聚，所谓剑意，便是凝聚到极致的意念，而她，便是通过斩杀诸王，凝聚自己胸中的剑意。
<br/>    
<br/>        这一日，卓青瑶在自己的大营之中休息，忽然察觉到身旁如同有人在窥伺，“谁！？”
<br/>    
<br/>        卓青瑶一声暴喝，看向大营之外，在一处大树之上，还留有离开的脚印，卓青瑶本想直接追过去，但是想到师父的话，还是背上了陈明给她的国器长剑。
<br/>    
<br/>        夜色茫茫之中，卓青瑶朝着对方的踪迹追了过去，追了一阵，卓青瑶忽然叫道:“不好，这是引诱我出来的伎俩！这些踪迹，实在是太过明显了！”
<br/>    
<br/>        卓青瑶迅速后撤，不过就在这时，天空之中，一只黑色的鬼爪忽然朝着她飞来，将她打得倒飞一里，索性卓青瑶肉身强大，并未受什么重伤。
<br/>    
<br/>        “你明白的太晚了！”
<br/>    
<br/>        一个黑衣人朝着卓青瑶走来，“要是继续放任你征伐下去，恐怕鬼仙大人麾下的诸王，会被你杀个干净，不过碍于战争条例，我也不好直接在大营之中出手。不过现在这荒山野岭，谁能知道你是死在我的手下！”
<br/>    
<br/>        黑衣人逐渐向着卓青瑶走去，“没有想到，受了我一爪，你居然没有受重伤，此番实力，我果然留你不得！”
<br/>    
<br/>        卓青瑶看向身前的黑衣人，目光灼灼，并没有半分怯意，“看来，你是一位大君？”
<br/>    
<br/>        “大君啊，我可是从来没有交手过的，而且是这般生死搏杀，就让我好好看看，大君到底有多强吧！”
<br/>    
<br/>        卓青瑶拔剑，巨阙直指黑衣人，浑身剑气暴卷。
