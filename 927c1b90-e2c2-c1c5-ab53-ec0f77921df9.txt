　　两头妖兽，身躯庞大，面容有些怪异。
<br/>　　怪异得，让人叫不出名字。
<br/>　　起码，哪怕是惊风长老与天赤长老这两位成名数百年以上的强者，都认不出这是何等妖兽。
<br/>　　面对洛前辈的淡漠之言，两头妖兽没有丝毫畏惧，反倒面露戏谑之色。
<br/>　　“姓洛的，给我们个理由。”
<br/>　　“若是我们满意，说不定洞府内的魂力本源，便借你了。”
<br/>　　洛前辈闻言，淡漠的面容上，皱了皱眉。
<br/>　　“武魂觉醒。”
<br/>　　洛前辈一如既往地淡漠，吐出四个字。
<br/>　　“武魂觉醒？”两头妖兽闻言，桀桀一笑。
<br/>　　“想来不是你这老东西要觉醒，是你要帮人觉醒。”
<br/>　　洛前辈没有多言，只是淡漠道，“让开吧。”
<br/>　　“让开？”两头妖兽戏谑一笑。
<br/>　　“那可不行，我们只说说不定借你，可没说一定借你。”
<br/>　　洛前辈淡漠的脸色，霎时一冷，“孽畜，你敢戏耍老夫？”
<br/>　　“老夫只说一遍，滚开。”
<br/>　　“哈哈哈哈。”两头妖兽，似是听到了天大的笑话般，仰天大笑。
<br/>　　“让我们滚开？笑话。”
<br/>　　“这里是我虚族的地盘，你当我等怕了你这老东西不成？”
<br/>　　“咦，不对。”两头妖兽，忽然紧紧地盯着洛前辈。
<br/>　　“姓洛的，你身上气息，怎地弱了许多？”
<br/>　　“啧啧，你这老东西，是越活越回去了。”
<br/>　　两头妖兽，一口一句‘老东西’，嘴上尽是戏谑。
<br/>　　洛前辈淡漠的神色，终于化作了单纯的冰冷。
<br/>　　刚要有所动作。
<br/>　　身后，惊风长老低言道，“院长，你全力替莫悠觉醒武魂不久，险受反噬。”
<br/>　　“这两头孽畜，绝非善类。”
<br/>　　“现今硬撼之，怕是会吃亏。”
<br/>　　洛前辈闻言，双眼一眯。
<br/>　　轰…轰…轰…
<br/>　　此时的幻光险地，魂力暴走不停。
<br/>　　整个天地，便在这股暴走中，震颤不断。
<br/>　　高空之上，空间崩塌破碎，惊雷轰鸣。
<br/>　　唯这两头巨大妖兽，屹立其中，不摇不动。
<br/>　　雷光闪烁，两头妖兽蓦地变得若隐若现。
<br/>　　轰…轰…轰…
<br/>　　无形的力量，忽然爆发。
<br/>　　原本肆虐于险地的浑厚魂力，忽然聚集。
<br/>　　洛前辈三人周遭，空间寸寸尽碎。
<br/>　　“走。”洛前辈脸色一凛，一把捉过惊风长老二人，闪身消失在原地。
<br/>　　……
<br/>　　幻光险地内的暴走以及沸腾，持续了整整一晚。
<br/>　　直至月圆之日逝去，险地之内，方恢复了平静。
<br/>　　而对于此，远在百万里之遥的萧逸三人，丝毫不知。
<br/>　　萧逸亦没有那个兴趣理会。
<br/>　　一晚时间，他已然出了幻光地域。
<br/>　　接下来，他会一路横跨幻光府的势力范围。
<br/>　　“萧逸公子，现在我们去哪？”
<br/>　　高空中，顾怜星轻柔问道。
<br/>　　萧逸刚要回答。
<br/>　　顾怜星却脸色一白，险些晕厥过去。
<br/>　　“怜星。”公孙火舞一惊。
<br/>　　萧逸一把抱过顾怜星，身影一闪，从高空中落到地面。
<br/>　　“小贼，这怎么回事？”公孙火舞急切问道。
<br/>　　“昨晚不是已经帮怜星修补好武魂了吗？”
<br/>　　萧逸将顾怜星放于地面，随后道道星光快速打入她体内。
<br/>　　“昨晚，不过是修补了武魂。”
<br/>　　“可她的天星血脉，仍旧残破不堪。”
<br/>　　“这二者，随便一样溃散凋零，都能要她性命。”
<br/>　　不错，萧逸要做的，不仅是让顾怜星修补武魂，还要修补血脉。
<br/>　　这二者，与顾怜星性命相连，亦是让顾怜星身体虚弱无比的原因。
<br/>　　二者随便一样未修补好，顾怜星都仍旧会危在旦夕。
<br/>　　不过，血脉方面，倒还好说些。
<br/>　　武魂方面，才是一个武者的重中之重，不能有丝毫闪失。
<br/>　　武魂，自觉醒后，便伴随武者一生。
<br/>　　若武魂有闪失，轻则便是修为尽失的下场，重则更是在武魂的溃散反噬下，生机散尽。
<br/>　　故萧逸率先来的地方，是幻光险地，率先让顾怜星先修补好武魂。
<br/>　　十数分钟后，顾怜星缓缓醒来。
<br/>　　“有劳萧逸公子费神。”顾怜星苍白的脸色恢复了正常，娇柔答谢一声。
<br/>　　萧逸淡淡一笑，“现在替你续命，倒是比之前轻松许多。”
<br/>　　所谓续命，乃是萧逸以他的星光力量，暂时让得顾怜星的血脉与武魂充盈，不再虚弱。
<br/>　　之前的续命，需要同时为血脉和武魂打入星光力量。
<br/>　　可昨晚之后，顾怜星的武魂已修复完好。
<br/>　　故萧逸此时的续命，只是单纯地为血脉打入星光力量。
<br/>　　自然的，现在轻松许多。
<br/>　　“对了，萧逸公子，我们现在去哪？”顾怜星问道。
<br/>　　“去血光险地。”萧逸回答道。
<br/>　　血光险地，在血光府的势力范围之内。
<br/>　　血光府，六光之一。
<br/>　　六光，水光府，金光府，雷光府，剑光府，血光府，幻光府。
<br/>　　血光府，距离幻光府地域，还隔着数府的距离。
<br/>　　“去血光险地？”公孙火舞面露疑惑之色。
<br/>　　萧逸点了点头，“血光府，以气血一道而闻名。”
<br/>　　“传言，无数年前创立血光府的那位上古强者，便是修的气血一道。”
<br/>　　公孙火舞闻言，翻了个白眼，“这和怜星的血脉有关系吗？”
<br/>　　“不知道。”萧逸摇了摇头，“但天地万物，皆在武道规则之下。”
<br/>　　“因有天地规则，方有万物。”
<br/>　　“如果血光禁地之内，留有相关的传承、重宝等等，或者是与那位上古强者有关的东西。”
<br/>　　“说不定能有什么办法。”
<br/>　　“当然，这也是我暂时能想到的办法。”
<br/>　　幻光险地，尚且出现过幻光漫天异象之时，武魂被修补的传闻。
<br/>　　可血光险地那边，没有这类的传闻。
<br/>　　萧逸也只是去碰碰运气。
<br/>　　公孙火舞闻言，瞪着萧逸，“小贼，你的意思是，如果连续的血脉不能修补的话；昨晚的武魂修补，不过是白搭？”
<br/>　　“不错。”萧逸点了点头，“怜星姑娘的‘病’，来自于武魂与血脉的冲突。”
<br/>　　“武魂与血脉，也因此变得虚弱不堪，残破凋零。”
<br/>　　“二者必须悉数修复。”
<br/>　　“且，即便此去血光险地，真的有所收获，也不过是续命数年。”
<br/>　　“二者的冲突，一日不消，便无法彻底治愈。”
<br/>　　“数年，够了。”顾怜星面容平淡，轻柔一笑。
