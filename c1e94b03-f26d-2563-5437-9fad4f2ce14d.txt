　　与聂乾和韩盈相处了几日，苏莫对于这两人还是比较信任的。
<br/>　　所以，他愿意去相信两人。
<br/>　　当然，就算两人真的出卖了他，这也没什么大问题。
<br/>　　毕竟，他在神武学府也没有仇人，若是天赋被学府中知晓，说不定会重点培养！
<br/>　　不过，苏莫还是想低调的修炼，不想处在风口浪尖之上。
<br/>　　少倾之后，苏莫收起了太古武魂榜的玉简，他的武魂还要以后摸索，至于太古武魂榜上的武魂，与他也没有多大的关系。
<br/>　　随即，他又开始查看太古异体榜，此榜单上记录了一千种最强的特殊体质。
<br/>　　这些特殊体质，只是人族或从人族中分化出的种族所有，与那些异族无关。
<br/>　　这太古异体榜的第一种特殊体质，就让苏莫来了兴趣，此体质名为‘混沌之体’。
<br/>　　他的大混沌五行神诀，修炼到极致，说是能铸就混沌不灭体，不知道和这种混沌之体是不是有异曲同工之妙。
<br/>　　不过，混沌之体乃是天生的神体，与后天炼成的混沌不灭体，估计不会相同。
<br/>　　除了混沌之体外，其它的特殊体质，看得苏莫是眼花缭乱。
<br/>　　永恒神体！
<br/>　　至尊战体！
<br/>　　神道之体！
<br/>　　不灭圣体！
<br/>　　一千种特殊体质，每一种都是强大异常，拥有毁天灭地的威能。
<br/>　　纯阴之体！
<br/>　　很快，苏莫便看到了熟悉的名字，这正是宏青璇所拥有的特殊体质。
<br/>　　“纯阴之体，修至阴之气，化天地至阳，夺阴阳造化，同时亦是天生炉鼎！”
<br/>　　纯阴之体的介绍，很简短，但是却是让苏莫大感无语。
<br/>　　因为按照这介绍，拥有纯阴之体的女子，应该修炼至阴的武学，才能发挥出这种体质的威力。
<br/>　　而他以前，却是让宏青璇传承长青宫的木属性武学，完全浪费了这种体质。
<br/>　　少倾之后，苏莫叹了口气，便收起了太古异体榜的玉简，又开始查看最后一枚玉简，太古大道榜。
<br/>　　所谓大道，自然是天地规则、奥义。
<br/>　　此榜单记录了一千种最强大的奥义，以空间为尊，时间为王。
<br/>　　换言之，空间奥义，便是最强的奥义，时间奥义次之。
<br/>　　除此之外，还有很多强大玄妙的奥义，比如不朽奥义，居然排在第六名。
<br/>　　对于奥义，苏莫准备过段时间再去详细了解，参悟适合自己吞噬武魂的奥义，现在=阶段还是先将武魂提升到天级九阶再说。
<br/>　　随后，苏莫便静静的打坐了起来，这几****虽然玄力消耗不大，但上次因为魔魂夺舍，精神力消耗很多。
<br/>　　时间流逝，苏莫和聂乾、韩盈静静的调息。
<br/>　　唰！
<br/>　　不知道过去多久之后，韩盈突然睁开了双眼，转头看向远处。
<br/>　　“怎么了？”聂乾问道。
<br/>　　“有人来了！”韩盈凝声道。
<br/>　　“难道是魔甲人？”聂乾心中一惊，顿时站了起来。
<br/>　　苏莫亦是睁开了眼眸，取出了长剑，准备迎击魔甲人。
<br/>　　不过，他也没有太多担心，魔云海的魔甲人虽多，但大得多都是低阶武帝，中阶或高阶的武帝境魔甲人并不多。
<br/>　　“不是，应该是学院的弟子！”韩盈摇了摇头。
<br/>　　韩盈话音一落，天际的尽头便响起了破空之声，有四道人影急速而来。
<br/>　　嗖！嗖！嗖！
<br/>　　四道身影速度极快，呼吸之间便飞到了小岛上空，而后一眼便看到了岛上的苏莫三人。
<br/>　　这四人是四名青年，个个身上气息强大，全部是武帝境二重的武者。
<br/>　　“你们是哪苑的弟子？”为首的一名青年朗声问道。
<br/>　　此人身穿华服，脸型很长，长的尖嘴猴腮，给人一种阴郁刻薄之感。
<br/>　　“我等是南苑弟子，在此落脚。”聂乾起身向四人抱拳说道。
<br/>　　华服青年仔细打量了苏莫三人一番，见只有聂乾一人是武帝境二重修为，韩盈武帝境一重，而苏莫才准帝之境，顿时嘴角勾起了一丝笑容。
<br/>　　“这岛屿是我四人的地盘，你们不跟我们打招呼，居然敢占据我们的地盘，真是好大的胆子！”华服青年脸色一板，冷声喝道。
<br/>　　聂乾和韩盈闻言，顿时面色一变，苏莫则是双眸微眯了下来。
<br/>　　这四人来者不善啊！
<br/>　　这魔云海乃是历练之地，所有神武学府的弟子都可以来历练，对方居然说岛屿是他们的地盘，这不是明摆着找茬吗？
<br/>　　“这位师兄，我们这就走！”聂乾急忙抱拳说道，他倒不是惧怕了，而是生性沉稳，不愿意招惹麻烦。
<br/>　　言罢，聂乾向苏莫和韩盈打了个眼色，就准备离开。
<br/>　　“站住！”华服青年见此，顿时大喝一声，随即四人身形一动，立刻拦住了苏莫三人的去路。
<br/>　　“占了我们的地盘，就想这么走了？”华服青年冷笑这说道。
<br/>　　“你们想怎么样？”苏莫开口了，冷冷的说道，他的眸中寒光一闪而逝。
<br/>　　“你们放心，我们都是神武学府的弟子，当然不会干杀人夺宝的勾当！”
<br/>　　华服青年轻笑一声，道：“你们占了我们的岛屿，每人出点赔偿便好！”
<br/>　　“多少？”聂乾皱眉问道，他没想到这几人如此的无耻，居然想敲诈他们。
<br/>　　“不多，每人一百块下品灵晶便可！”华服青年笑着说道。
<br/>　　“你们怎么不去抢？”韩盈闻言，顿时娇喝一声，气的酥.胸起伏。
<br/>　　一百块下品灵晶，这简直是在抢劫！
<br/>　　她身上的所有灵晶加起来，也才一百多块，对方居然开口就要一百块。
<br/>　　苏莫和聂乾两人，闻言也是脸黑了下来，这种敲诈未免也太狠了吧！
<br/>　　“若是你们不给，那我们只能抢了！”华服青年耸了耸肩，完全有恃无恐，仿佛吃定了苏莫三人一般。
<br/>　　在他看来，他们四人都是武帝境二重修为，完全碾压苏莫三人。
<br/>　　而且苏莫三人非常面生，并不是学府中那些天赋妖孽的奇才，根本不足为惧。
<br/>　　“你们是哪苑的弟子？”苏莫一步踏出，走到了前方，直面华服青年。
<br/>　　“北苑！”华服青年淡淡的说道，他倒是也没有隐瞒。
<br/>　　“很好！”
<br/>　　苏莫淡漠的点了点头，话锋一转，道：“你们打扰了我静修，将你们的储物戒都交出来为补偿，我就放你们一马！”
<br/>　　“什么？”华服青年闻言，顿时一呆，满脸懵逼之色。
<br/>　　他们来敲诈对方，对方居然反过来要敲诈他们？
