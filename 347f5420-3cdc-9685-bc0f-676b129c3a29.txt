　　客栈之中，一间客房之内，苏莫和聂乾、韩盈，在桌前相对而坐。
<br/>　　“两位，你们也知道我这次来天冥星，是要救一位朋友，而这位朋友被巫族一个叫巫天御的人所擒，据我得到的消息，巫天御就居住在黑耀城。”
<br/>　　苏莫目光望着两人，徐徐说道。
<br/>　　“这个巫天御是什么修为？”聂乾沉声问道。
<br/>　　“巫天御的修为倒是不高，一年前是准帝修为，现在的话，想来可能达到了武帝境一重境界！”苏莫道。
<br/>　　“修为不高，那倒还好！”聂乾闻言松了口气。
<br/>　　“嗯，我们现在首要的任务，就是找到黑耀城的位置，此事倒是不难，随便打听一番便能知晓！”
<br/>　　苏莫面色稍稍有些凝重，继续道：“然后，我们要了解一番天冥星巫族的情况，最后再动身前往黑耀城！”
<br/>　　聂乾两人闻言点头，苏莫此法倒是非常稳妥。
<br/>　　“事不宜迟，那我们现在便去查探吧！”韩盈说道。
<br/>　　“嗯！”
<br/>　　苏莫点头，略一沉吟，道：“为了避免为日后带来麻烦，你们两人也稍稍改变一番容貌吧。”
<br/>　　“好！”聂乾两人立刻同意，他们也担心惹下麻烦。
<br/>　　改变容貌对于武帝境武者来说，轻而易举，瞬息之间，两人面部蠕动，便大变了模样。
<br/>　　聂乾的脸盘变的较为宽广，而韩盈本来娇俏的容颜，则变的非常的平凡，甚至略显丑陋。
<br/>　　随即，三人商量了一番，均是离开了房间，分开打探消息。
<br/>　　苏莫前往客栈的大堂，点了一些酒菜，一边饮酒一边听着大堂内人族武者和巫族武者的闲谈。
<br/>　　而聂乾和韩盈两人，则是离开了客栈，外出查探。
<br/>　　苏莫在客栈的大堂之中，听到了不少有用的消息，足足坐了近两个时辰，聂乾和韩盈回来了。
<br/>　　而后，三人便再次回到了房间，交流了一番各自查探的消息。
<br/>　　根据三人查探的消息，巫族天冥星支脉，共有三百亿族人。
<br/>　　这三百亿巫族之人，大半以上都居住在天冥圣城和其九大卫城之中，只有一小半居住在其它城池，比如五华城。
<br/>　　这天冥圣城和九大卫城，汇聚了巫族天冥星支脉中，九成以上的巅峰强者。
<br/>　　而黑耀城，正是九大卫城之一，其具体的位置，也打探清楚了。
<br/>　　只不过，根据三人打探的消息，天冥圣城和其九大卫城，是天冥星巫族支脉的核心城池，只允许巫族之人居住。
<br/>　　换言之，这十座城池之中，除了巫族之人的奴仆，或者是巫族请的贵客之外，不允许人族和其它任何种族居住。
<br/>　　这一下，就让苏莫三人犯难了。
<br/>　　“苏兄，黑耀城作为九大卫城之一，不允许外族居住，想要进城必须有合理的事由，并且还要经过严格的盘查，这如何是好？”聂乾皱眉问道，这样的话他们很难混进去。
<br/>　　“最主要的是，这种全是巫族的城池，我们就算进去了，也很难有机会！”韩盈叹道。
<br/>　　苏莫闻言，沉默了片刻，眸中精光一闪，道：“只有一个办法了！”
<br/>　　“什么办法？”聂乾和韩盈异口同声的问道。
<br/>　　“夺——舍！”苏莫一字一句的说道，夺舍巫族之人，这是唯一的办法，除此之外，他根本想不出任何办法。
<br/>　　聂乾两人闻言，顿时眼眸一亮，这的确是个好办法。
<br/>　　只是，夺舍对于自身的实力，影响很大。
<br/>　　高修为的人，根本夺舍不了，夺舍低修为的人，自身实力会大降。
<br/>　　再者，就算夺舍的人修为很高，战魂和新的肉身，需要很长时间的契合，短时间之内，也发挥不出巅峰战力。
<br/>　　“苏兄，夺舍的确是个好办法，但风险很大，你要不要再考虑考虑！”韩盈俏脸有些凝重。
<br/>　　“别无它法了！”苏莫摇了摇头，面色同样凝重，他可是从来没有夺舍过别人，心中也没有多少底。
<br/>　　不过，在他看来，夺舍反而更安全。
<br/>　　最起码，暴露身份的几率要低很多。
<br/>　　“那我们如何选择夺舍的对象？”聂乾问道，这是一个关键，牵扯到之后在巫族的身份。
<br/>　　“不！你们不用了，我一个人混进黑耀城便可！”苏莫摇了摇头，此去黑耀城太过危险，他不能让两人和他一起进城。
<br/>　　“这怎么行？你单独进城实在太危险了，一旦暴露，必死无疑！”聂乾沉声道。
<br/>　　“若是暴露身份的话，你们进去只会跟着送死！”
<br/>　　苏莫叹了口气，继续道：“你们在城外接应我，我们也好有个照应！”
<br/>　　“这……！”两人闻言有些迟疑。
<br/>　　“好了，就这么定了！”苏莫不容两人反驳，便定下来此事。
<br/>　　随后，三人详细的商议了一番夺舍的计划，便离开了客栈，动身前往黑耀城。
<br/>　　天冥星非常的大，黑耀城距五华城，足有数千万里。
<br/>　　三人一路疾行，数日之后，终于接近了黑耀城数万里之内。
<br/>　　一座青山之巅，苏莫三人身形伫立。
<br/>　　苏莫眸中幽光爆射，将幽冥魔瞳武魂催动到了极致，遥望远方。
<br/>　　在他的前方三万余里之外，坐落着十座庞大无比的黑色巨城。
<br/>　　这十座巨城，中央的一座主城最为雄伟壮阔，城高千丈，延绵无尽。
<br/>　　而另外的九座城池，则分散在四周，拱卫着最中央的主城。
<br/>　　这里便是天冥圣城和其九大卫城，黑耀城正在其中。
<br/>　　“聂兄、韩姑娘，黑耀城就在前方！”
<br/>　　少倾之后，苏莫眸中幽光收敛，沉声说道。
<br/>　　“苏兄，没想到你还修炼了瞳术，当真了得！”聂乾略带惊讶的说道，他还以为苏莫是运用的瞳术。
<br/>　　苏莫闻言轻笑一声，他也没有过多解释，不过，他现在虽然目力越来越强，但全部是修为提高带来的增幅，幽冥魔瞳武魂的效果，已经大大减弱了。
<br/>　　没办法，这武魂的等级太低，已经渐渐不堪大用了。
<br/>　　苏莫心中，甚至有将幽冥魔瞳武魂也进化成战魂的想法，但又放弃了这个想法。
<br/>　　等级太低的武魂，就算进化成了战魂，也不堪大用。
<br/>　　“好多强者！”韩盈侧耳倾听了片刻，声音略带颤抖的说道。
<br/>　　她聆听到了数万里之外的十座城池之中，有无数恐怖的气息波动。
<br/>　　有的气息波动，几乎强大到让她心脏抽搐，不敢聆听。
