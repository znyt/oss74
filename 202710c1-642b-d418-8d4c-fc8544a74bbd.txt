　　第66章
<br/>　　很快黄三元带着廖甲廖乙两兄弟来到房间。
<br/>　　“大当家，什么事？”
<br/>　　“咱们山寨现在有多少现银？”
<br/>　　“现银有一百二十二两。”
<br/>　　银子是短缺货，山寨铜钱是有的，但银子不过一百多两，还是上次讹诈薛半川的。不过黄三元不愧是山寨总管，肚子里有本帐，财物的数目他信口拈来。
<br/>　　“取一百两，给小蝶拿去。”
<br/>　　“用，用不了这么多。”小蝶突然吓了一跳，其实有一二十两银子，她们就能度日。
<br/>　　“拿着。”程大雷道：“你随着黄总管去库房取，里面有绸缎，看的上的选两匹，给你家小姐带回去。”
<br/>　　二人走了，程大雷目光落在廖甲兄弟身上。
<br/>　　“二位，你们身上没有悬赏吧？”
<br/>　　廖甲有些惭愧：“这个……不好意思，我们兄弟还没来得及犯案，不过您放心，下次有机会，我们一定也做一两件惊天动地的案子。”
<br/>　　程大雷无语，合着你们以为这是什么好事呐。
<br/>　　他的意思是让二兄弟去保护苏樱，二人虽然没什么本事，但力气是有的，对付几个小混混不成问题。
<br/>　　之所以让他们去，是因为蛤蟆寨原先几个人，有一个算一个，都背着悬赏，柳家的人，程大雷目前还暂时信不过。
<br/>　　“我现在交给你们一个任务，去落叶城保护一个人。”
<br/>　　“什么人？”
<br/>　　程大雷深吸一口气，一本正经道：“某未来的压寨夫人。”
<br/>　　听到这话，二兄弟立刻拍着胸脯道：“大当家放心，有我们在，绝不让大嫂出事。”
<br/>　　“那么，如果有人欺负你们未来大嫂，你们会怎么办？”程大雷缓缓说着，眼角突然溢出浓郁的戾气：“给我抽丫挺的！”
<br/>　　“呃……不是直接杀掉么？”廖甲廖乙同时问。
<br/>　　“……”程大雷。
<br/>　　趁着小蝶还没回来，程大雷让柳芷准备笔墨，准备把前世见到的，经营酒店的经验照搬写下来，由小蝶带回去交给苏樱。
<br/>　　不过程大雷鬼画符一般的字，实在是惨不忍睹。最后变成他口述，由柳芷代为执笔。
<br/>　　柳芷伺候人是不擅长的，端茶倒水，铺床叠被一窍不通，可是在琴棋书画，女红针线上却很深的造诣，这也应该，人家从小到大是按照千金小姐培养的，可不是要去给谁为奴为婢。
<br/>　　一个个蝇头小楷落在纸上，当真是无比秀美，怕就算前生的书法大家见到也要汗颜。
<br/>　　不多时小蝶回来，程大雷本来的意识是让她在这里住一宿，明日安排她回去。可她说苏樱孤身一人，她不太放心，今天一定要赶回去。
<br/>　　无奈，程大雷只能安排一辆马车，拉着绸缎和银子，还有辣椒，送她回去。这条路不太平，最后程大雷还亲自送了一程，直到看到落叶城的城门，他才独自返回来。
<br/>　　……
<br/>　　黄昏时刻，苏樱在酒楼大堂，让小二上板打烊。
<br/>　　其实，在这个时代，一个女子抛头露面是很丢人的事情，纵然苏樱常以黑纱罩面，却仍难免风言风语。只有那些上了年纪的妇人，才不介意这些。
<br/>　　这个时代的变态之处在于，一方面无比看重贞洁，恨不得青年女子被男人碰到手臂，就得自杀谢罪。而另外一方面，对于一些结婚的妇人来说，却又无比随便，什么公媳扒灰，叔嫂偷情的事情屡见不鲜。
<br/>　　不过对于苏樱来说，她也不在意丢人了，她身上背负得也不差这点事。
<br/>　　这里说是酒楼，其实破败得紧，年久失修，白蚁蛀得楼梯常吱呀呀响。生意也是差到极点，苏樱都不记得自家还有这样一座酒楼，不过到最后，这酒楼却成为了自己的，是自己被赶出家门后唯一获得的东西。
<br/>　　唯一的优点，大概就是地理位置还算不差吧。对面就是苏家的四海楼，常常高朋满座，宾客络绎不绝。
<br/>　　四海楼的掌柜姓李，外号李大头，是苏樱哪位后妈的大哥，也就是苏樱的舅舅。当然，也别指望他们有什么感情了。
<br/>　　他站在四海楼的门口，用毛巾拍打着衣服，往这边看时，脸上露出一个不善的笑容。
<br/>　　苏四海身染重病，活不过多长时间了，只要能将这小妮子赶出落叶城，那么苏家从此之后就要姓李。可令他没想到的是，这小妮子竟真的舍得下脸面，黑布一蒙脸，就要打开门做生意。
<br/>　　哼，可惜你终究是个女儿身，且看你能撑到什么时候滚蛋。
<br/>　　眼睛看到几个大汉走进对面，他咧嘴笑了笑，转身走入四海楼。
<br/>　　走入苏樱酒楼的四个大汉明显不善，脸生横肉，眼睛滴溜溜在苏樱身上乱转。
<br/>　　“老板娘，考虑得怎么样了，这家酒楼什么时候出手？”
<br/>　　“话已经说明白了，这酒楼我是不会卖给你们的。”
<br/>　　原来这四人已经不是第一次来到这里，三天两日便过来寻衅滋事，目的就是要以极低的价格拿下这酒楼，把苏樱赶出落叶城。
<br/>　　“老板娘，我劝你别逞强，这是个爷们的天地，不是你一个小娘们能吃得开的。”
<br/>　　“四位，请回吧，我们要打烊了。”
<br/>　　“打烊！没看到我们四个来喝酒么，开门做生意，还有把客人往外客人赶的道理。”
<br/>　　苏樱眉头颦起，冲小二使个眼色，让他斟酒打发了这几个人。
<br/>　　小二端着酒来，那大汉挥手就是一个耳光，将小二砸到地上。
<br/>　　“呸，你他娘的什么东西，被你摸过的酒也能给大爷喝。”大汉目光落在苏樱身上，更加肆无忌惮：“我要老板娘倒酒给我喝！”
<br/>　　“听说你黑纱后面的脸不给人看，今天大爷非要看看你能怎样！”
<br/>　　“都他娘被山贼上过的娘们，装什么纯！”
<br/>　　苏樱已出离愤怒，可今年才17岁，纵然出身商人之家，也是看得多，做得少，她不知该如何处理眼前的场面，方寸已乱。
<br/>　　一辆马车停在门口，小蝶和廖家廖乙两兄弟从门外走进来。
<br/>　　廖甲目光转一圈，抓起门口一个酒坛，挥手就砸在这人的脑袋上。
<br/>　　啪！
<br/>　　声音响亮，酒浆和血液一起流，破碎的陶片落在地上，湿漉漉一片。
<br/>(本章完)
