        &lt;h3&gt;第三卷第三百零二章  人生路上我和你&lt;/h3&gt;
<br/>
<br/>        陈子昂将陈子建电脑里的数据恢复出来了，解决了陈子建的大问题，陈子建也不会再要想着去找冯睿要资料。
<br/>
<br/>        二秀对他的情意他也是看在了眼里，之前一直想着冯睿，可是那一次夏颜的话彻底浇灭了他心里所有的希望。
<br/>
<br/>        也许，他应该重新开始。
<br/>
<br/>        陈子昂要回北京，父亲陈元良叮咛陈子建去送送陈子昂，在陈元良的心中，他知道他的小女儿有可能会和他的儿子一样踏上一条不归途。
<br/>
<br/>        陈子寒是一个人，可是陈子昂还有很多人，陈子寒在暗，陈子昂在明。他不知道他的俩个孩子能不能顺利见面，也不知道陈子昂能不能将他的儿子带回来，但是他心里充满了希望，俩个人并肩总比一个人单打独斗的好。
<br/>
<br/>        陈子寒的身份在他们家里只有陈元良、冉灵卉、陈子昂知道，因为知道的越多对他们越危险，现在已经没有陈子寒了。
<br/>
<br/>        陈子昂从三姐的口中推理了整个事情的经过，她猜测陈子寒背负着一项艰巨的任务，但是她却不知道陈子寒已经失去了组织和身份，如今陈子寒和祁峥是无家可归的小鸟，他们只有现在的身份。
<br/>
<br/>        甸城，势必行。原来是猜测，现在是证实，她必须和陈子寒一起。
<br/>
<br/>        站在30楼看着外面的车水马龙，西城的街道对她来说已经很熟悉了，她在这里不知道来来回回走了多少次，这一次的离开让她心生不舍。
<br/>
<br/>        她看了看办公室里的一切，将陈子建的笔记本准备交给陈子霖，陈子建却和江俞轩一前一后进了她办公室的门。
<br/>
<br/>        “你们怎么来了？”
<br/>
<br/>        “我来送你。”江俞轩伸手接过陈子昂手中的皮箱，就像之前俩个人出差一样，那么的顺其自然。
<br/>
<br/>        “我也是来送你，爸让我来的。”陈子建怕妹妹说他把生病的父亲扔在病房。
<br/>
<br/>        “你们都回去吧，七哥送我去机场就行了，我又不是不回来了。”陈子昂有些嫌弃的说。
<br/>
<br/>        “二哥回去吧，我送子昂就行了。”江俞轩开口一个二哥叫得溜溜的。
<br/>
<br/>        陈子建一直以为陈子昂和江俞轩是一对，可是那天张倩楠在他们家参加了家庭聚会，他才明白江俞轩和陈子昂是同事和朋友的关系。
<br/>
<br/>        江俞轩的那句“二哥”叫得他有点别扭，难道真的是他陈子建自作多情？
<br/>
<br/>        江俞轩才不管陈子建别不别扭，直接托着行李箱迈开大长腿朝电梯口走过去。
<br/>
<br/>        “二哥，你回去吧，不用送我到机场了，回去告诉爸，我一定会回来看他的，让他保重身体，将身体养的好好的。”
<br/>
<br/>        “那好吧，一路平安。”陈子建想着陈子昂回北京，不让他送他也就不送了。
<br/>
<br/>        陈子霖拿着车钥匙嘟嘟囔囔的从办公室出来，看见江俞轩：“俞轩，你送子昂？”
<br/>
<br/>        “嗯，你要是忙就不用去了，我开车着呢。”江俞轩才不愿意有灯泡跟着他们。
<br/>
<br/>        “行，我这里还有事情处理，那你就送子昂吧。子昂，那我就不去了啊，你注意平安，我和倾秋商量个事情。二哥，你不坐一会？”陈子霖一句话问候了他们三个人，看来事情是比较紧急。
<br/>
<br/>        陈子昂的行程已经定了，西城的事情有陈子霖和易倾秋她也无需多管，也就没有过问陈子霖是什么事情。
<br/>
<br/>        陈子昂不知道的是她前脚走没有多少时间，后脚他们西城公分司差点被查封了。
<br/>
<br/>        江俞轩送陈子昂去机场的路上，一路絮絮叨叨的叮咛陈子昂要注意身体，少喝酒，别事事都冲在前面。
<br/>
<br/>        陈子昂一一的答应了：“俞轩，你最近还回北京吗？”
<br/>
<br/>        “耗材上市的那一天我回去看看吧，到时候叫上大哥和文文去给你压阵去。”江俞轩笑着说。
<br/>
<br/>        “好。”
<br/>
<br/>        “子昂，你去甸城，俩个孩子怎么办？大哥那天的话你还是的注意点。”江俞轩叮咛陈子昂。
<br/>
<br/>        其实陈子昂从知道江俞轩将她的行踪经常给张函和张倩楠说了之后，基本上有很多事情都不会告诉江俞轩了，再加上江俞轩在西城，北京有些事情他是不知道的。
<br/>
<br/>        包括目前他们的安防系统和绝密铁三角，包括守中的事情，江俞轩一概不知。陈子昂也不会告诉江俞轩她要将孩子送走。
<br/>
<br/>        “我知道了，孩子就放家里，有春姨和财叔看着呢，我也会经常回来的。”陈子昂知道江俞轩对她是没有坏心思，但是他和张倩楠毕竟现在是夫妻，谁也不能保证江俞轩会不会给张倩楠再汇报她的行踪，她现在不想和张家的人有任何瓜葛。
<br/>
<br/>        目前看来，陆家和张家是一体的。
<br/>
<br/>        “子昂，无论以后有什么事情，我都会和你在一起，人总是会成长起来的。我也会成长起来的，我们还年轻，还有很长的路要走，希望你的人生路上有我陪着你。”
<br/>
<br/>        “谢谢你陪着我这么久，无论以后的人生会是什么样子，我一直都会感激你和我一起走过的路。感谢你陪着我走过了那么多艰难的岁月，人生路上有你我很幸运。”
<br/>
<br/>        语言有时候真的很苍白，此时的陈子昂心里五味杂陈，她能说什么呢？
<br/>
<br/>        如今，他们也只能是以朋友的身份相处了，至于以后的人生谁也不能保证不会缺席，目前这种状态是最好的。
<br/>
<br/>        我曾经为你动过心，但是我就像是一只刺猬，将自己紧紧的裹在了自己周身的刺里，我怕再次受到伤害，我怕我的爱得不到回应，我怕我那可怜的自尊被碾压得体无完肤。
<br/>
<br/>        我曾经也有过梦想，梦想和你走过人生最美的时光，梦想能够和你一起逐云踏浪，一切不过都是我最好的梦想。
<br/>
<br/>        陈子昂笑着和江俞轩告别，“俞轩，希望你的事业更上层楼，希望你幸福平安!”
<br/>
<br/>        “我的事业也是你的事业，你的幸福平安就是我的幸福平安，你是我心中最美好的存在，你是我想要用一生去呵护的人。”江俞轩将陈子昂送到安检口，对陈子昂说。
<br/>
<br/>        陈子昂微微一笑：“彼此彼此，各自安好才是最好的守望。”
<br/>
<br/>        飞机载着陈子昂飞过西城的上空，看着远去的城市越来越小，陈子昂心中默默的说：“亲爱的们，你们一定能行。”
<br/>
<br/>        这里有她的亲人，希望他们能够安好。
<br/>
<br/>        人生路上有你们的陪伴我才不会孤单，人生路上有你们的陪伴我才能够勇往直前，是你们给了我信心，给了我努力下去的希望。
<br/>
<br/>        亲人们，请一定要保重！
