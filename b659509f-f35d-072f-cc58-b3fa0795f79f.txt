　　元始天尊看了一眼申公豹，缓缓道：“榜上的都是天地神纹，天道赐之，不可言，不可传，在封神结束之前，更不能观之。你不认得，也是正常。白鹤童儿，将此榜收起，交于你师叔！”
<br/>　　白鹤童子取下封神榜，卷了起来，成为一卷轴，申公豹按住心中激动，双手接过。元始天尊又让白鹤童子取出一柄金色小刀，交给申公豹，打神鞭并没有给他。
<br/>　　元始道：“你去吧，此去手持封神榜，建立封神台，代本座拜将封神。此乃定数，你与姜子牙皆是应劫而生之人，亦只有尔等可享封神之功。
<br/>　　此刀乃是绝望之刃，为先天庚金之气所炼，可绝杀准圣以下者，留与你防身。但你需切记，此刃不可多用。此刃既伤人，亦伤己，以你之力，颇为特殊，但亦最多使用五次，否则命丧黄泉。”
<br/>　　好东西啊！申公豹双眼放光，盯着手中金色小刀，之后朝着元始拜谢道：“多谢老师！”
<br/>　　接着，他又开口说道：“弟子还有一事禀告！”
<br/>　　“说。”
<br/>　　“启禀老师，弟子此次上山，因成汤太师闻仲，亲自率领大军征伐西岐，有九龙岛四圣道人、金鳌岛十绝天君等人随行辅助，他们俱皆为法力高深之辈，弟子道法卑微，不敌此等奇人异士，请老师慈悲，救助西岐万民！”
<br/>　　元始淡淡看了一眼他，好似能够看清他心中所想，让申公豹心中一凛，就在他担忧之时，元始淡淡道：“此事本座已经知晓，商灭周兴，此乃天数，西岐乃应天圣主，危急之时，自由高人来助，你亦不必担忧，去吧！”
<br/>　　申公豹不敢再问，只好扣首出宫，下了山崖，骑上黑虎向着西岐走去。
<br/>　　元始在座上，手持玉如意，淡淡道：“尔等随时注意着封神一事，不可懈怠，都散去吧，日后自由会面之时！”
<br/>　　诸阐教弟子皆起身扣首，鱼贯而出。
<br/>　　申公豹一路向前奔走，忽逢一座山脉，见到两个道人正在比试道法，两人都颇为不凡，有玄仙之力，但道法一般，像是没有好的传承一般的散修。
<br/>　　两人见到申公豹，便停了下来，之后对视一眼，便要离开。申公豹见此，心道我如今正在与殷商大战，正是缺少帮手的时候，此二人颇为不凡，或许能为我所用。
<br/>　　见到两人要走，申公豹淡淡道了一声：“道友，请留步！”
<br/>　　突然，一股莫名的力量在涌动，让前方转身要走的两人，竟然生生止住了脚步，身体不受控住地转过身来，但他们心中却颇为焦急，因为他们的身体竟然不受自己控制！
<br/>　　“道友，请留步！”又一句莫名道音传出，让两人内心一颤。
<br/>　　此时，两人的内心平静下来，看着申公豹，只感觉此人颇为和善，两人笑道：“道长为何人，要我等留下干什么？”
<br/>　　“贫道申公豹，乃是西周国师，殷商无道，天下生灵涂炭，此次西周征讨殷商，正是需要两位道兄帮助的时候啊！建功立业，正在眼前！”接下来，申公豹将自己的情况说明，向两人做出的邀请。
<br/>　　“纣王昏庸，我等兄弟自当出一份力，道兄相邀，莫敢不从！”两人瞬间便做出回应，语气之中，大气凛然。
<br/>　　“那再好不过了，两位道兄姓甚名谁，为何我之前从没听说过。”申公豹大喜，接着问道。
<br/>　　“我乃武夷山散人曹宝是也！”
<br/>　　“我是萧升！”
<br/>　　……
<br/>　　闻仲发兵七路大军征讨西岐，俱皆被申公豹所伤，反倒是折损了成汤十七名上将，七十多万兵马，西周威名大振，四方八百诸侯都有归降之意。
<br/>　　商帝辛二十四年，太师闻仲亲自带兵征伐西岐，有金鳌岛十绝天君秦完、赵江、董全、袁角、孙良、白礼、姚宾、王奕、张绍、金光圣母以及西海九龙岛四圣杨森、王魔、高友乾、李兴霸都受到太师所邀，自海外前来，为其助阵。
<br/>　　十绝天君在岐山按甲、乙、丙、丁、戊、己、庚、辛、壬、葵天干之数布下天绝、地烈、风吼、寒冰、金光、化血、烈焰、落魄、红水、红沙十阵，循环相连，合为一阵，称为周天十绝大阵。
<br/>　　十绝阵翻起苍茫杀气，将岐山笼罩其内，申公豹无奈，不能解决此阵，便不能继续向前进攻，而且己方也不会好过，他只好派人去请阐教众仙了。
<br/>　　在众人焦急等待之时，忽然一道红光闪过，一名道人从天而降，但见他黑须大袖，手中抓着一个葫芦，正是少华山云霄洞的赤精子。申公豹见到来人，顿时大喜，连忙上前行礼：“师兄，怎么只有你一人前来？”
<br/>　　“师弟，其他师兄弟自会前来，贫道只是先行一步罢了。”
<br/>　　翌日，不多时，岐山之上黑云之中瑞光飞腾，仙光透露，玉虚门下燃灯道人带着金翅大鹏迦楼罗、黄金道人踏云而来。
<br/>　　众人连忙出殿相迎，彼此见过礼后，上前来观看那十绝天阵。只见岐山之上黑沉沉一片，自成一界，其中阴风怒吼，血光隐现，金光飞射，众人看得心惊胆战，不敢轻言出战。
<br/>　　燃灯开口道：“十绝合一，周天旋转，乃世间一大凶阵，众位之前，可曾与碧游宫各位道友相见了？”
<br/>　　申公豹道：“吾等见那凶阵，不知底细，不敢轻易涉足。”
<br/>　　燃灯道：“尔等派人到商营下一封信，我与各位道友到那阵中先走一遭，试探阵情，之后再言破阵之法。”
<br/>　　申公豹即刻命人去做，金钟一响，双方都来到阵前。西周一方，燃灯带头，之后是金翅大鹏与黄金道人、赤精子、申公豹，再之后是杨戬、哪吒、雷震子、曹宝、萧升，最后是韩毒龙、薛恶虎、金吒、木吒等。
<br/>　　另一边，十绝天君、九龙岛四圣闻太师，都到了阵前，双方对视而立。
<br/>　　燃灯对着众仙道：“尔等在此等待，贫道先走一遭。”
<br/>　　燃灯上前几步，对着对方大声道：“各位道兄都是高明之士，今商天子无道，绝灭纲纪，西方仁君已现，当顺天而为。各位道兄深悟至道，如何不明此理，而行那逆天之事？”
<br/>　　金光圣母催动身下之虎出列，冷笑道：“昔日玄鸟生商，你又怎知殷商没有天命？更何况，我等只是前来助闻道兄一臂之力，扶持殷商再兴，你又何必用大道理来欺压我等？”
<br/>　　其他几人也开口道：“圣母所言极是，我等来此，便是应了劫运，燃灯道兄也不必多言，你我两教也颇有渊源，双方都时常比试，今日便来见个高低！”
<br/>　　燃灯深深叹了一口气，道：“各位道兄既然执意如此，我等也不再多说，唯恐一个不慎，伤了尔等，丧了道果，送了性命，颇为可惜！”
<br/>　　金光圣母大怒，道：“燃灯，休要放肆，大言不惭，你有何本事，来破我十阵，你敢入阵吗？”
<br/>　　说完，金光圣母转身进入阵内，其余九天君也各自进入阵中，黑云滚滚，迷雾笼罩，众人一会儿便消失不见。
<br/>　　……
