　　第55章
<br/>　　三匹快马逆流而上，在城门要关上的最后时候，再次进入落叶城。
<br/>　　夜里有宵禁，晚上街道便没有多少行人。当然，宵禁仅仅对普通人有效，一些大户人家照样夜夜笙歌，通宵达旦，却也不会有人去管。
<br/>　　不管任何一个时代，总有一些人不守规则拘束。或者说，不管任何一个时代，规则也仅仅是对一部分人有效。
<br/>　　在这种情况下，想要找到黄家位置并不困难，那家有歌舞乐声，总是找不错的。
<br/>　　在黄家门外，程大雷找到了鬼鬼祟祟的林少羽。
<br/>　　“大当家，你们怎么也来了？”
<br/>　　“嘘，噤声！”程大雷摆摆手，问：“里面什么情况？”
<br/>　　“柳家小姐就在里面，今天是黄员外的寿宴，城里的大户来得不少。”
<br/>　　目光望着黄家方向，也是一座好大的宅子。
<br/>　　“黄家的守卫力量怎样？”这句话是问孟子云。
<br/>　　“黄家大概有七八十家丁，专门护院的有十几人。”
<br/>　　像黄家这样的家族，很像程大雷前生了解的公司，家里有千亩良田，有专门种田的长工，有管账的账房先生，有管理家中大小事务的管家，有丫鬟、家丁，负责喂马的，负责做饭的，负责种花的，当然也有负责专门看家护院的，很类似保安的职能。
<br/>　　毕竟，因为有程大雷这样的人，这时代的治安不是很好。
<br/>　　现在程大雷这边只有四个人，不过，前后经历两场战斗，都是以少胜多，程大雷心里倒也不发憷。
<br/>　　关键是抢人成功后，如何脱身。
<br/>　　“走，随我去看看那条小路。”
<br/>　　今夜的黄家，张灯结彩，歌舞欢声。
<br/>　　院内摆满八仙桌，高朋满座，来得都是相熟的朋友。今天是黄员外的五十大寿，在这个平均年龄不足四十岁的时代，值得好好庆祝。就连仆人也有自己的位置，只不过在偏角落的地方。
<br/>　　戏台上是黄家自己养的乐班，此时正表演着歌舞，欢声笑语一片。
<br/>　　程大雷来到这个世界已经有段时间，据他感受这个世界是很落后的，科技落后，经济落后，饮食落后，文化落后……所以他觉得，这个时代每个人都活在水深火热中，枯燥而乏味的度过每一天。
<br/>　　这种想法是不正确的。
<br/>　　程大雷目前接触的大多是底层生活的人，对于他们而言，生活自然糟糕，不管在哪个时代。而对这个时代的贵人而言，生活其实还蛮多姿多彩的。
<br/>　　起居有丫鬟服侍，包括更衣吃饭，甚至是入厕。夏天没有空调，有人摇扇，冬天没有暖气，有人生炉，没有汽车高铁，自然有人抬轿子。没有网络、直播，家里有豢养的乐班，面对面表演歌舞。没有游戏，可以驱犬架鹰外出打猎。此外，青楼，诗会，十几岁就有暖床丫头，保管温柔乖巧，公主病什么的，想都不要想。
<br/>　　与这些相比，吃饭没有辣椒，信息落后，买东西要用马车拉钱等等，就太微不足道了。
<br/>　　就像今天黄家的寿宴，院子内有乐班表演的歌舞，屋里相熟的才子聚在一起，有青楼请来的歌女陪着，偶尔赋诗一首，小酌几杯。女眷在今天也热闹起来，凑在一起，讨论着女红，胭脂，还有谁家公子长得最俊。
<br/>　　而能进入大堂的，则都是黄员外的重要客人，大家分座，中间留出一片空地。
<br/>　　现在在空地上表演舞蹈的是柳芷，不，应该称呼她的花名——胭脂红。
<br/>　　着一身红衣，脸上挂着纱，舞动之时裙摆飞扬，露出一双三寸金莲。
<br/>　　这是从京州传来的舞蹈，据说大武天子最是喜欢，上有所好，下必效焉。幽州落叶城偏僻些，舞蹈传到这里已经有些变味。而柳芷也不过刚学这支舞蹈不久，动作远谈不上熟练。
<br/>　　可如花似玉的年纪，如花似玉的相貌，便是站在面前一动不动就足够迷人，更何况是手还舞之，足还蹈之，二八芳龄的身子舒展开，自然添几分迷人。
<br/>　　观舞之人喝着酒，手轻轻敲着桌子，合着音乐的节拍。到这个时间，大家都有些微醺，话也就不像清醒时那么拘谨。
<br/>　　“黄员外好大面子，平日里想要见胭脂红一面可不容易，今天也就是托黄员外的福，能有着福气。”
<br/>　　“这女子跳舞却还是有几分滋味，今日请大家品鉴一番，聊以下酒。”
<br/>　　“回燕楼的头牌，自然不同那庸脂俗粉，这双玉足便值浮一大白。”
<br/>　　“王员外如果喜欢，今夜就让这女子陪你。”
<br/>　　“哦，不敢，不敢，君子不夺人所爱。不过过几天我家里摆宴，若是能请黄员外的面子，把这女子请到家中，却得承黄员外一份情。”
<br/>　　伴着音乐声，胭脂红舞了一支又一支，知道脚尖发疼，小腿发酸，又饮了一杯酒，才被老姆请着离开大厅，来到一间小屋。
<br/>　　“姑娘且在这里歇歇，等寿宴散了，就差人送您回去。”
<br/>　　老姆搁下一封银子，便离开了，这应该可以算是她今夜的出场费。
<br/>　　胭脂红一人坐在屋子里，屋外还有隐隐的乐声传来，心情不知该如何述说，总是有些慌张忐忑。
<br/>　　吱呀一声响，屋门忽然开来，走进来一身材臃肿的年轻人。
<br/>　　这是黄员外的长子，目前在城内守卫军任职，他喝了不少酒，脸上挂着笑容。
<br/>　　“胭脂红姑娘，你长得可真是俊呐。”
<br/>　　她一惊：“你，你要干什么？”
<br/>　　“哈哈，我还能干什么，自然要和姑娘行周公之礼，做那欢喜夫妻。”他回声将房门关了。
<br/>　　她跑下床，拉开和男人的距离。男人嘻嘻笑着，如同老鹰做小鸡的模样，围着桌子绕圈。
<br/>　　在今夜，胭脂红已经是他的囊中之物，他不介意拖延一下时间，享受猫捉老鼠的愉悦感。
<br/>　　这时代有捧戏子的，有捧风尘女子的，一掷千金，哭天喊地闹了相思病的人也是有的。但风尘女子毕竟是风尘女子，四等贱民，那句话怎么说来着，哦，我捧你是你是个玻璃杯，松手时你就是玻璃碴子。
<br/>　　被请到府里表演，后被主人要了，最后也不过给笔钱了事，这种事连新闻都算不上。
<br/>　　这个时候，外面的宴席仍在继续，欢歌笑语一片。
<br/>　　突然，一支火把从天而降，生生砸在戏台上，正演奏的乐班大乱。
<br/>　　“蛤蟆大王来了！”
<br/>　　从府外传来高呼，紧跟着
<br/>　　四匹快马入府，踏破繁华。
<br/>　　
<br/>(本章完)
