        诺筱颖拿着树枝将那条银环蛇送走后，回头看到韩剑锋那滑稽的样子，忍俊不禁起来。
<br/>
<br/>        她这回眸一笑，就像百花盛开时，清丽脱俗，带着馨香的甜味，让傅夜沉看出了神。
<br/>
<br/>        韩剑锋见傅夜沉发愣，连忙拍了拍傅夜沉的胸膛，咳了一声：“咳咳——那个……四少，我们还有正事要干咧！”
<br/>
<br/>        提起正事，傅夜沉那温和的俊脸立马变得十分严肃了起来。
<br/>
<br/>        “你们要去干什么正事？”诺筱颖一时好奇，随口问了一句。
<br/>
<br/>        傅夜沉和韩剑锋面面相觑后，彼此交换了一个不明意义的眼神。
<br/>
<br/>        诺筱颖怔怔地看着他俩，见他俩不说，脑海里突然回想起了那次在临海城里，看到这男人上了一辆军车，如今，他又出现在他们这儿的深山老林里。
<br/>
<br/>        由此看来……
<br/>
<br/>        诺筱颖心里忽然间联想到了什么，便走到自己的竹篓旁，将里面那几株紫色的罂粟花给翻了来，朝傅夜沉递了过去：“你们是不是在找这个？”
<br/>
<br/>        “这……”韩剑锋看到诺筱颖手里的花后，和傅夜沉面面相觑。
<br/>
<br/>        傅夜沉剑眉紧锁，冷声质问：“你从哪儿摘来的？”
<br/>
<br/>        “爬上这座山的山顶，山那边的梯田里，种的全是这种花。你们俩，是不是缉毒警察？”诺筱颖下意识地问。
<br/>
<br/>        他们这种地方，地处边境，经常会有这类似的事情发生。
<br/>
<br/>        傅夜沉和韩剑锋彼此交换了眼神后，对诺筱颖的问题保持沉默。
<br/>
<br/>        诺筱颖心里明白，只是微微一笑，将手里的罂粟花全给了傅夜沉：“这些花，你们拿回去吧！你们现在这种情况，就不用爬上山顶去探看了。山那边，群山围绕，越往那边，越是下山容易上山难。万一你们被困了，我也救不了你们。所以……你们还是回去想想其他办法吧！”
<br/>
<br/>        傅夜沉收好了花，递给了韩剑锋。
<br/>
<br/>        韩剑锋将罂粟花塞入了背后的旅行包中，对诺筱颖微笑着说道：“我们只是来旅游的。”
<br/>
<br/>        “那一起下山吧！”诺筱颖自然不信他这话，但也没揭穿。
<br/>
<br/>        傅夜沉将韩剑锋的臂膀绕过自己的肩头，然后利索地将韩剑锋背在了背上。
<br/>
<br/>        诺筱颖下意识地瞥了傅夜沉一眼，阳光透过茂密的树叶，撕碎了光斑洒在了他的身上，恍若在他俊美的脸上镀上了一层金光。
<br/>
<br/>        不修边幅却仍旧非常好看的男人，诺筱颖从未见过如此。
<br/>
<br/>        傅夜沉无意间偏过头来，与诺筱颖的目光不期而遇。
<br/>
<br/>        诺筱颖猛地回过神，赧然地将目光从傅夜沉身上挪开。
<br/>
<br/>        傅夜沉看着诺筱颖娇羞的模样，不禁会心一笑。
<br/>
<br/>        诺筱颖带着他们下山后，才知道他们开了一辆本地的越野车停在了山脚下。
<br/>
<br/>        她起初是一座山头连着一座山头爬过来的，都没发现这山脚下还听着这么一辆破烂的本地越野车。
<br/>
<br/>        傅夜沉将韩剑锋背到车后座躺好后，关了车门，转身回到了诺筱颖的面前。
<br/>
<br/>        “探望完你爷爷奶奶后，就赶紧回去好吗？”傅夜沉看着诺筱颖，眼神里满是宠溺。
<br/>
<br/>        刘管家说，苏漫雪买了一辆上百万的宝马车回老家探望她的爷爷奶奶。
<br/>
<br/>        苏漫雪的老家就是这儿的，所以，他在这儿遇上她，他一点都不觉得奇怪。
<br/>
<br/>        傅夜沉只是觉得，他和苏漫雪之间被月老牵了一根无形的红线，正是这条红线，让他每每遇到困难时，她都能及时出现替他解围。
<br/>
<br/>        也正是因为这条红线，让他的心在向她慢慢靠拢。
<br/>
<br/>        哪怕面前的苏漫雪，真的犹如刘管家口中所描述的那么不堪，他傅夜沉也认了。
<br/>
<br/>        她就是他心爱的女人，她花他的钱，是应该的，他也愿意给她钱花，只要她开心就好。
<br/>
<br/>        诺筱颖抬眸看着傅夜沉的眼睛，仍旧不知道这男人到底为什么要跟她说这些莫名其妙的话。
<br/>
<br/>        当她想问他什么的时候，他突然捉住了她的双肩，凑近了一步，微微俯身，偏头，猝不及防地吻住了她的唇。
<br/>
<br/>        诺筱颖忽然感到一丝凉意，犹如银丝般绕上了脖颈。
<br/>
<br/>        男人的吻，像蜻蜓点水似的在她唇上停留了几秒钟，便离开了。
<br/>
<br/>        傅夜沉附在诺筱颖的耳畔，呢喃细语地接着说，“宝贝，我会尽快处理完手里的事情，回来陪你的，可能没法回家去看你了，但你在家里要耐心地等着我回来。”
<br/>
<br/>        他所说的“家”是他给“苏漫雪”住的那幢依山别苑，他一直以为面前的“苏漫雪”，就是住进他依山别苑里的那个苏漫雪。
<br/>
<br/>        他一直没空回依山别苑去看她，却没料到，缘分可以让两人千里来相会。他不用回依山别苑，也能遇见她，真好！
<br/>
<br/>        傅夜沉说完后，在诺筱颖的额头上印上了一个离别吻。
<br/>
<br/>        诺筱颖还一脸茫然地愣在原地。
<br/>
<br/>        这个吻，对于诺筱颖来说，就像一把火，烧得她耳根发烫，直到这个吻了她的男人已经转身上了驾驶座，留在她唇上和额头上那属于他的唇温，还影响着她所有的思绪。
<br/>
<br/>        越野车已经驶动，在风中扬长而去。
<br/>
<br/>        诺筱颖对这个男人，有一种说不出的情愫。
<br/>
<br/>        明明没有脸红心跳，但是刚刚，他的那个吻，却让她有些心慌意乱，甚至不知所措。
<br/>
<br/>        那晚后，这是她第二次与他见面了吧！
<br/>
<br/>        第一次，是在水果超市外，他突然强吻了她，然后急匆匆地走了。
<br/>
<br/>        这一次，他又来得如此突然，去得也如此突然。
<br/>
<br/>        他总是对她说那些莫名其妙的话，让她摸不着北。
<br/>
<br/>        算了，别想多了，还是先回家吧！
