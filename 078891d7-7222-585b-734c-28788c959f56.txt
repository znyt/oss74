　　十个呼吸之后，第四城的护城大阵，已经如同大海中的扁舟，风雨飘摇，随时都有要崩溃的趋势。
<br/>　　苏莫牢牢的注意着其它几座城池的局势，严防有人浑水摸鱼。
<br/>　　不过，其它几座城池，全部风平浪静，如同是一潭死水。
<br/>　　轰！
<br/>　　终于，又过去了四个呼吸之后，第四城的护城大阵，轰然奔溃开来。
<br/>　　“出手！”
<br/>　　在护城大阵奔溃的那一刻，第四城之中，响起一声洪亮的暴喝声。
<br/>　　紧接着，铺天盖地的攻击，从第四城之中轰击而出，如同天地洪流，浩浩荡荡。
<br/>　　“杀！”
<br/>　　龙羽暴喝，身躯化为了龙族战体，率领众人疯狂还击。
<br/>　　轰！轰！轰隆隆！！
<br/>　　一瞬间，无尽的攻击碰撞，爆响震天，天塌地陷，各种混乱的气流，如同群蛇乱舞。
<br/>　　一众第二城之人，在龙羽的带领之下，挡住了第四城的第一轮攻击，而后纷纷杀进了第四城之中。
<br/>　　第四城之中，人数虽然也很多，有一百零四人之众，但是，却并没有多少太厉害的人物，为首之人乃是天星神朝的无极公子。
<br/>　　第二城的人杀入第四城，大混战彻底的爆发了，爆炸声震耳欲聋，到处都是汹涌的攻击。
<br/>　　不过，龙羽等人虽然人少，但是，由于大多人都比较强大，并没有落入下风，一时之间，两方人马杀的昏天黑地。
<br/>　　不时的有人被重创，仓皇向上方逃穿，逃出大阵笼罩的范围，自行出局。
<br/>　　倒是没有人向城外而逃，因为城池被攻破，城中的阵基若是被摧毁的话，他们就算逃走也要出局了。
<br/>　　不过，陨落的人倒是不多，因为，大家都是天才，都有保命的实力，不敌之下逃走的本领还是有的。
<br/>　　再者，众人也不会下杀手，追杀被重创之人。
<br/>　　两方的战斗极为激烈，甚至，第二城之人还有人反水，反水之人自然是天星神朝之人，他们发现第四城中是无极公子之后，立刻就掉转枪了口。
<br/>　　“苏莫，我们要不要支援龙羽他们，只要少派一些人，就能彻底击败第四城！”公孙长绝面色凝重的问道。
<br/>　　“不用！”苏莫摇了摇头，他们只需要静观其变就好，一旦出手，局势就可能会有大的变化。
<br/>　　苏莫面色平静，遥望着第四城的方向，同时，他也在留意着己方的阵营。
<br/>　　己方阵营之中，有一位天星神朝之人，不过，对方倒是沉的住气，并没有去支援第四城的打算。
<br/>　　战斗在继续，没过多久，双方的战斗到了白热化的程度，第四城的人减少了三分之一。
<br/>　　但是，龙羽所带领的第二城之人，也只剩下了四十余人，大多数受了重创之人，都急速逃回了第二城，也有少数几人被擒住之后直接被扔出了大阵之外，惨遭淘汰。
<br/>　　渐渐的，龙羽等人落在了下风，因为人数太少，终究露出了败势。
<br/>　　无极公子的实力非常强大，龙羽和另外两名天才联手，都无法击败对方。
<br/>　　终于，第二城之中的其他人坐不住了，贺兰殇出城了，带着二十余人，火速的飞向第四城。
<br/>　　有了贺兰殇等人的加入，局势立马反转，第四城之人被杀的节节败退。
<br/>　　“大家准备行动！”苏莫遥望着这一切，朗声说道，他不相信其它城池之人会坐视不理，所以，他们也该准备了。
<br/>　　果然，就在片刻之后，第八城之中，有浩浩荡荡的人群，急速的飞了出来。
<br/>　　这群人足有九十人左右，为首之人，正是披着黑色的披风的古魔太子。
<br/>　　“杀！”
<br/>　　凶厉的吼叫声，从古魔太子的口中响起，一大群人径直向第二城扑了过去。
<br/>　　古魔太子的打算，昭然若揭了，就是要趁着第二城空虚之际，火速击败第二城。
<br/>　　“不好！”贺兰殇和龙羽等人，一见古魔太子出手了，顿时面色一变。
<br/>　　因为古魔太子，可谓是所有人之中，他们最为忌惮的人。
<br/>　　而此刻，古魔太子几乎是倾巢而出了，足足带着九十人。
<br/>　　贺兰殇和龙羽心中清楚，第二城的护城阵法，最多只能挡住古魔太子十个呼吸的时间，一旦阵法被破，第二城危矣。
<br/>　　他们虽然和第三城联盟了，但是，并不绝对信任第三城。
<br/>　　龙羽虽然知道苏莫在第三城，但是，他自身也不想被淘汰出局，因为，苏莫也不一定能得到最后的胜利。
<br/>　　“托住他们！”无极公子见到古魔太子杀向第二城，顿时精神大震，吩咐所有人托住龙羽等第二城之人。
<br/>　　只要他们托住第二城之人，让得龙羽等人无法回援，第二城就会被击溃，然后第二城之人全部都要被淘汰，他们的危机自然也就解除了。
<br/>　　轰轰轰！！
<br/>　　一瞬间，所有第四城之人，纷纷拼命的攻击，企图托住所有第二城之人。
<br/>　　他们的确是达到了目的，在他们猛烈的攻击之下，第二城之人，短时间内难以脱身。
<br/>　　眼见着古魔太子，带着约九十人，即将接近第二城，第三城中众多的人都焦急了起来。
<br/>　　“苏莫，我们出手吧？”龙双双面色焦急的道。
<br/>　　“苏莫，事不宜迟！”公孙长绝同样催促，若是晚了第二城可能就要被灭了。
<br/>　　“我们只要托住古魔太子等人片刻，龙羽和贺兰殇他们必胜！”
<br/>　　“不错，只需要挡住古魔太子他们片刻就行了！”
<br/>　　不少人纷纷开口，他们倒是没有奢望击败古魔太子，只需要拦住片刻，给予龙羽等人争取时间就行。
<br/>　　苏莫面色平静，心念急转，他们第三城若是大举出动，内部空之下，也难免会遭到其他城池的袭杀。
<br/>　　目前，第一城，第五城、第六城、第七城、第九城全部是毫无动静。
<br/>　　“公孙长绝，你带人去袭击第八城！”苏莫没有过多迟疑，立刻吩咐道。
<br/>　　这是围魏救赵，他自己并不准备出战，只要他在城中，第三城基本上就能无忧，而以公孙长绝等人去硬碰古魔太子并不妥当。
<br/>　　“袭击第八城？”公孙长绝闻言愕然，其它人也是一愣，他们没有想到，苏莫不是要派人抵挡古魔太子，而是让人去袭击古魔太子的老巢。
