        “离婚？为什么啊？为什么要跟大少爷离婚呢？大少爷对你那么好，他是真的很爱你啊！”刘婶忽然替傅夜沉感到悲哀与不平。
<br/>
<br/>        “他爱我？他爱的人是苏漫雪，噢，不，是那个女秘书落以柔。反正，他爱的女人很多。刘婶，你就别拿我开玩笑了。”童筱颖觉得自己像是听到了一个一点都不好笑的冷笑话。
<br/>
<br/>        刘婶深沉地叹了口气，将事情的始末，娓娓道来：“苏漫雪拿着大少爷留给你的玉坠顶替了你的身份成为了大少爷的未婚妻这事，大少爷一直心知肚明。大少爷发现我弟弟（刘管家）接错人时，你已经是二少爷的妻子了，大少爷为了化解你和他之间的尴尬，故意和苏漫雪在你和二少爷面前逢场作戏。后来，你和二少爷去了澳洲。这三年的时间里，大少爷对你思念成疾，落下了心痛病。连医生想过很多办法替大少爷治疗，都是束手无策。你说的那个什么落以柔，我也听连医生提起过，连医生说想利用这个女人给大少爷治病。但是，我感觉大少爷对这个女人似乎不来电。这心病果然还是需心药医，自从你来了后，大少爷脸上多了笑容，所以，你才是大少爷的心药啊！你若是要跟大少爷离婚，那岂不是要要了他的命！”

<br/>        “……”童筱颖此刻的心情格外沉重。
<br/>
<br/>        “大少奶奶，你有在听我说话吗？”刘婶滔滔不绝地说完，发现听筒那边没声音了，又问道。
<br/>
<br/>        童筱颖五味陈杂地应了声：“在。刘婶，不打扰你休息了。我先挂电话了。”
<br/>
<br/>        “大少奶奶，你在哪里啊？什么时候回来？”刘婶问道。
<br/>
<br/>        但是童筱颖并未听完刘婶的问话，就直接将电话给挂断了。
<br/>
<br/>        大哥真的如刘婶说的那样，是因为爱她，才落下的心病？
<br/>
<br/>        所以，这就是连华生“仇视”她的原因吗？
<br/>
<br/>        童筱颖此刻心慌得厉害，匍匐在圆木桌上有些不知所措。
<br/>
<br/>        为什么，她从大哥身上看不到他对她的爱？
<br/>
<br/>        滨江城，中心医院VIP病房。
<br/>
<br/>        落以柔坐在了病床边，心疼地看着病床上脸色惨白的傅夜沉。
<br/>
<br/>        她是被连华生叫过来照顾他的，其实，她不需要连华生开口，他病了，她也会主动过来照顾他。
<br/>
<br/>        好端端的一个大男人，为什么会被气得吐血昏迷？
<br/>
<br/>        傅夜沉和童筱颖之间的事情，是她再三逼问连华生，连华生才肯跟她说了实话。
<br/>
<br/>        更让她受打击的是，连华生之所以“改造”她，只是为了给傅夜沉治心病。
<br/>
<br/>        直到此刻，她才知道，原来她是最后那个被蒙在鼓里的人。
<br/>
<br/>        好像全世界都知道他傅夜沉深爱着童筱颖，他爱她爱到病入膏肓，只有她落以柔一个人不知道。
<br/>
<br/>        说到底，是她不敢对他轻易地开口表达自己任何一句暗恋他与爱慕他的话。
<br/>
<br/>        如果，她开口了，他会接受她吗？
<br/>
<br/>        她不确定，也不敢去确定。
<br/>
<br/>        或许，这是她懦弱的一面。
<br/>
<br/>        其实，更多的，她只是卑微地守着这一份爱他的心意，只想与他在公司里每天分担他工作上的负担。
<br/>
<br/>        这是她爱到最卑微的心愿了。
<br/>
<br/>        但是，这样被爱所伤的傅夜沉让落以柔很是心疼，也因此更无法忽视对他的这份爱慕。

<br/>        落以柔的水眸更加放肆地停留在傅夜沉那双紧闭的眼眸上。

<br/>        她知道，他英挺的浓眉下，是一双迷人如黑钻般闪耀的眸子。

<br/>        要是，这双眸子里，能有她的身影该多好！
<br/>
<br/>        落以柔现在在想，如果能这样一辈子守护在他身边，一定是一件非常幸福的事情。
<br/>
<br/>        而她的眼睛里，心里，全都只有他傅夜沉一个人。
<br/>
<br/>        像他傅夜沉这样优秀的男人，不应该为情所伤才对！

<br/>        他是个应该是这个世上让女人去宠护的男人。
