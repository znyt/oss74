　　这老这便是太清老子，青年是上清通天，因在元始之后才化形而出，故称元始为大哥。
<br/>　　老子见到元始，躬身道：“见过大兄！”
<br/>　　之后又问道：“大兄此行可顺利？”
<br/>　　“哈哈，以本座如今的修为，天下之大，随处可去，你们就不用担心我啦，到是你们——”元始在两人身上看了看，接着说道：
<br/>　　“多年不见，你们的修为提升的很快啊，都到了大罗金仙后期了，特别是老子，你快要到大罗圆满了。”
<br/>　　“这点修为，比起大兄可是远远不如啊。”青年脸上掩饰不住的喜悦，就好像是大兄的夸赞令他非常高兴一般。
<br/>　　“让大兄见笑了。”老子面无比表情道。
<br/>　　“无妨，自家兄弟。”元始一挥手，毫不在意道。接着又说道：
<br/>　　“来！给你们看看，为兄给你们带来了什么？”说着，元始取出两座十一品净世白莲。
<br/>　　“大兄，这——这是白莲？！”
<br/>　　“嗯，不错，这是十一品净世白莲，上品先天灵宝，既能驱除心中杂念，助人悟道，又能防御攻击，而且防御力也十分惊人。来，一人一座。”
<br/>　　说完，元始便将两座白莲扔给通天与老子。
<br/>　　“你们快快炼化，为兄给你们护法！”
<br/>　　“好，多谢大兄！”通天毫不客气地收下，在他看来，既然是自家兄长给的东西，那就没必要拒绝，说多了伤感情。
<br/>　　“多谢大兄。”对比通天，老子就要含蓄一些了，尽管这是大哥给的东西，但也有所迟疑。
<br/>　　“来，赶紧炼化，大劫已经开始了，在大劫之中，能多增加一分实力就多增加一分。我帮你们炼化。”
<br/>　　“那……那就麻烦大兄了。”老子也不再矫情，接受了元始的馈赠。
<br/>　　两人盘坐于蒲团之上，元始直接用大道之力加持于身，就像炼器一般来助他们炼化净世白莲。法力汩汩，如同大海无垠，两人最直观的感受就是太强了。
<br/>　　通天内心倒是没有多少心思，大哥法力高强，他很是高兴，也不会嫉妒什么的。但，老子内心却是，大兄的法力到底有多强？他到底是何等实力，我距离他到底有多远？
<br/>　　元始并没有多管他们的心思，心是他们的，他们爱怎么想就怎么想，只要不对他起什么不好的心思，就行了。要真是以后兄弟反目，他也绝不会手下留情。
<br/>　　有着元始的帮助，两人仅仅用了两百年时间，就将上品灵宝十一品净世白莲炼化。炼化之后，两人都感受到了其中的好处，都兴奋不已。
<br/>　　“好了，既然都炼化了，那么在这大劫之中你们也有一份保障了。不至于遇到危险无力反抗。”持续两百年的法力输出，元始并没有感到丝毫不适，面色依旧，这点法力对于他来说，就是九牛一毛。
<br/>　　“多谢大兄！”两人再次感谢道。
<br/>　　“行了，一家兄弟不说两家话。”元始摆摆手道。
<br/>　　“大兄，你说的大劫是——”老子注意到元始口中的大劫，他猜测这与这段时间天地间煞气增多的原因有关。
<br/>　　元始知道老子看似无为，实际上他也很心细，观察到了天地的变化，于是对着两人叮嘱道：
<br/>　　“不错，你们注意到这天地间煞气的增多，有时甚至会蒙蔽他人的神志，叫人胡乱杀伐，这就是天地大劫的原因。你们之前一直在这昆仑之中，并没有出去洪荒之中走动过，故而，也并不知道，前段时间，洪荒三大种族龙族、麒麟族、凤凰族大战，争夺天地霸主之位，将洪荒闹得是乌烟瘴气。
<br/>　　大战厮杀范围极广，上道九天，下到十地，牵扯众多，高手如云，大神通大法力之辈更是数不胜数，一不小心牵入其中，就会身死道消。故而，我让你们好好修炼，不要胡乱走动。”
<br/>　　“那，我等如今意欲何为？”通天道。
<br/>　　“想必大兄早有想法。”老子淡言道。
<br/>　　“不错，本座这些年游历洪荒，也发现了几处仙家福地，吾此次前来，一是回来看看你们，而是来引你们前去那里静心修炼。”元始道。
<br/>　　“天地大劫，到哪儿都一样吧。”
<br/>　　“不错，大劫之中，哪里都一样。不过，吾找到的那个地方，有遮天大阵，能够遮掩天机，你们到了那里，就能静心修炼，不会受到劫煞的影响。等到了大劫一过，就是我们名动洪荒之时。”
<br/>　　“那——大兄，此地又将如何？”通天看着这昆仑神山，有些不舍道，毕竟此处他们从开天之初就住到现在，整整数千元会。
<br/>　　“昆仑乃洪荒第二神山，此处龙脉众多，呈拱龙之势，仙根灵草遍布，神泉仙气繁多，是不可多得的地方，我们也不能放弃，以后此处我们还会回来。
<br/>　　既然如此，那——我便在此布下大阵守护，再立碑警告。有此，此处无忧矣！”元始看着这昆仑神山淡淡说道。
<br/>　　昆仑元始并不想舍弃，毕竟后世昆仑可是众神之乡，洪荒第一神山不周山倒塌后，昆仑就是洪荒第一神山，有此神山，元始又岂会轻易舍弃，他甚至只将永恒岛当成一处行宫，而日后的根基，还是要回到昆仑来发展。
<br/>　　“好，既然如此，那我们就走吧，我还没走出过昆仑呢！”通天有些激动道。
<br/>　　“贫道听大兄的。”老子也点头同意道。
<br/>　　“好，既然你俩都同意了，那本座就在此布下大阵，立下警示。”元始道。
<br/>　　之后，元始径直来到昆仑神山之上，凭立虚空，往下看去，昆仑地势尽收眼底，要布置什么阵法，心中已有自己的想法。
<br/>　　元始取出器材，一枚枚的炼化，作为阵脚，之后再将这些阵脚一枚枚地扔到山下，阵脚入山即没，却是遁入山下地脉之中。
<br/>　　“九龙锁空阵，给本座起！”元始手捏印诀，虚空震动，九龙腾空，相互交织，封锁上空。
<br/>　　元始不停，依旧布置大阵。
<br/>　　“玄元封天阵，起——”
<br/>　　“梦魇惑心阵，起——”
<br/>　　“阴阳两仪阵，起——”
<br/>　　“五行神雷阵，起——”
<br/>　　“冰火绞龙杀阵，起——”
<br/>　　……
<br/>　　一口气之下，元始布下十数个大阵，既有防护大阵，也有杀敌大阵，还有迷惑他人的大阵，各种大阵，相互配合，层层包围，元始不信有人见此大阵，还会不识趣地前来捣乱。
<br/>　　将所有大阵隐于虚空，元始就落了下来，之后，元始伸手一抓，便将昆仑地底的一块赤色神材抓到手中，淡淡的声音传出：
<br/>　　“魔血赤金，还算不错，就你了。”
<br/>　　寄出神火，片刻时间，元始便将手中的魔血赤金炼化成一块方形大碑。之后，他在碑上刻上禁制铭文，大碑没有别的作用，就有一个优点，那就是牢固。就算准圣后期的强者，也休想一击击碎它。
<br/>　　最后，元始刻下警示：
<br/>　　“三清圣地，擅闯者死！”
<br/>(本章完)
