　　朱博文和白玲两人都为之一愣，
<br/>　　继而，
<br/>　　朱博文忍不住笑道：
<br/>　　“若曦，你不是在说笑吧？我调查过那小子的背景，他不过就是来自一个普通家庭而已，除了人长得帅点，根本没有其他能够称道的地方！”
<br/>　　“他和我们这个层次的人，本就不在同一个世界！”
<br/>　　“我不知道他给你灌了什么迷魂汤，但是像这样的小瘪三，哪里配得上你？”
<br/>　　萧若曦淡淡一笑，像是在自言自语，轻声道：
<br/>　　“你对他根本一无所知……”
<br/>　　朱博文眉头一挑，
<br/>　　“你这话是什么意思？”
<br/>　　萧若曦才懒得向朱博文解释，抬起手腕看一眼手臂，淡淡道：
<br/>　　“现在已经过去了5分钟，朱博文，你还有十分钟的时间，你到底想要做什么，还是直说了吧！”
<br/>　　朱博文眉头一凝，深吸了一口气，缓缓道：
<br/>　　“好！那我就实话告诉你吧！”
<br/>　　“我已经让人打电话通知那小子，告诉他你现在在我这里，而且派了一帮人在一楼把守，他要是真的爱你，就应该不顾一切的冲过来，站到我的面前！”
<br/>　　“否则也就说明，他并没有那么在乎你！”
<br/>　　一旁的白玲听到这话，顿时急了，
<br/>　　“朱学长，你答应过我，绝对不会为难若曦的话，你怎么可以做这种事情？你这是绑架！”
<br/>　　朱博文淡淡一笑，
<br/>　　“白玲学妹，我并没有为难若曦啊！我这是在帮她，帮她考验她的男朋友，到底有没有那么爱她？”
<br/>　　说到这里，朱博文又望向萧若曦，
<br/>　　在他看来，萧若曦肯定会很生气，然后拂袖而去，不过他早已经让人从外面反锁，萧若曦想走也走不了。
<br/>　　然而，
<br/>　　让朱博文没有想到的是
<br/>　　萧若曦竟然十分平静，甚至脸上还挂着一抹嘲讽之色，淡淡道：
<br/>　　“朱博文，如果我是你的，我就绝对不会这么做！你根本不知道，自己所面对的，是怎么的存在！”
<br/>　　朱博文听到这话，顿时一愣，显然没有明白萧若曦这话的意思。
<br/>　　……
<br/>　　另外一边，
<br/>　　正在军训中的叶尘，手机忽然响起。
<br/>　　他其实早已经通过神念，将私人会所中的情况尽收眼底。
<br/>　　“谁的手机？我有没有说过，军训期间，手机一律关机！是谁的手机在响，给我站出来！”
<br/>　　负责军训生物学院的教官，顿时怒吼了起来。
<br/>　　叶尘一脸平淡的从人群中走出，拿出手机接通，
<br/>　　“喂，你是叶尘吧？萧若曦现在在我的手上，限你十分钟之内，赶到雅尚娱乐会所，否则后果自负！”
<br/>　　对方撂下一句话，便直接挂断了电话。
<br/>　　那名教官，见叶尘竟然敢当着他的面接电话，顿时气得肺都要炸了，
<br/>　　“是谁让你接电话的？你知不知自己现在在干嘛？无视军训纪律！给我罚做一百个俯卧撑！”
<br/>　　众人看到叶尘被教官训斥惩罚，顿时纷纷露出幸灾乐祸的表情。
<br/>　　叶尘却仿佛没有听到一般，
<br/>　　“我有事情要出去一趟，请个假！”
<br/>　　说完，没等教官开口同意，叶尘直接迈步朝外面走去。
<br/>　　所有人顿时全看呆了！
<br/>　　“握草！不是吧，这家伙是要逆天吗？”
<br/>　　“连教官都敢不鸟，是谁给他这么大的胆子？”
<br/>　　“听说计算机学院有个学生，和教官顶撞，结果被罚站三个小时！看着吧，这下有他受的了！”
<br/>　　……
<br/>　　众人顿时纷纷幸灾乐祸的嘲讽起来。
<br/>　　那教官更是气的，连杀人的心都有了，
<br/>　　“谁允许你走的？你给我站住！”
<br/>　　叶尘却连头都没有回，继续朝操场外走去，就连其他学院的学生，也被这边的情况所吸引，纷纷扭头望了过来。
<br/>　　那教官见叶尘，如此不把他放在眼里，顿时便再也忍无可忍，当即一个健步上前，一把按住了叶尘的肩头，
<br/>　　“老子让你站住，你特么是聋了吗？”
<br/>　　教官一声暴喝，同时用力往后一拉，本以为可以轻松将这个体格瘦削的少年拽回来，可是却万万没有想到，对方如同钉在地上一般，居然纹丝未动！
<br/>　　“怎么会？”
<br/>　　教官顿时大惊失色，
<br/>　　他的力量在整个连队里，都是数一数二的，却没想到竟然连一个大学生都拉不动。
<br/>　　“我有事要去处理，没时间和你废话！”
<br/>　　说着，叶尘肩头微微一震，后者顿时感觉整个人，如同触电了一般，全身都变得麻痹起来，一时间竟然无法动弹，甚至连开口说话都做不到！
<br/>　　教官心头惊骇万分，可是整个人，偏偏如同被施了定身术，丝毫动弹不得，只能眼睁睁的看着叶尘渐行渐远。
<br/>　　一直到，叶尘的声音，彻底消失在众人的视野之中，教官这才重新恢复了自由，满脸的骇然之色。
<br/>　　“这家伙……难道会妖术不成？”
<br/>　　而落在其他众人眼中，叶尘居然就这么堂而皇之的离开了，也是个个一脸懵逼。
<br/>　　“什么情况？”
<br/>　　“教官居然就这么放他走了？”
<br/>　　“这……太不可思议了吧？”
<br/>　　……
<br/>　　唯独杨之凡眉头紧皱，似乎在思索着什么。
<br/>　　如果说，前几天晚上，在酒店发生的怪事，是叶尘和那帮混混串通，合伙演的一场好戏。
<br/>　　那今天这一幕，又该如何解释？
<br/>　　难道他连教官也收买了？
<br/>　　杨之凡百思不得其解。
<br/>　　他所接触的层面，远比普通人要高，自然知道这个世上，存在着很多不可思议的事情，很多不可思议的人！
<br/>　　“叶尘啊叶尘，我现在对你还真是越来越好奇了！”
<br/>　　……
<br/>　　却说叶尘离开操场之后，直接身体一晃，已经出了天海大学，几乎一转眼的功夫，就来到了雅尚娱乐会所。

<br/>　　叶尘脚下步子不停，直接随手一挥，
<br/>　　“一群蝼蚁！”
<br/>　　砰砰砰！
<br/>　　十几人顿时全部飞了出去。

<br/>　　二楼上的朱博文，听到下面的动静，顿时露出一抹得意的笑容，
<br/>　　“若曦，这小子如果真的愿意自废一条腿，那就证明他对你是真心的，我发誓以后绝对不会再纠缠你！”
<br/>　　朱博文这话才刚刚落下，
<br/>　　彭！
<br/>　　一声巨响，
<br/>　　房门上直接出现了一个大洞，继而一道人影，从门洞中缓步走了进来，同时一道淡漠的声音响起，
<br/>　　“就是你，要废我一条腿？”
