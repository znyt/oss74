        傅夜沉突然觉得，今天这个小女人很奇怪。
<br/>
<br/>        被自己所爱的男人二度抛弃，她不但没像第一次被抛弃时的那样哭得撕心裂肺，也没有郁郁寡欢，反倒是非常淡定地坐在他面前，一本正经地在吃饭。
<br/>
<br/>        他的这个小女人心里到底是怎么想的？
<br/>
<br/>        都说“女人心海底针”，他真有些看不透她的心思。
<br/>
<br/>        童筱颖把饭吃完后，放下碗筷时，傅夜沉连忙将目光从她的身上收回。
<br/>
<br/>        她将碗筷和饭盒收好，在茶几上的纸巾盒里抽了张纸巾擦拭嘴上的油渍。
<br/>
<br/>        “那么，大哥，你好好休息，我就先回去了。明天再来看你。”童筱颖提起饭盒，从沙发上起身，将擦脏了的纸巾扔到了茶几旁的垃圾桶中，淡定自若地说道。
<br/>
<br/>        傅夜沉倚着床头板，下意识地从一旁柜子上拿起一本财经杂志来，随手翻着，头也不抬地应了声：“好。”
<br/>
<br/>        童筱颖随后去了一趟卫生间，出来的时候见阳台上晾着他洗干净了的衣服，心想着十有八九是落以柔做的，于是情不自禁地笑了笑。
<br/>
<br/>        看来，她的二婚老公是真的被其他女人给惦记上了。
<br/>
<br/>        她倒是一点都不吃醋，反倒有点儿小开心，至少证明，他傅夜沉还是有魅力、有优点，能俘获其他女人的芳心。
<br/>
<br/>        但是，话说回来，他最终会不会也像昼景那样，出轨呢？
<br/>
<br/>        童筱颖觉得自己的感情路走得很悲催，前男友劈腿、前夫出轨，二婚老公又被其他女人惦记着。
<br/>
<br/>        刘婶说，大哥喜欢的人是她，算不算是上天对她的二次眷顾？
<br/>
<br/>        前两次的感情失败，让她心里多了一丝芥蒂。
<br/>
<br/>        为了不让自己在感情上再次受伤，这次她决定让自己时刻保持清醒与理智，不会再轻易交付自己的真心。
<br/>
<br/>        既然他明知道她不爱他，还不愿意跟她离婚，想要这样跟她耗一辈子，那么她就索性安安分分地当他一辈子的“挂名太太”好了。
<br/>
<br/>        更何况，与其嫁给自己喜欢的人，不如嫁给喜欢自己的人。
<br/>
<br/>        至少这样，等哪天，他若是不喜欢她了，想通了要跟她离婚了，到时候，她还可以洒脱地庆祝自己恢复单身。
<br/>
<br/>        童筱颖看着傅夜沉，走到病床边，夺走了他手中的财经杂志后，摆正了放回他的手中，忍俊不禁道：“大哥，你真厉害，书本都能倒着看。”
<br/>
<br/>        “……”傅夜沉汗颜。
<br/>
<br/>        童筱颖挥了挥手，莞尔一笑：“明天见。”
<br/>
<br/>        语毕，她转身离开了病房。
<br/>
<br/>        童筱颖回到私宅里时，刘婶把汤端了出来，结果，她喊刘婶坐下来，一起喝这牡蛎鸡汤。
<br/>
<br/>        “大少奶奶，你不打算给大少爷送去了吗？”刘婶纳闷地问道。
<br/>
<br/>        童筱颖坐在餐桌前，手里拿着瓷勺，舀起一勺汤，一边吹了吹，一边淡定地回答道：“他已经吃过了，不需要我再送饭去了。这汤不吃浪费了，我们俩也来补补身子。”
<br/>
<br/>        “你是不是在医院里又跟大少爷吵架呢？”刘婶试探性地问。
<br/>
<br/>        童筱颖小啜了一口汤，微笑着回答道：“我没跟他吵架，只是感觉，他对我好像生疏了点。”
<br/>
<br/>        “好端端的，怎么会对你生疏了呢？”刘婶感到有些纳闷，不应该是这样吧！
<br/>
<br/>        以大少爷那性格，看到大少奶奶回来后，肯定会高兴得药到病除。
<br/>
<br/>        “没关系，我跟他来日方长。”童筱颖会心一笑。

<br/>        还说他喜欢她，看看他今天对她那态度，跟见了仇人似的。

<br/>        不过话说回来，他对她那种态度，也算是合乎情理。

<br/>        其实，她真的好像明明什么也没做，却总是被安上那些莫须有的“罪名”。

<br/>        她也只是听连华生碎碎念叨过，说大少奶奶跟二少爷“私奔”了，至于是不是真的，她就不得而知了。

<br/>        “他误会我什么？我需要解释什么？”童筱颖一脸诧异地反问，“难道是向他解释我和昼景的事情吗？但是，已经没什么好解释的了。他都看见了，我想解释也解释不清楚了！”
