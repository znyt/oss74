        &lt;h3&gt;第三卷第三百零八章  成长中的陈虞落妍&lt;/h3&gt;
<br/>
<br/>        陈子昂从西城回来直接就投入到工作中。
<br/>
<br/>        她的绝密铁三角已经形成，呼延寰宇和赵绾绾已经为他搜集了很多重要的信息。
<br/>
<br/>        宁雅集合大家也已经将所有的工作做到位了。
<br/>
<br/>        她回到家的时候，守中带着陈虞和落妍在院子里锻炼身体，守中对俩个孩子的要求比较严格。
<br/>
<br/>        守中见到陈子昂回来了，并没有让陈虞和落妍停下来，他走到陈子昂的身边：“陈姐，你回来了？”
<br/>
<br/>        “守中，这几天辛苦你了。”陈子昂笑着说。
<br/>
<br/>        “不辛苦，和这俩个调皮鬼在一起，很幸福。”守中开心一笑，虽然陈虞和落妍还只是孩子，他和这俩个孩子倒是很能相处得来。
<br/>
<br/>        他们活泼可爱但又不失优雅和大体，他们有孩子的淘气和顽皮也有成人的思维和风度，和他们在一起比和有些成年人在一起更让人感觉舒坦。
<br/>
<br/>        “哈哈哈，很幸福，好吧，我也很幸福，我看到你们的杰作了。”在进门的时候，陈子昂已经看到了他们的系统正式投入使用了。
<br/>
<br/>        她刚到门口，远红外就检测到了她的信息，门就自动打开了。
<br/>
<br/>        “怎么样？还可以吗？”守中得意的一笑。
<br/>
<br/>        “是非常可以。”陈子昂也很满意，回西城的这一段时间，守中和陈虞落妍都没有闲着，尤其是守中的作用最大，陈虞和落妍只是孩子。
<br/>
<br/>        “他们俩个人的功劳最大。”守中看着正在锻炼的俩个孩子，眼里是心疼，这俩个孩子完全和大人一样，这一段时间可以说是没日没夜的修改，矫正，软件硬件都进行手把手的调试。
<br/>
<br/>        陈子昂微微一笑，陈虞和落妍心里接受能力是很强的，自从新年张函一家人来家里之后，孩子们就变了，虽然这种转变让她感觉很心疼，但是人都是要成长的，迟一天早一天有什么区别呢？
<br/>
<br/>        这一段时间张函也没有来找孩子们，孩子们心里肯定会很难过吧。
<br/>
<br/>        “还有多长时间就结束？”陈子昂看着俩个正在腾挪跳跃的孩子问守中。
<br/>
<br/>        “还有二十分钟！”守中看着俩个孩子：“他们俩个人很刻苦，承受能力超出了我的想象。”
<br/>
<br/>        “身体还吃得消吧？”
<br/>
<br/>        “他们已经超越了极限值，后期会越来越轻松的，不过再过一段时间，身体会出现疲劳期，扛过疲劳期身体又会出现转折点。”守中微微一笑。
<br/>
<br/>        “嗯，他们现在正是成长期，耐受力应该不错，这俩个孩子的意志力我还是很相信的。从小就被我拽着跑步。”陈子昂看着陈虞和落妍，心里微暖。
<br/>
<br/>        “要不，你回去歇着，等他们结束了我们就回去了。”守中看着拉着皮箱的陈子昂说。
<br/>
<br/>        “不用，我看着他们，等他们结束了我们一起回，你不用管我，去指导他们吧。”
<br/>
<br/>        “那我过去了。”
<br/>
<br/>        陈虞和落妍不是没有看到妈妈回来了，他们心里很开心，每次妈妈回来他们都会迎上去，可是这一次不行，因为守中老师说了，在他们锻炼的时候一定要心无旁鹭，还有他们希望妈妈能看到他们的成长。
<br/>
<br/>        “哥哥，老妈儿回来了，你开不开心？”落妍边和哥哥对练着边问，虽然在说话，可是身体一刻也没有闲着。
<br/>
<br/>        “傻瓜才问这样幼稚的问题，我拜托你用点心好不好？你难道要让妈妈看着你被我揍得满地找牙的样子？”陈虞对于妹妹问这样的问题感觉不满。
<br/>
<br/>        “就你这样子还能揍得我满地找牙？我落妍展翅。”
<br/>
<br/>        “我陈虞掏心。”陈虞直接一拳朝妹妹的胸前挥去。
<br/>
<br/>        “啊啊啊，你讨厌，我是你的妹妹。”落妍立马哭唧唧的表情。
<br/>
<br/>        陈虞微微一滞，落妍闪身躲过陈虞的一拳，一个手刀落在陈虞的脖子上。
<br/>
<br/>        “陈虞，你这软弱的小心脏该改一改了。今天是我，要是别人你就躺尸了。”落妍语气冰冷，没有一点妹妹的样子。
<br/>
<br/>        “可是你都哭了说你是我妹妹。”陈虞表示不服气，要是对手是别人他才不会上当受骗呢。
<br/>
<br/>        每次他都会败给：“我是你妹妹”的这一句话。
<br/>
<br/>        “那要是你的对手和我长得一样，也哭着说，我是你妹妹，你是不是就上当受骗了？你可长点心吧，别让老妈和老妹替你担心了。”落妍拍拍手，这一局她赢了。
<br/>
<br/>        陈虞有些不服气，但是妹妹的话让他无从反驳，她说得很有道理。
<br/>
<br/>        “怎么了，垂头丧气的？”守中看着陈虞笑着问，俩个孩子其实水平相当，不过落妍毕竟是女孩子，多少还是差别于男孩子，只是落妍古灵精怪，常常让陈虞会落下风。
<br/>
<br/>        “她使诈！”陈虞看了一眼站在远处的妈妈，他还说长大了要保护妈妈呢，现在连妹妹都打不过，怎么去保护妈妈？
<br/>
<br/>        “兵不厌诈哦。你还是的长点心。”守中笑着看着有些气馁的陈虞。
<br/>
<br/>        “好的吧。”陈虞偷偷看了妈妈一眼，发现妈妈在看着他们，竟然觉得眼眶一热，想哭了。
<br/>
<br/>        “来，今天给你们俩人一个表现的机会，拿出你们的真本事来，让妈妈看看，你们俩个一起来对付我，你们要把我当做你们的敌人，你们要想着这是你们保命的机会，来吧！”守中话刚说完，就动手了，连反应都不留给陈虞和落妍。
<br/>
<br/>        陈虞和落妍反应也快，在守中出手的同时，俩个孩子也出手了，一对二，师徒三个人就在院子里你追我赶，看得陈子昂眼花缭乱，给她上演了一场武打片。
<br/>
<br/>        陈子昂站得有点累了，索性坐在了行李箱上面，津津有味的欣赏着三个人在院子里展开的一场打斗，她对这些是不懂的，只是觉得很好看，看着孩子灵活的身影，矫健的身姿，陈子昂满心欢喜。
<br/>
<br/>        三个人来来去去的大概有半个小时，守中就结束了战斗，俩个孩子也让他有些吃力，主要是他们是双胞胎，似乎心意相通，俩个人的想法出奇的一致，让他有些防不胜防。
<br/>
<br/>        “妈妈。”一结束战斗，俩个孩子就冲过来了，一如既往地一人吊着一只胳膊。
<br/>
<br/>        陈子昂看着俩个人那湿乎乎的头发，红扑扑的脸蛋，心里是既满足又心疼。
<br/>
<br/>        “累不累？”陈子昂用额头各碰了一下孩子们，笑咪咪的问。
<br/>
<br/>        “不累，妈妈你累吗？”落妍双眼冒星星的看着妈妈。
<br/>
<br/>        “妈妈，你都在这里呆了好久了，你肯定累了。”陈虞小声的说。
<br/>
<br/>        “我不累，你们俩人好厉害哦，将你们的师傅追赶的屁滚尿流。”陈子昂抑制不住的想笑。
<br/>
<br/>        “我是让着他们好不好？”守中翻了个白眼，没见过这样夸自己的孩子并且这样损孩子老师的人。
<br/>
<br/>        “让不让他们我不知道，反正我看得出来你很狼狈，哈哈哈哈。走啰，回家啰。”陈子昂拍拍手，拉着俩个孩子，将行李箱扔给了守中，回头冲守中一笑。
