        刘婶见苏漫雪态度变好，人也跟着温柔体贴了许多，还在私底下替她在傅夜沉的面前说起了好话。
<br/>
<br/>        “大少爷，童筱颖小姐，噢，不，是二少奶奶既然已经不可能跟您在一起了。如今，大少奶奶又知错能改，愿意为了您改变自己的性子，努力变成您喜欢的样子，您何不考虑考虑一下大少奶奶呢？”刘婶苦口婆心地劝。
<br/>
<br/>        傅夜沉却仍旧不以为然。
<br/>
<br/>        刘婶只看到了苏漫雪的表面，从未去真正了解苏漫雪的内心。
<br/>
<br/>        苏漫雪为了什么才会做如此改变，他傅夜沉的心里比谁都清楚。
<br/>
<br/>        俗话说，有钱能使鬼推磨。到了苏漫雪那儿，有钱能使她心甘情愿地去出卖自己的灵魂。
<br/>
<br/>        而在苏漫雪的心里，确实如此。
<br/>
<br/>        这世上没有什么东西能比钱来得更实在，苏漫雪贪财，有了钱，能给她想要的一切物质。
<br/>
<br/>        她苏漫雪可以不爱任何男人，但唯独做不到不去爱钱。
<br/>
<br/>        另一边，童筱颖整理了行李箱，上车后，坐在副驾驶座上。
<br/>
<br/>        当她看到驾驶座上的傅昼景从皮夹里抽出一叠崭新的六百六十六块钱塞入到一个祝寿红包里，不禁下意识地问道：“要不要问问大哥大嫂，看看他们包多少？”
<br/>
<br/>        “他们一定会包六万六，我可包不起这么多钱。”傅昼景苦笑。
<br/>
<br/>        童筱颖咽了咽口水，便不再多问了。
<br/>
<br/>        果然，有钱人的世界里，跟他们这平民的世界完全不同。
<br/>
<br/>        其实，傅昼景本来不打算去的，但是爷爷说，既然他已经领了结婚证，就得带着新媳妇去见亲戚。
<br/>
<br/>        因为两人未办酒席，所以，筱颖的身份，只能是他傅昼景的未婚妻。
<br/>
<br/>        在去临海城的途中，傅昼景一边开车，一边跟童筱颖讲了一些关于他们傅家的事情。
<br/>
<br/>        傅昼景其实知道的并不多，所以跟童筱颖说得也不多。
<br/>
<br/>        童筱颖只知道，爷爷傅炎廷是他们那一辈的嫡长子，所以，爷爷继承了傅家大庄园那幢古宅和傅氏集团。
<br/>
<br/>        到了他们这一辈，大哥傅夜沉是嫡长子，所以，傅家大庄园和傅氏集团自然是由大哥傅夜沉全部继承。
<br/>
<br/>        “如果大哥和大嫂生的只有女儿，没有儿子，怎么办？”童筱颖忍不住好奇地问。
<br/>
<br/>        傅昼景忍俊不禁道：“你这是诅咒大哥生不出儿子吗？”
<br/>
<br/>        “哪有，就是好奇，问问嘛！”童筱颖赧然一笑。
<br/>
<br/>        傅昼景笑着回答道：“嫡长女也可以继承傅家的一切，只不过，到时候嫡长女的孩子必须姓‘冷’，这是傅家祖辈订下的规矩！”
<br/>
<br/>        “也就是说，大哥的女儿到时候要招婿？”童筱颖似懂非懂地点了点头。
<br/>
<br/>        “不过，傅家每代都是男丁旺盛，所以，大哥不会没有儿子的。”傅昼景接着说道。
<br/>
<br/>        童筱颖会心一笑，一手托腮，靠在车窗上，突然很庆幸地说道：“还好，小卫你不用担心这些。继承那么大的家产，其实心里还是会有压力的吧！而且，万一自己的下一代，为了争夺家产，手足相残，岂不是令人痛惜。”
<br/>
<br/>        “所以，筱颍也觉得，傅家大庄园和傅氏集团，就应该都是大哥的吗？”傅昼景的笑容渐敛，脸部微微僵硬，说话时的黑眸，意味不明地深邃了几分。
<br/>
<br/>        童筱颖并未察觉到傅昼景的异样，而是自顾自地回答道：“这不是傅家祖上订下的规矩吗？既然傅家的规矩不能破，那么大哥继承傅家大庄园和傅氏集团，就无可非议了。”
<br/>
<br/>        此时此刻，傅昼景不再吭声了。
<br/>
<br/>        或许，这就是筱颖和思妍的区别。
<br/>
<br/>        季思妍也知道傅家的规矩，但是季思妍却鼓励他傅昼景翻身为王。
<br/>
<br/>        而童筱颖呢？
<br/>
<br/>        她却循规蹈矩，默认傅家祖上所安排的一切。
<br/>
<br/>        果然，筱颍只是一个居家型的小女人，而懂他心思的人，只有季思妍。
<br/>
<br/>        殊不知，童筱颖只是认为，既然不是自己的东西，就没必要去强取。
<br/>
<br/>        如果有家产可以继承，这当然是好事，但比起继承家产，童筱颖更向往用自己的双手去创造属于自己的家产。
<br/>
<br/>        或许，这期间会很累，但一定过得很充实。
<br/>
<br/>        可在傅昼景心里，并不这么想。
<br/>
<br/>        既然都是傅家的儿子，那么傅氏集团，应该有他傅昼景的一份，凭什么全是大哥傅夜沉的？
<br/>
<br/>        不，傅昼景想要的，是傅氏集团的全部。
<br/>
<br/>        他的心思，筱颍永远都不可能懂。
<br/>
<br/>        从小到大，大哥傅夜沉的光辉总是将他傅昼景没入无限的阴影之中。
<br/>
<br/>        不管是傅家的直系亲戚还是旁系亲戚，他们那群人的心里眼里，全都只有大哥傅夜沉。
