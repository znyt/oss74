　　话音落下，什造的立即动了起来，那如同闪电一般的速度，让刘哔看的都眼花缭乱。
<br/>　　喰种自然没有他这样的反应速度，零点几秒的时间里，身上就被密密麻麻的黑色匕首插满，随后，整个胸膛都被什造的拳头洞穿。
<br/>　　一声尖锐的惨叫声响起，什造收回拳头，甩掉了上面恶心的血肉，随后一把掐住了喰种的咽喉。
<br/>　　虽然只是刚刚开始觉醒，但最强人类的基因，也给什造带来了极为强大的力量增幅，喰种在什造手中死命挣扎，却无论如何都挣脱不了。
<br/>　　“好孩子得分，加十分。”用自己才能听到的语气低喃一声，什造自嘲般的一笑，扭断了喰种的脖子。
<br/>　　血液顺着她雪白的臂膀，一路流到衣襟处，但什造似乎没有任何感觉一般，目光稍稍有些呆滞。
<br/>　　喰种的四肢无力垂下，失去主人的赫子还在不断蠕动着，似乎想要重新找到一个归宿一般，刘哔嘴角微微上扬，抬脚走了过去。
<br/>　　“什造，今天可真棒呢。”伏在什造诱人的脖子上，刘哔轻轻吹了一口气，将嘴唇贴了上去。
<br/>　　这么美妙的身体，他真的，好想把它吃掉啊。
<br/>　　听到耳边的低喃声，什造小小的身体一抖，脸上露出一抹痛楚，她想到了一些不好的记忆。
<br/>　　空气中血肉的香气，将什造身上那股令人烦躁的清香味盖住，也彻底勾起了刘哔心底的食欲，一丝晶莹的口水，从他嘴边滑落，滴在什造细腻的肌肤上。
<br/>　　一抹淡淡的猩红，自眼眸开始逐渐覆盖上整个眼眶。
<br/>　　突然，像是感应到什么，刘哔闭上了呼吸，收起了已经暴露出来的獠牙。
<br/>　　眼中的红色逐渐褪去，似乎是为了隐藏什么，他伸手搂住了什造柔软的身体。
<br/>　　有人来了，很强大。
<br/>　　“哟，小家伙们，在我们巡视的区域杀喰种，似乎逾越了规矩吧。”不远处的麦田中，两个手提银色密码箱的男子，从土路上走了出来。
<br/>　　这两人，一个很是苍老，一个正值青年，虽然穿着一样的衣服，但两人的容貌气质却有天壤之别。
<br/>　　那个面容苍老的，背后像是长了一个瘤子，整个人都夸张的驼着，长相也是十分怪异，最让人深刻的，是他的眼睛。
<br/>　　一只如铜铃般大，一只却只有绿豆那般大小，第一眼看上去，还有些喜感。
<br/>　　可即便是长得这么奇怪，这个老人也不敢让人有一点轻视之心，就算他只是随便站在那里，却都如同一只蛰伏的野兽一般。
<br/>　　那个年龄稍小一些的，长得倒是英武，约有一米七八的个头，这在岛国并不常见，甚至可以说十分稀有了。
<br/>　　虽然他身上的气质并不如老人那般具有压迫感，可依旧给人一种厚重的感觉，同样不敢让人有丝毫轻视。
<br/>　　他们给刘哔带来的感觉，比他在《僵尸先生》位面里面碰见的‘鬼鹰’还要强大。
<br/>　　“哦，是真户上等和亚门二等啊，我们只是碰巧遇到这个喰种，就顺手解决了。”从茫然中回过神来，什造从刘哔的环抱中仰过头来，对两人眨了眨眼睛，
<br/>　　哎呀，真是好麻烦呢，不就是杀个喰种吗，这些讨厌的人，又来管她了。
<br/>　　但还是要听话的吧，否则的话，筱原大叔那里可不好交代呢。
<br/>　　“是这样吗，这个喰种我们已经盯了很久了，正准备今天动手，却被你们给杀了。”真户脸上露出一个难看的笑容，继续说道：“不过，杀了就好，尸体我就帮你们带回去了，回头你们来找我拿库克因。”
<br/>　　说罢，真户拿过亚门递过来的一只特制麻袋，将地上的喰种装起，用随身携带的手机拍了一些照片，随手写了一些什么。
<br/>　　“好了，你们和我一起回去吧，筱原特等听说你不见了，现在可是很着急呢。”咧了咧嘴，真户扛起地上的麻袋，对什造说道。
<br/>　　不疑有它，什造‘哦’了一声，和刘哔一起跟在真户后面，上了一辆挂着CCG专用牌照的轿车。
<br/>　　被破坏了进食，刘哔的心情有些烦躁，但这两人都不好惹，他也没办法做些什么。
<br/>　　如果自己的身份暴露出去，引来了整个CCG的追杀，那可不是件多好的事情。
<br/>　　算了，反正人在这里，他没必要急于一时，再重新找个机会就好了。
<br/>　　车辆一路平稳行驶，什造有些无聊的看着窗外，后备箱中传来的浓郁血腥味道，飘满了整个车厢，让人作呕。
<br/>　　她讨厌血的味道，但喜欢杀人的感觉，就好像她喜欢喝酒，却讨厌喝醉一样。
<br/>　　或者说，她喜欢享受一切事物的过程，却很拒绝看到结果，那样，会让她感觉索然无味。
<br/>　　真户刚才说的意思，应该是让她把击杀喰种的功劳算在他们小组上面吧，那样的话，自己和刘君也不必接受惩罚了。
<br/>　　无所谓，她不在乎功劳，反正真户也答应了事后把库克因给她，那就已经足够了。
<br/>　　不过她身边的这个家伙，倒是蛮有趣的呢。
<br/>　　至少不像其它那些人一样，在看到她击杀喰种的过程之后，露出一副宛如看异类一般的奇怪眼神。
<br/>　　虽然那家伙搂了她，不过没关系，同伴之间表达亲密，大概都会这样做？
<br/>　　可能是想要安慰她吧，不然也不会说出那种话。
<br/>　　想到这里，什造嘴角露出一抹笑容，看着窗外的眼睛也弯了起来。
<br/>　　她还没有意识到，身边这个男人的危险性。
<br/>　　耽搁了这么长时间，现在已经是下午四五点钟了，正处于下班的高峰期，路上有些拥堵，来来往往的行人，如同蚂蚁一般，将整个城市覆盖。
<br/>　　车辆艰难的在道路上穿梭着，原本半小时不到的路程，足足比往常多花费了一个多小时。
<br/>　　到达CCG总部大楼的时候，真户的脸色阴沉的已经可以渗出水来。
<br/>　　大厅内如同平日一般，形形色色的人群正在忙碌着，有的是搜查官，但大部分人还只是负责维护这里的工作人员。
<br/>　　整个CCG都如同一个正在运转的庞大机器，身上无数个零件，都在各司其职的发挥着作用。
