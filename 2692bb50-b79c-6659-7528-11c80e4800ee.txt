
凉绣站了起来，负手而立到小草身旁，轻声说道：“你看我哪里长得像个无所事事的驴友，会如此认真的陪你在渭河上浪费这么多天的时间，要是没有所求的话，我可有大把的事情要去忙呢，不过还好，这些天的时间我没有白白的浪费掉，想知道的都知道了，想得到的也都得到了……比如你”<br/>　　<br/>　　三艘快艇翻滚着浪花迅速赶至，然后环绕在竹筏附近，每艘船上都站着两个男子，朝着凉绣点头示意道：“凉绣小姐！”<br/>　　<br/>　　小草皱了皱眉，嘴里就吐出了三个字：“为什么？”<br/>　　<br/>　　凉绣轻笑道：“还能是为什么？我自然不可能真的是为那味道平淡无奇的渭河鲤鱼来接近你，自然是为了你那个情郎王惊蛰啊，他拿走了不该拿的东西，知道很多别人都不知道的东西，碰巧我对这些都很感兴趣，又碰巧我现在找不到他的人，就只好从你的身上下工夫了，还好我还没有失望，这些天籁听你说起和王惊蛰的感情，让我挺欣慰的，至少我觉得他会舍弃我要的那些东西，然后把你换回去的，也不枉费我浪费这么多天了，你们两个倒也是真的很般配呢，一个呆呆萌萌，一个呆呆傻傻，真是天造地设的一对”<br/>　　<br/>　　茅小草拢了下长发，叹了口气，淡淡的说道：“你把我引到这里，就是怕动手劫我的时候跑了风声，才会找个挺烂的借口来到下游，早就让人在这里埋伏好了，真是难美你忍我这么多天了”<br/>　　<br/>　　“毕竟你出身天道峰，想无声无息的带你走，可能会有点麻烦，所以只好把你带过来了，我暂时还没有准备好彻底围困住王惊蛰，所以你消失的消息得先要瞒一瞒了”凉绣指着竹筏上小草的手机，说道：“给家里打个电话吧，说你陪着一个驴友去散心了，小草啊，可千万不要耍什么滑头哦，我还得把你完完整整的交给王惊蛰呢，不然你要是少了一根头发，他可恐怕是会不高兴的呢，毕竟……你们感情这么真挚，对不对？”<br/>　　<br/>　　茅小草皱了皱眉，似乎挺伤感的说道：“你让我把自认为是友情的一段情谊，感觉全都喂了狗了”<br/>　　<br/>　　凉绣哈哈直笑的说道：“值几个钱啊，喂就喂吧……”<br/>　　<br/>　　茅小草当下拿起手机拨了出去给家里打了个电话，就说自己打算出去散散心，茅清水也好奇她怎么走的这么突然，小草就说，这是一场说走就走的旅行，突然决定突然就走了。<br/>　　<br/>　　李家对她向来不太干涉，见小草说的很平淡就也没多想，只是嘱咐她注意下安全。<br/>　　<br/>　　挂了电话，凉绣伸手示意道：“请吧”<br/>　　<br/>　　茅小草随后被带上了一艘快艇，她上来后，三艘船掉头快速离去，二十多分钟后抵达一处岸边，凉绣和她一同下来上了等候在旁边的一辆车上，小草脑袋上被套了个头罩，车子开动不知去往何处。<br/>　　<br/>　　“其实，你挺让我惊讶的”上车的时候，凉绣坐在她身边说道。<br/>　　<br/>　　“是觉得我太镇定了，应该像个小女人似的哭爹喊娘大喊大叫？”<br/>　　<br/>　　凉绣想了想，说道：“幸好你还真不是那种一哭二闹三上吊的小女人，不然我该觉得王惊蛰的眼光也真是太俗气了点……”<br/>　　<br/>　　一天后，远隔千里的王惊蛰和林汶骐在陈田村牌坊门口等了没多久，王令歌开着那辆迈腾就过来接上了他俩，然后往海珠的方向走，打算从那里过海关进入大澳，此时的王惊蛰并不知道，原以为会一直寻找他的韩观山和凉绣，其中一人已经把力气用到了茅小草的身上，并且还成功的把人给带走了。<br/>　　<br/>　　可能多天后，迎接他的就是一个没有任何悬念的抉择了。<br/>　　<br/>　　车子到了口岸以后，王令歌管两人要了身份证，就去办理通关的通行证了。<br/>　　<br/>　　林汶骐就跟王惊蛰说道：“这王家在岭南经营的可以啊，护照，港澳通行证都没有就能直接过关，这还真有异姓王的风采了”<br/>　　<br/>　　“那也是多少辈的努力，才能让王家落下这么大的威望，主要还得看当家人的决策啊，每一步都走对了才能屹立至今，不然一个走不好可能就倒了，那就叫家道中落了……”<br/>　　<br/>　　王令歌带回来两张通行证交给他俩，三人直接过关进入大澳境内，口岸的一头，停着一辆大气的奔驰迈巴赫，一个戴着手套穿着西装的中年男子等候在车旁，见王令歌过来后，就拉开车门十分绅士的伸出了手示意他们上车。<br/>　　<br/>　　“这回怎么不低调了呢？”王惊蛰诧异的问道。<br/>　　<br/>　　“低调也是分地方的，在内地我们家里的人尽量做到与世无争，但却能时刻都处于山峰顶上的姿态，不过在这里么……”王令歌笑了笑，说道：“人都有跋扈和傲娇的一面，我也不例外，该释放就释放，不用憋着，毕竟制度不同么，我在这里要是在一直低调的话，谁知道我王家人能有多少力度啊，是不是这个道理？”<br/>　　<br/>　　“行了，你快别说了，说的我都要汗颜了，都是姓王的，我们朝歌王氏向来都是走的波澜不惊的，何谓跋扈，我他么到现在都没领教过”<br/>　　<br/>　　“哈哈！”王令歌拍了拍他的肩膀说道：“那是你不想，你们若想，也一样可以”<br/>　　<br/>　　王令歌这话说的不算恭维和奉承，朝歌王氏赊刀人确实在千年历史中都始终保持着祖训，就是只赊刀不涉尘世，他们若是真想往人前走走的话，多少代以来，可能说不上有几朝几代的国师，都对他们虚位以待了。<br/>　　<br/>　　大澳面积不大，弹丸之地，奔驰迈巴赫开了不到半个小时，就到了主教山上的盘龙山庄，顺着大路进入到山庄内部，王惊蛰看着窗外幽静的一栋栋院子，头也没回的忽然说道：“好地方啊”<br/>　　<br/>　　王令歌点头说道：“大澳的人笃信风水到了令人发指的地步，几乎深入到了每个人的骨子里，从住家到商场，赌场和办公地，基本都有风水格局”<br/>　　<br/>　　王惊蛰笑道：“有多少出自你们家里？”<br/>　　<br/>　　王令歌想了想，说道：“蟠龙山庄里大概有四成左右吧，至于其他的地方，零零散散的也有一些”<br/>　　<br/>　　(=)
