“爸，夏夕绾的妈咪究竟是什么人，那部医典是不是她所著，她跟帝都城又有什么关系？”夏妍妍问。
<br/>
<br/>        想到昔日的女神，夏振国这个狂热粉露出了怅然，温柔和怀念，他摇了摇头，“我什么都不知道，我对她知之甚少，我只知道医学大成都出自帝都城，帝都城藏龙卧虎，医学派系枝叶繁多，那里的中心绝不是我们这些凡人能触碰到的，她…来自于帝都城，至于其他的…老爷子应该知道一些，不过老爷子从来没有告诉过我。”
<br/>
<br/>        夏夕绾的妈咪林水瑶就是一个谜，没有人真正的看清过她。
<br/>
<br/>        夏妍妍没有再说话，林水瑶的过往她无从得知，从她的直觉而言，林水瑶身上好像带着一股神秘而强大的力量，如果那本医典是她所著，那她真的是…太可怕了。
<br/>
<br/>        她只见过医典第一卷，已经觉得惊世骇俗，她不知道医典还有多少卷，现在在谁的手里，那些帝都老头一直在搜查医典的下落。
<br/>
<br/>        这医典第一卷，是林水瑶去世之前留给自己的女儿夏夕绾的。
<br/>
<br/>        只不过，夏振国将这医典第一卷交给了自己的女儿夏妍妍。
<br/>
<br/>        不管怎么样，林水瑶已经死了，而她的女儿夏夕绾从小就被丢在乡下，只不过是一个医学废才而已！
<br/>
<br/>        这点认知让夏妍妍开心了一点，“爸，虽然夏夕绾现在是陆太太了，但是这改变不了她只有高中学历和医学废才的事实，陆寒霆被她暂时蛊惑了，不过我相信他很快就会腻的，我已经联系了F国圣黎院的钱教授，钱教授是我的导师，就这两天钱教授会飞来海城。”
<br/>
<br/>        夏振国双目一喜，“真的么？那太好了！现在外面的传言对你十分的不友善，圣黎院作为医学的殿堂，也是你的母校，钱教授的到来很快就会将大众视线拉到你是医学天才少女的身上，而夏夕绾跟你一对比，她什么都不是。”
<br/>
<br/>        夏妍妍知道自己现在的情势并不好，她不敢乱动，但她十分有信心，才华一直是最好的洗白方式。
<br/>
<br/>        而夏夕绾只不过是一个废才！
<br/>
<br/>        “爸，我现在有点累了，先去休息了，还有我希望你们不要再打架了，如果被那些媒体记者捕捉到，很快就会登上头条的。”
<br/>
<br/>        说完，夏妍妍转身上楼。
<br/>
<br/>        “妍妍…”李玉兰拉住了夏妍妍的手。
<br/>
<br/>        夏妍妍看了李玉兰一眼，她真的还是怨恨的，如果没有李玉兰策划的那场替嫁，那她现在就是陆太太了！
<br/>
<br/>        夏妍妍态度冷淡的抽回了自己的手，上了楼。
<br/>
<br/>        李玉兰一脸的委屈和黯淡，夏妍妍可是她最后的依靠了，现在母女俩的感情被夏夕绾三言两语的一挑拨已经出现了致命的裂痕，夏振国这里又开始厌恶她，她的日子简直是雪上加霜了。
<br/>
<br/>        李玉兰感觉自己又被打回了原型，甚至这一次更惨。
