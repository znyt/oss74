
“不一定是你，是你的话就冲你下手了，人家这么干你也防不住的，没必要拐个弯到她身上了”王惊蛰解释了一句，又问道：“这户人家，会说普通话么？”<br/>　　<br/>　　“能交流……”<br/>　　<br/>　　王惊蛰随即过去和对方聊了几句，就简单问了下他们家里是不是有人过世了。<br/>　　<br/>　　“有的，我娘在几天前刚过世了，才下葬”<br/>　　<br/>　　沈君心里一抽抽，说实话之前他对王惊蛰抱着的是试试看的心态，因为他始终觉得年轻人嘴上没毛办事不牢，这小朋友看着太年轻了，要不是陈重的面子放在那里，他早就另换他人了。<br/>　　<br/>　　“葬在哪了？”<br/>　　<br/>　　“就我们村后的山上”<br/>　　<br/>　　王惊蛰说道：“你要是信我的话就去你娘的坟墓看看，我估计坟可能被动过了”<br/>　　<br/>　　对方脸顿时就变了，不可置信的说道：“这不可能吧，谁动我娘的坟干什么？我们就是户农家，老实本分的，也从来没得罪过什么人”<br/>　　<br/>　　“有没有动过你去看看就知道了，看过之后如果有问题了，你打这个电话给我”王惊蛰把自己的号码告诉对方，然后跟沈君说道：“把你妹妹弄走，她要是不干的话那就绑走了，真要是到了晚上她爬到那老头的床上去，你不闹心啊？传出去了你没脸，她以后也不好做人了啊”<br/>　　<br/>　　“这他么的！”沈君骂了一句，就让手下的人强行把沈沉鱼给拽了起来，然后直接用绳子就给捆塞到了车里，期间沈沉鱼挣扎的很厉害，喊得撕心裂肺的都破音了。<br/>　　<br/>　　几分钟后，一行人从大凉山脚下离开，王惊蛰跟沈君坐的一辆车，在路上的时候他说道：“你妹妹估计是被人下的手脚，这样……”<br/>　　<br/>　　王惊蛰跟沈君交代了几句，让他把沈沉鱼所有的贴身东西全都给找出来，特别是那些常用的东西。<br/>　　<br/>　　“接灵媒得有一个媒介才行，就是个传播的方式，很有可能是她身上的随身物品，什么东西都有可能，所以你得注意点这个，千万别遗漏掉了，不然很难找到根在哪”<br/>　　<br/>　　“行，我知道了”沈君随即拿出手机联系上了沈沉鱼的助理和经纪人，跟两人吩咐着，这时候王惊蛰的电话也响了，打来的是那户人家，在电话里他告诉王惊蛰，他娘葬在山上的坟果然有被我挖开的痕迹。<br/>　　<br/>　　“把棺材打开，看看你母亲的头发或者衣服，还有指甲什么的，有没有少的”<br/>　　<br/>　　大凉山上，一处山坡的坟地中，之前那户农家的中年拎着铁锹就跳进了坟坑里，然后直接就把棺材板给撬开了，棺材里面躺着个穿着寿衣的老人，这个天气尸体都已经腐臭发烂了，有的地方还生出了蛆虫。<br/>　　<br/>　　这中年毫不忌讳的伸手就扒拉着尸体，先是把老太太带着的寿帽给摘了下来，就看见老人盘着的头发少了一截，是被剪断的，然后再往下寿衣也少了一角。<br/>　　<br/>　　他顿时冷汗就从额头上绷了出来，咬牙切齿的骂了一句，拿出手机给王惊蛰拨了过去，得到对方的回复，王惊蛰说了句果然是这么回事，然后就赶紧催促着沈君快点往回开。<br/>　　<br/>　　“那老人是没过头七就被人把魂给截了下来，然后嫁接到了你妹妹的身上，到现在几天了？”<br/>　　<br/>　　“今晚正好一周”<br/>　　<br/>　　王惊蛰叹了口气，说道：“快点开吧，赶在半夜之前就得回去了，不然过了今晚午夜十二点就是又一个头七了，到时候你妹妹的命就没了……”<br/>　　<br/>　　王惊蛰说完，沈君就赶紧催促了一声：“快点开，车扣了没事我再买，你们驾照分扣没了就再考，我就一个要求两个小时以内必须给我开回川中去”<br/>　　<br/>　　一路上，几辆车都是超速行驶的，在要抵达川中的时候都是直接逆行加闯红灯过去的，等他们到了庄园门口的时候，除了这边的几辆车外，车队后面还跟了三辆追了一路的警车。<br/>　　<br/>　　“嘎吱，嘎吱”车子停到庄园门口，沈君从车上下来，就拍着司机的肩膀吩咐道：“你领人配合一下，该怎么办就怎么办，咱得守法，能用钱和关系解决的都不是问题，明白么？”<br/>　　<br/>　　沈君说完就跟王惊蛰进了屋里，还有被绑着的沈沉鱼，这位小姐此时的状态瞅着挺吓人的，眼珠子都充血了，嘴唇全都给咬破了，明显在车厢里挣扎的挺狠的。<br/>　　<br/>　　他们回来的时候，客厅里面已经来了四个人，是沈沉鱼的经纪人和两个助理，还有个随行的保姆，这四个人都是平时沈沉鱼最贴近的人，无论去哪都会带着她们。<br/>　　<br/>　　客厅的地上，摆了一堆杂乱的东西，从衣物到化妆品，甚至就连贴身的内衣都有，这都是沈沉鱼平时的随身物品。<br/>　　<br/>　　“翻找一下这些东西”王惊蛰蹲在地上用手扒拉着几件衣服，其中还有两三件蕾丝透明的那种，看着能让人呲呲流鼻血的小布料。<br/>　　<br/>　　王惊蛰有点懵的用手指夹着一条前后就两块布加一条绳的东西，提起来诧异的问道：“口罩啊，这嘴型好像也不对呢？”<br/>　　<br/>　　菜刀文顿时呲牙笑了：“嗯呢，新款式，丁字口罩，除了造型独特以外还自带原味的呢”<br/>　　<br/>　　“唰”沈沉鱼的助理和经纪人都是女性，听见他俩的对话之后脸瞬间就红了，都在心里骂了一句：“这两男人真骚气……”<br/>　　<br/>　　沈沉鱼的东西挺多的，几个人翻了半天之后才给翻了一遍，但从头到尾也没翻出什么猫腻来，都是比较常见的东西，根本看不出有什么被动了手脚的地方。<br/>　　<br/>　　“啪”王惊蛰皱着眉头点了根烟，沈君担忧的问道：“不行？”<br/>　　<br/>　　王惊蛰“嗯”了一声，抬头问助理道：“东西全都在这呢么，有没有少的？”<br/>　　<br/>　　两个助理和保姆都摇头，经纪人说道：“小鱼儿有什么东西都会经过我们的手，不管是自己买的还是粉丝送的，我们几乎天天都和她生活在一起，全都看见过，能找到的都在这里了”<br/>　　<br/>　　“还有什么是找不到的？”王惊蛰愕然问道。<br/>　　<br/>　　菜刀文顿时无语了：“擦屁股纸，卫生巾，用完肯定就扔了，你见过谁啊用了之后还拿框装裱上珍藏起来啊”<br/>　　<br/>　　“你起开，别在这里添乱……”王惊蛰指着助理和经济人说道：“这事很关键，你们必须得给我好好回忆下，有什么东西在这几天是经过了她手的，一直都带着的”
