        “以后，我不会再让这样的事情发生。”傅夜沉信誓旦旦地允诺。
<br/>
<br/>        童筱颖甜甜地笑着，水眸好似流光溢彩地注视着傅夜沉的黑眸。
<br/>
<br/>        两人对视了一会儿。
<br/>
<br/>        童筱颖双手攀附在傅夜沉健硕的肩头上，柔软的身体主动贴入他的怀中，闭上眼睛吻上他的薄唇。
<br/>
<br/>        这无疑是她在点火……
<br/>
<br/>        傅夜沉勾唇一笑，大手托住童筱颖的后脑勺，缠缠绵绵地吻了回去。
<br/>
<br/>        下一刻，浴缸里的水花四溅，放置在台面上的沐浴露、洗发水以及各种洗漱用品的瓶子随着波光粼粼，震得七倒八歪。
<br/>
<br/>        浴室里一片暧昧又狼藉的旖旎。
<br/>
<br/>        N年后……
<br/>
<br/>        偌大的会议室里，各位西装革履的高管们，就一个策划案的推行计划里几处意见争得面红耳赤。
<br/>
<br/>        简直就是众说纷纭，各有各的道理。
<br/>
<br/>        会议桌的主位上，傅夜沉听得耳朵都起茧子了，又不得不听下去，只能一手抚着额头，一手拿着手机在会议桌底下看信息。
<br/>
<br/>        爹地，你出来！——小情人。
<br/>
<br/>        嗯，爹地在了。
<br/>
<br/>        傅夜沉快速回了过去。
<br/>
<br/>        爹地，那个明希哲抢走我家洛羽的角色了，你快帮我家洛羽把角色拿回来嘛！——小情人。
<br/>
<br/>        傅夜沉顿时看得一脸懵。
<br/>
<br/>        明希哲是谁？
<br/>
<br/>        洛羽又是谁？
<br/>
<br/>        傅夜沉忍不住在会议桌上敲了敲。
<br/>
<br/>        会议室里瞬间鸦雀无声。
<br/>
<br/>        “你们谁能给我解释一下，明希哲抢洛羽的角色一事是怎么回事？”傅夜沉这话一出。
<br/>
<br/>        在场负责娱乐影视这块的四个高管立即抢着毛遂自荐。
<br/>
<br/>        “公司正在计划拍摄的科幻电影《联盟》原定的男主角是洛羽，但是洛羽的咖位不够，又是十八线以外刚出道的，而明希哲是流量明星，所以，为了今后电影的票房，我们决定用明希哲来当男主角。”其中一个高管解释道。
<br/>
<br/>        另一个高管不以为然地反驳道：“洛羽虽然刚出道不久，但是他是科班生出来的，有扎扎实实的演技，而且颜值也不输给明希哲。我觉得，我们除了注重电影本身剧本的质量以外，还应该要注重一下演员的演技。流量明星效益，并不是长远的发展。”
<br/>
<br/>        “我也觉得用明希哲比较好！在剧本好的情况下，流量明星也是一种自带的利润。”
<br/>
<br/>        四个高管，三个投票选择明希哲，只有一个可怜吧唧地依旧支持洛羽。
<br/>
<br/>        傅夜沉还未吭声，他的“小情人”又发来了一条信息。
<br/>
<br/>        爹地、爹地！我家洛羽是最棒的！你要是不帮我家洛羽干掉明希哲，我的高考志愿就不填临海大学了，到时候我要去美国念书，离爹地妈咪越远越好！╭（╯^╰）╮——小情人。
<br/>
<br/>        看完这条短信后，傅夜沉冷不丁地对在场的各位高管说道：“那就定洛羽吧！”
<br/>
<br/>        “？！”各位高管有些措手不及。
<br/>
<br/>        这么突然？！
<br/>
<br/>        难道这个洛羽，跟他们的傅大Boss是亲戚关系？
<br/>
<br/>        傅夜沉决定的事情，在场没人敢反对。
<br/>
<br/>        下会后，傅夜沉对身旁的郑忠威命令道：“派人去查一下那个洛什么来着，哦，洛羽的底细，还有，打听一下大小姐是不是有心仪的男孩了。”
<br/>
<br/>        “是。”郑忠威颔首。
<br/>
<br/>        忙了一天回到私宅，傅夜沉见自己的小情人还未回家，便询问起了正在厨房里和刘婶一起准备晚餐的童筱颖。
<br/>
<br/>        “星儿上哪儿玩去了？”傅夜沉问道。
<br/>
<br/>        童筱颖舀了一勺汤放在碗里尝了下咸味，笑着回答道：“星儿每次出门，你都派十个保镖全方位的暗中保护，你还怕那小丫头丢了不成。你放心吧！今晚是他们的高中毕业会餐，吃完晚饭后，他们还会有同学活动。估计他们会玩通宵，星儿可能要明早才会回。”
<br/>
<br/>        “大少爷，我觉得吧！小小姐是您的宝贝疙瘩，小少爷倒像是您捡的。”刘婶忍不住打趣起来。
<br/>
<br/>        童筱颖先后给傅夜沉生了一儿一女，儿子被傅夜沉送去了部队当兵，女儿则被他留在了身边。
<br/>
<br/>        那天，校领导说，星儿的成绩优异，被美国HF大学提前录取了，结果，傅夜沉竟然不同意她去。
<br/>
<br/>        在傅夜沉看来，他的女儿不需要成为社会的精英白领，也不需要成为赚钱的工具，更不需要她才华横溢、满腹经纶，她只需要身心健康、快乐，拥有无忧无虑的幸福生活即可。
<br/>
<br/>        但是反过来，对于儿子来说，他傅夜沉的要求就比较苛刻了。
<br/>
<br/>        对于育儿理念，童筱颖保持中立状态。
<br/>
<br/>        傅夜沉做得过头了的时候，她才会站出来，其他时候，她基本上是不会插手。
