<br/>　　现在以他的全力，打出的孽龙之爪，居然都无法撼动这薄薄一层菱形晶体？

<br/>　　能够在半步天人境之中凝炼出不朽真身的一个雏形，就算得上是绝代天才了。

<br/>　　楚风眠嘴角露出了几分笑意。
<br/>　　他口中的不错，指的自然是不朽真身，现在楚风眠的不朽真身，寻常的半步天人境武者妖神，除非是动用了真正的杀招，不然都伤不到楚风眠了。
<br/>　　“只有这点手段吗？”
<br/>　　楚风眠目光看向撼龙王，风轻云淡的开口道。
<br/>　　“看来我的手中，又要多一条龙筋了。”
<br/>　　楚风眠目光看过去的一刻，撼龙王都是连连退后了几步，尤其是听到龙筋两个字，他更是感觉到了一种不寒而栗。

<br/>　　撼龙王长啸一声，现在也不顾颜面了，大声吼道。
<br/>　　他现在已经没有想要单打独斗杀了楚风眠的心思了。
<br/>　　这里是苍龙宗，是他的老巢，他现在便是要借助苍龙宗的力量，诛杀楚风眠。

<br/>　　尹龙仙见到这杀阵显现的一刻，都是浑身战栗了一刻。

<br/>　　苍龙宗培养至今，一共才培养出了六十三位金龙卫，上一次为了抓捕楚风眠，动用了五位金龙卫。

<br/>　　饶是尹龙仙知晓楚风眠的实力，深不可测，可依然为楚风眠担忧了起来。
