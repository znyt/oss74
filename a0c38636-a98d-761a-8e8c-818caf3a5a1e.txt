
<br/>　　<br/>　　    王冬至再回到屋内的时候，王惊蛰坐在床上低着脑袋，两手掐着一叠相片，他这么坐着保持一个姿势已经很长时间了，脑袋一如既往的很乱。<br/>　　<br/>　　    “咣当”王冬至拽过一把椅子，坐在他对面，轻声问道：“想聊聊？”<br/>　　<br/>　　    王惊蛰低着脑袋说道：“你确实是我姐姐·······”<br/>　　<br/>　　    王惊蛰手里面掐着几张照片，第一张是个穿开裆裤的，那是他的周岁黑白照，下面还有几张是他跟王冬至的合影。<br/>　　<br/>　　    “我就想知道，在我的身上到底发生了什么事，怎么忽然之间一下子就什么都不记得了呢？”<br/>　　<br/>　　    “电视剧里经常有这样的情节呀，因为一点意外头部重创什么的然后一下子就失忆了，这很正常啊”<br/>　　<br/>　　    “我想问的是什么意外？还有，我以前是干什么的，我是什么样的人，我有没有什么朋友”王惊蛰抬头问道：“你可以和我说说么？”<br/>　　<br/>　　    王冬至注视了他良久，最后才说道：“不可以······既然你已经失忆了，那你就彻底把过去给忘记吧，什么时候想起来什么时候算，如果想不起来那也没什么，你就当是自己重新再活一次好了”<br/>　　<br/>　　    王冬至确实没有打算把前因后果都告诉他，她觉得既然那一晚孟婆汤已经让王惊蛰把过往全都给忘记了，那就干脆彻底让他重头再来过好了，最关键的是，她觉得如果在这个时候突然告诉王惊蛰他可能活不过三十岁的话，这个打击也许让他接受不了，有的时候人和人之间的关系，是需要用善意的谎言来维持的。<br/>　　<br/>　　    也许，王冬至想的是让王惊蛰忘记他那被天谴的命运，也未尝不是一件好事。<br/>　　<br/>　　    “你不觉得这样太残忍了么？”王惊蛰嘴唇哆嗦着问道。<br/>　　<br/>　　    “放宽心，你以后遇到的，可能比这残忍的事还有很多，习惯就好了”王冬至站了起来，拍了拍他的肩膀，说道：“我明天就要离开家出去了，你自己静静之后，随便想要做什么都可以，哦，对了，柜子的一个抽屉里面我放了些钱，还有手机，也有一些衣服，我劝你如果实在无法接受这件事的话，就出去随便走走，随便你想去哪里都可以，因为家里也没什么人了”<br/>　　<br/>　　    “我不是还有个爸爸呢么？”<br/>　　<br/>　　    王冬至寻思了一下，说道：“你就当他不存在好了，毕竟你没失忆的时候，也好几年没有见过他了，以后也说不准什么时候能再见了·····”<br/>　　<br/>　　    王惊蛰觉得他这个家很奇怪，姐姐很怪，父亲更怪，怪在哪里他却也说不出来。<br/>　　<br/>　　    王冬至刚从屋中出来，就看见院子外面有人火急火燎的就冲了进来，对方见到王冬至后明显有点哆嗦。<br/>　　<br/>　　    “冬至姐？”丁武小心翼翼的说道。<br/>　　<br/>　　    “啪”王冬至笑眯眯的伸出两根手指掐了掐他的脸蛋子，说道：“坐了几年大牢，看起来有点人样了”<br/>　　<br/>　　    丁武缩了缩脖子，尴尬的笑了笑，对王冬至他是打从骨子里就发怵的，小的时候跟王惊蛰厮混在一起，他没少被这个只大了不到一岁的冬至姐收拾，整的现在都有心理阴影了。<br/>　　<br/>　　    “惊蛰回来了？”丁武连忙问道。<br/>　　<br/>　　    “在里面呢”<br/>　　<br/>　　    “哦，我去看看，他怎么突然就回来了呢”<br/>　　<br/>　　    “唉！”王冬至招呼了他一声，叮嘱道：“惊蛰之前出了点意外，什么都不记得了，你看见他的时候，如果他问你以前的什么事，你不要说的太多了”<br/>　　<br/>　　    “啊？失忆········”<br/>　　<br/>　　    “嘴严实一点，话别太多了，去吧！”<br/>　　<br/>　　    丁武带着疑惑进了房间，王惊蛰茫然的看着他，想了半天后才想起来，有一张照片是他和一个小孩尿尿和泥的时候被王冬至给拍下来的，那个自己撒尿尿在泥里后，用手搅拌的孩子，就是面前这个。<br/>　　<br/>　　    “你叫？”<br/>　　<br/>　　    丁武从身上掏出一包烟，抽出两根递了过去：“丁武，你以前叫我小武的，我听说你和冬至姐回来了就过来看看，没想到，哎······”<br/>　　<br/>　　    王惊蛰仿佛抓住了一根救命的稻草，一把抓住丁武的胳膊，急促的问道：“你跟我说说，我以前的事呗？”<br/>　　<br/>　　    “啪”丁武点上烟，然后凑了过去。<br/>　　<br/>　　    王惊蛰愣了愣，然后点着烟抽了两口，就剧烈的咳嗽了起来。<br/>　　<br/>　　    丁武皱眉说道：“忘记抽烟了，也是好事，那就干脆别抽了，我还正愁戒不了呢”<br/>　　<br/>　　    “不，不，你说，你说·······”王惊蛰抹着眼泪说道。<br/>　　<br/>　　    丁武叼着烟，神色复杂的说道：“我和你小的时候是一起玩的，就是发小，不过我在十几岁的时候就跟随我爸妈去了京城，之后咱俩就一直没怎么见面了，再往后我因为犯了一个案子蹲了监狱，一蹲就是六年多的时间，前段时间我才出来回到家里，所以你问我的话，我可能只会告诉你十几岁以前的事，再往后你有什么经历，那我也不清楚了”<br/>　　<br/>　　    王惊蛰顿时失望的叹了口气，半天后才说道：“那就说说十来岁以前的吧”<br/>　　<br/>　　    丁武看着他，没有吭声。<br/>　　<br/>　　    王惊蛰皱眉说道：“你倒是说啊？”<br/>　　<br/>　　    丁武摊着手，苦笑道：“有什么好说的呢？说咱俩去村东头扒着王寡妇家的窗户看她洗澡？还是去地里偷人家的红薯和玉米烤着吃？那几年咱俩就是村里最典型的恶霸，无恶不作的那种，没干过什么好事，咱俩经常被家里人用扫帚堵在墙角里使劲的抽，都打折好几把了”<br/>　　<br/>　　    “没了啊？”<br/>　　<br/>　　    “嗯，就这些”<br/>　　<br/>　　    王惊蛰咬着烟嘴，沉沉的叹了口气：“我怎么觉得，我的经历应该是很复杂的那种呢，可是我了解到的，却好像我的生活就只是一张白纸而已呢”<br/>　　<br/>　　    “想不通就别想了，以后也许你会慢慢记起来的，我过两天就要离开朝歌去川中了，你要是在家不太好过的话，可以跟我出去走走·······”<br/>　　<br/>　　    王惊蛰想了想，问道：“不麻烦么？”<br/>　　<br/>　　    丁武直接怼了他一拳头说道：“我这辈子只有你这一个朋友，你也是！”
