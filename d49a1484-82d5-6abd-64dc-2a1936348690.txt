        下一个瞬间，一团漆黑的氤氲，慢慢地将傅昼景的身影吞噬。
<br/>
<br/>        童筱颖无力地抬起手来，想要触碰她心心念念惦记的这个男人，但终究是体力不支，眼前一黑，她晕了过去。
<br/>
<br/>        “筱颖！”
<br/>
<br/>        傅昼景下意识地抱住童筱颖，连忙将她打横抱起，放到了床上。
<br/>
<br/>        她真的又累又困，只要他平安无事的回来，她终于可以放心了。
<br/>
<br/>        “筱颖，对不起。”将童筱颖平放到床上后，傅昼景匍匐在床边，嘴里呢喃地吻了吻童筱颖的额头。
<br/>
<br/>        无意间，他看到一旁餐桌上未动过的饭菜后，爬上了床，心疼地将童筱颖搂入了怀中。
<br/>
<br/>        他家的丫头，为什么总是这么傻呢？傻得让他好心疼又好心怜。
<br/>
<br/>        “昼景……我好想你……”不知过了多久，童筱颖迷迷糊糊的呓语着。
<br/>
<br/>        她的话，他听得一清二楚，让他忍不住紧紧地将她圈入怀中。
<br/>
<br/>        守护着筱颖，就像是在守护着另一个脆弱的自己。
<br/>
<br/>        傅昼景的下巴抵在了童筱颖的头顶上，微微合上了双眼。
<br/>
<br/>        等到童筱颖再次醒过来的时候，是被饿醒的，而身边早已经没了傅昼景的身影。
<br/>
<br/>        童筱颖知道，他一定又去上班了，不过这次令她意外的是，餐桌上的饭菜都被人吃光了。
<br/>
<br/>        是昼景吃的吗？可那饭菜是昨晚做的啊！
<br/>
<br/>        童筱颖从床上下来，走到餐桌边，看到了傅昼景的留言条。
<br/>
<br/>        筱颖，我公司里还有工作要忙，先去上班了，晚饭没法在家里吃，不过，我会在十二点前回来陪你。——昼景。
<br/>
<br/>        “叮咚——”
<br/>
<br/>        童筱颖刚把纸条看完，门铃就响了。放下纸条后，她又跑去开了门，没料到是送外卖的。
<br/>
<br/>        正当童筱颖纳闷是不是送错了的时候，她无意间看到外卖纸条上的备注后，甜甜地向外卖员说了声：“谢谢。”
<br/>
<br/>        昼景真的很体贴她，想着她睡醒后一定会肚子饿，所以直接给她点了外卖。
<br/>
<br/>        傅昼景点的东西，都是她童筱颖爱吃的。
<br/>
<br/>        虽然是一个人吃饭，但童筱颖觉得自己很幸福。
<br/>
<br/>        吃完饭后，童筱颖一如既往地开始收拾家务。当她在卫生间里看到洗脸台下的桶子里有傅昼景换下的衣物未洗，因为是白衬衫和西装裤，便主动去拿来了脸盆，打算手洗。
<br/>
<br/>        不过，令童筱颖感到奇怪的是，昼景的白衬衫很干净，不像是穿了一天一夜的样子，而且白衬衫上还有一股淡淡的非常好闻的香水味。
<br/>
<br/>        昼景平常也用香水的吗？
<br/>
<br/>        童筱颖不禁纳闷了起来，她完全没往另一个方向去想。
<br/>
<br/>        就算真有美女向傅昼景投怀送抱，童筱颖也相信傅昼景一定会为了自己而拒绝对方。
<br/>
<br/>        不为别的，只因为这是夫妻之间，最起码的信任与尊重。
<br/>
<br/>        哪怕昼景不爱她，以昼景的为人，定会拒绝外边的各色野花。
<br/>
<br/>        童筱颖打心底的信任傅昼景，所以才不会胡思乱想。
<br/>
<br/>        原本，童筱颖想着，既然昼景回来了，两个人可以好好地在一起吃一顿晚餐，结果在她做晚饭前，傅昼景却发了条短信告知她，公司里有聚餐，晚上不会回来吃饭。
<br/>
<br/>        下班后，傅昼景确实有聚餐，是他作为上司好好犒劳这段时间为他做标书的员工。他这次出差不仅替傅氏集团拿下了北城扩建的项目，而且还代表为自己在建筑行业打出了知名度。
<br/>
<br/>        他就是这样一步一个脚印地往上爬，而不像大哥傅夜沉直接空降金字塔顶端。
<br/>
<br/>        傅昼景知道，自己跟大哥相比，只不过是输在了投错了娘胎上而已。
<br/>
<br/>        或许，是他心胸太狭隘了，才会这么想……
<br/>
<br/>        恍神间，同在一个酒桌上的助理赵斌忽然询问道：“傅总，他们说，想去‘夜惑’唱歌。”
<br/>
<br/>        “那就去吧！我请客！”傅昼景会心一笑，落落大方地回答。
<br/>
<br/>        赵斌怔了一下，刚想说什么，又欲言而止。
<br/>
<br/>        傅昼景现在的经济条件，去“夜惑”那种高档娱乐会所消费，着实有点儿难堪。
<br/>
<br/>        但是，男人谁不好面子？
<br/>
<br/>        而且，说要去“夜惑”唱歌的“始作俑者”，还是季思妍。
<br/>
<br/>        谁都看得出，季思妍对傅昼景有那方面的意思，只是季思妍平日在公司里，把同事关系处理得非常融洽，让大家心照不宣。
<br/>
<br/>        晚饭后，酒店前台，傅昼景结了账，季思妍走过来，便将一张“夜惑”的黑卡递给了傅昼景。
<br/>
<br/>        傅昼景瞥了季思妍一眼，并未接她的卡，季思妍见状，拉起傅昼景的手，将自己手里的黑卡强塞到了他的手中。
<br/>
<br/>        “我哥是‘夜惑’里的VIP，刚好今晚免费送一个KTV包厢可以唱到晚上十二点。”季思妍微笑着解释道，“更何况，发动大家要去唱歌的人是我。”
<br/>
<br/>        傅昼景皱了下眉头，还是将季思妍塞在自己手心里的这张黑卡收下了。
<br/>
<br/>        什么时候起，他也不得不为了省钱，贪起了这样的小便宜？
<br/>
<br/>        傅昼景在心底嘲讽自己，终究还是败给了穷困潦倒的现实。
<br/>
<br/>        大家一伙人有十多个，进入“夜惑”的大门后，就是一条金碧辉煌的通道，而通道的右边便是清吧，去夜惑的KTV部必须经过这里。
<br/>
<br/>        忽然，一阵婉转动听的歌声入耳，这熟悉的嗓音，让傅昼景怔愣了一下，不经意间地驻足，循声望去，双眸捕捉到一抹亮丽的白影。
<br/>
<br/>        身旁的男女，见傅昼景停下了步伐，也纷纷跟着伫足。此时，一个熟悉这里的男同事，顺着傅昼景的目光看过去后，便笑着介绍道：“现在台上唱歌的那个女人，在这里的艺名叫‘小谢’，因为歌声很甜，所以在清吧里的人气很高。传闻，前段日子，有两位贵少为了她大打出手。”
<br/>
<br/>        傅昼景却在不知不觉中，皱了皱眉头。
<br/>
<br/>        那边台上的女孩，脸上戴着白羽假面眼镜，梳着公主头，穿着白色的小洋装，嘴里唱着婉转抒情的英文歌。
<br/>
<br/>        或许，别人看不出台上那女孩的真实面容。
<br/>
<br/>        但是，傅昼景却能一眼望穿。
<br/>
<br/>        小谢？艺名？
<br/>
<br/>        她是他的筱颖啊！
<br/>
<br/>        筱颖为什么会在这夜店里卖唱？
<br/>
<br/>        哪怕只是在清吧里卖唱，但也……
<br/>
<br/>        傅昼景收回目光，心情阴沉，脸上的表情却很淡漠地继续往前走去。
<br/>
<br/>        刚刚他的一颦一笑，季思妍全看在眼里，甚至还产生了一种，他很在意那个卖唱的女人的错觉。
<br/>
<br/>        一个在夜店里卖唱的女人，身心都不干净，阿景肯定不会喜欢的，是自己想多了！
<br/>
<br/>        季思妍暗自在心底劝说自己。
<br/>
<br/>        一群人走进包厢后便开唱，傅昼景却在一个偏暗的角落里的沙发上坐下想自己的事情。
<br/>
<br/>        如果，他没猜错的话，筱颖一定是因为想要替他还债，才出来兼职打工的。
<br/>
<br/>        白天，她在小公司里做设计，晚上她又在夜店里卖唱。
<br/>
<br/>        她就这么不相信他的能力吗？
<br/>
<br/>        她是觉得，他养不起她，所以才如此的吗？
<br/>
<br/>        种种复杂的情绪交织在心里，让傅昼景顿时心烦意燥起来。
<br/>
<br/>        筱颖现在是他唯一的心灵支柱，可现在连她的心灵支柱都不相信他的实力了，那他这段日子如此辛苦地努力工作又有什么意义？
<br/>
<br/>        或许，是他太大男子主义了。
<br/>
<br/>        其实，他压根就不希望筱颖出去工作。
<br/>
<br/>        他觉得自己是个大男人，就应该赚钱养家，女人就应该在家里好好待着。
<br/>
<br/>        季思妍见傅昼景一个人闷在角落里不吭声，于是端着两杯啤酒走了过去，并在他身边坐了下来。
<br/>
<br/>        “阿景，我们喝一杯吧！”季思妍柔情似水地附在傅昼景的耳边说道，“祝你将来的事业步步高升。”
