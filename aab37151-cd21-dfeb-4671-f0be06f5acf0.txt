
<br/>　　<br/>　　    孔德容说完，神情轻松的看了眼那一拨死士，这些被墨家每一代巨子培养出来的死士，放出去一个都能当十个用，最关键的是被洗脑之后的他们，很有种为墨家巨子堵枪眼的觉悟，本来挺可怕的一群人，此时身形忽然摇摇欲坠了起来。<br/>　　<br/>　　    嗯，手无缚鸡之力了！<br/>　　<br/>　　    刚刚进来鱼馆，他们吃的饭菜里全都给下了蒙汗药，此时可能他们连抬手拿刀的力气都没有了。<br/>　　<br/>　　    “读圣贤书，立君子品，习圣人学，知心明路”凉绣怒极反笑，指着孔德荣说道：“孔圣人之道如今都被你们给糟蹋成什么样子了，居然连下药这种事都会干了，真够不要脸的了”<br/>　　<br/>　　    孔德荣背着手淡淡的说道：“老祖宗还告诉我们一句话，叫不择手段不达目的誓不罢休呢！”<br/>　　<br/>　　    此时凉绣对自己忽然生出一种极度挫败的感觉，她这人向来自负，就属于那种谁谁都瞧不起的一类人，但她现在忽然发觉，她最自负的东西，已经被摧残的体无完肤了，小草从头到尾把她给耍了个团团转，王惊蛰更是将她玩弄于股掌之间了，这脸都被按在地上摩擦几个来回了？<br/>　　<br/>　　    “咣当”店外的门被人猛的推开，王惊蛰面无表情的走进来，很不屑的瞥着她说道：“凉绣还玩么？不在你家的唐梁山上，你嘴中所说的墨家死士又眼看着都趴下了，来，你告诉我，你还想有什么手段能玩下去？你这女人啊，忒不要脸了点，什么招都使，你现在也尝尝自己被人给圈住了是啥感觉吧，孔府的人再加上我在这里就是借你一对翅膀你往哪里飞啊？更何况，你身上还中了蛊……”<br/>　　<br/>　　    凉绣身形摇摇欲坠，她跌坐在椅子上，喃喃的摇着头，神态都有点魔怔了，嘴里一连嘟囔着：“不，这不可能，你们骗我，你们怎么能瞒着我做了这么多，我都不知道”<br/>　　<br/>　　    凉绣这状态，明显是心里防线已经全被击垮了，这种情况，俗称于走火入魔。<br/>　　<br/>　　    在自己最为擅长和自负的方面，来了个满盘皆输，她无疑于是被十八个壮汉给来了个亲密的接触，试问就这种状况，谁能接受得了。<br/>　　<br/>　　    孔德荣扭头问道：“这女人……”<br/>　　<br/>　　    王惊蛰沉思片刻，点头说道：“杀了吧，留着也是个祸害，打蛇不死反被咬的事层出不穷啊，这里风景不错，有山有水的，杀了正好埋上”<br/>　　<br/>　　    “哎！”<br/>　　<br/>　　    王惊蛰一番话说完，忽然之间，从屋外传来一声叹息。<br/>　　<br/>　　    三人脸色同时惊变，这一声叹传来之后，居然久久都未散去，有一种余音缭绕而三日未绝的感觉。<br/>　　<br/>　　    “唰”王惊蛰，孔德荣还有茅小草一同转头，透过玻璃看向窗外，就见路对面站着个年岁看不出多大的老人，他脑袋上头发稀松，穿着身印着铜钱的唐装，微微佝偻着肩膀，但一双眼睛却十分有神的透了过来。<br/>　　<br/>　　    就这么一瞬间，他们三个都很有种被野兽给盯上了的感觉。<br/>　　<br/>　　    这老人很可怕，有种深不见底的感觉。<br/>　　<br/>　　    三人如芒在背，被盯的头皮都麻了，特别是当对方一步步迈步过来的时候，犹如危险在一点点的临近，愣是逼的他们一个字一个动作都没整出来，仿佛瞬间就被定格了一样。<br/>　　<br/>　　    这老人来到门口之后说了一句：“得饶人处且饶人吧”<br/>　　<br/>　　    然后推门进来，又说了一句：“真还想要赶尽杀绝么？”<br/>　　<br/>　　    当门关上后，他说了第三句话：“墨家就这一根独苗了，你们说她若死了，我得用多少人来陪葬才能消得了我的心头之恨？”<br/>　　<br/>　　    这老人三句话说完，他们几个心中瞬间就通透，知道来人是谁了。<br/>　　<br/>　　    墨家历代神秘兮兮的巨子，几乎没人知道他们是谁，巨子深藏于市井，山野之中，从不显山露水，也不涉足世事，似乎过着与世无争的生活，传承着他们老祖宗墨子的墨家思想。<br/>　　<br/>　　    虽然巨子不出世，但知道其人的人还真不能把他不当回事，毕竟人家被封杀了千年都还没断了传承，就足以说明有可圈可点之处了。<br/>　　<br/>　　    这老人进来之后，也不和他们三人说话，而是来到了还在喃喃自语，一遍又一遍的重复着“不可能”三个字的凉绣身后，一脸溺爱的用手摸着她的头发，很是惋惜和悔恨的说道：“其实我早就该拦着你了，自从你父母离世之后，我太惯着你了，教了你一身的本事，却没想到也教出了你的目中无人，狂妄自大，我小心谨慎，聪明了几十年，没想到却在自己的孙女身上犯了糊涂……”<br/>　　<br/>　　    这老人说话的时候，一扫之前的跋扈气势，只是像个普普通通的老长辈了，言谈间完全是对至亲之人的一种发自内心的疼爱和悔恨。<br/>　　<br/>　　    凉绣还在魔怔，失神的她仿佛一直都没有听到身后爷爷说的话，就两眼呆滞的看着前方。<br/>　　<br/>　　    刚刚，从茅小草嘴里说出的一番话，还有孔德荣和王惊蛰的出现，击垮了她的内心防线后，这姑娘已经彻底被摧残的失常了。<br/>　　<br/>　　    这老人虽然进来后没有同三人说话，但王惊蛰他们一直如临大敌的防着对方，精神都绷紧了，大有蓄势待发就突然出手的意思。<br/>　　<br/>　　    虽然不敌，但肯定不会束手就擒。<br/>　　<br/>　　    忽然间，墨家巨子抬头看着王惊蛰说道：“我本来只想让凉绣当个普普通通的良家姑娘，安稳的活一辈子就算了，从来没想过让她在外界抛头露面，可这孩子的性格也太倔强了一些，我拦是拦不住的，最后就只能放任了，小朋友这一页不知可不可以掀过去？有什么得罪的地方，我代她陪个不是如何……”
