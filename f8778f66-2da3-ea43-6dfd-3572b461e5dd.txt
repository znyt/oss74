<br/>        在酒桌上，夏颜对大家说：“你们和呼延都是好朋友，以后就相互扶持，西省目前信息化发展很快，对大家都机遇，你们和我一起好好的干一番事情，我们需要相互帮助，站在同一条线上将西省的工作做好。”
<br/>
<br/>        “夏总吩咐我们怎么干我们就怎么干，我长生都是你的马仔，鞍前马后在所不辞。”宋长生举起酒杯表态。
<br/>
<br/>        顾恩赐笑眯眯的说：“我倒是希望俩个小美女能和我们一起大干一番，那样才有意思，夏总，你说是不是？”
<br/>
<br/>        顾恩赐在呼延临庆找他之后就将呼延临庆和陈子昂之间的几次交往调查清楚了，其实不是呼延临庆和陈子昂有矛盾，而是夏颜和陈子昂有矛盾，女人呐，因爱生恨真是可怕。
<br/>
<br/>        顾恩赐和西城分公司也一直有来往，不过，踩高压低似乎是人的本性，他们这几年一直顺风风水让人多少有些眼红，能看到他们吃吃苦头还是很开心的。
<br/>
<br/>        夏颜看着易倾秋和陈天天，俩个人正甜甜的笑着，一副人畜无害的样子。
<br/>
<br/>        “小易，天天？”夏颜笑着喊易倾秋和陈天天。
<br/>
<br/>        “嗯，吃了人家的嘴软，拿了人家的手短，我们都已经吃了你们的饭，喝了你们的酒，能不和你们一起干吗？”陈天天举起饮料。
<br/>
<br/>        “哈哈哈，小姑娘干脆，那今后我们就是同一条战线上的人了。”呼延临庆以为易倾秋他们不会来参加这次聚会。
<br/>
<br/>        “我替我们陈总干了这杯酒，感谢大家将她也拉上你们的同一条战线。”易倾秋举起酒杯。
<br/>
<br/>        “易姐姐，在座的只是想拉我们俩，可不是想拉陈总呢。”陈天天脆生生的声音又响起来了。
<br/>
<br/>        “哈哈哈哈，小姑娘童言无忌，童言无忌，别真相。”呼延临庆哈哈大笑。
<br/>
<br/>        “是这样的吗？夏总？”易倾秋一脸的期待看着夏颜。
<br/>
<br/>        “你想是这样就是这样，你不想也行啊。”夏颜看着易倾秋。
<br/>
<br/>        “跟着夏总有肉吃吗？”易倾秋开始讨价还价了。
<br/>
<br/>        “肉是有得吃，不过得看你的本事能不能吃到了。”宋长生手中的酒还端着。
<br/>
<br/>        “嗯，我觉得我吃的肉肯定比宋总多。”易倾秋笑眯眯的。
<br/>
<br/>        “为什么？”几个人齐声问易倾秋。
<br/>
<br/>        “因为我是陈子昂公司商务部经理，你们几个目前的实力都比不上陈子昂，只有报团取暖才有得一拼。如果说你们单独让我和你们站在一条线上，我是不会放弃目前的公司的。”易倾秋侃侃而谈，这里面宋长生和谷恩赐都表态了，她想知道许弋澄是什么意思。
<br/>
<br/>        “许总，你觉得易姐姐的话对不对？”陈天天问坐在他旁边的许弋澄。
<br/>
<br/>        “她说得很对，这个世界上大家都需要报团取暖，只不过要找志同道合的人而已。”许弋澄从始至终都很少说话，包括夏颜来的时候，他就是点点头打了个招呼。
<br/>
<br/>        “那么许总会和我们报团取暖吗？”陈天天丹凤眼闪着狡黠的光。
<br/>
<br/>        “我们？我和你吗？这个我得考虑一下，天天陈小妹妹！”许弋澄哈哈哈大笑，拿起手中的酒一饮而尽。
<br/>
<br/>        “特么的，许弋澄就是只老狐狸。”易倾秋心里暗骂，许弋澄竟然知道天天。
<br/>
<br/>        “许总，我们当然会志同道合的，和夏总在一起会飞黄腾达，谁不愿意谁就是瓜怂。”易倾秋用西城的方言回了许弋澄一句。
<br/>
<br/>        “哈哈哈哈！来，干杯！”夏颜开心的大笑，这个小姑娘有趣，太有趣了。
<br/>
<br/>        “谁要是当天天小妹妹是瓜怂，谁就是瓜怂，是吧天天小妹妹，你们家个个智商爆表，你这小脑袋瓜子也不会差吧？”许弋澄逗起了坐在他身边的陈天天。
<br/>
<br/>        “弋澄，你认识天天的家人？”夏颜问许弋澄。
<br/>
<br/>        “很久之前认识她姑姑，不过这都很多年没有见了。”许弋澄笑着说。
<br/>
<br/>        “我咋不知道？”夏颜接着问。
<br/>
<br/>        “这么多年我们也不曾联系，你怎么能知道呢？”许弋澄反问。
<br/>
<br/>        “你........你还在怪我？”夏颜三杯酒下肚，就会有点飘。
<br/>
<br/>        “临庆，你可不对啊，你知道颜颜不会喝酒，身为狗腿子的你在她身边不好好照顾她，让她天天醉酒可不是什么好事情。”许弋澄差开了话题。
<br/>
<br/>        “是啊，我们罚呼总喝三杯。”顾恩赐也开始火上浇油。
<br/>
<br/>        “三杯怎么能够了，三碗，夏姐姐，你如此美貌动人，呼总都不知道怜香惜玉，还不如顾总对你好。”陈天天适时的叫了一声“姐姐。”
<br/>
<br/>        “来，夏总，你杯中酒我替你喝了。”宋长生起身拿起夏颜面前的酒仰头就喝下去了。
<br/>
<br/>        “尼玛，这个宋长生太恶心人了。”陈天天看着宋长生那狗腿的样子翻了一下白眼。
<br/>
<br/>        “夏总人中龙凤，女中豪杰，不过是一杯酒而已，用得着宋总如此贬低她？”陈天天冷冷的声音响起来。
<br/>
<br/>        “我怎么是贬低她了？”宋长生一愣。
<br/>
<br/>        “宋总是故意的呢还是故意的？你不知道我们女生，尤其是美丽的女生，只有自己喜欢的人才能动自己用过的碗筷或者酒杯吗？”易倾秋清冷的小脸上带着嘲弄。
<br/>
<br/>        宋长生尴尬的看着夏颜面前空着的酒杯，站也不是坐也不是。
<br/>
<br/>        “那有那么多讲究。”呼延临庆为了缓和尴尬的气氛，拿起分酒器又给夏颜虚了一杯酒：“我经常提夏总挡酒。”
<br/>
<br/>        夏颜心中火起，脸上红一阵白一阵，呼延临庆真不愧是猪队友，尤其还当着许弋澄的面。
<br/>
<br/>        “服务生。重新拿一个酒杯好吗？”易倾秋问服务生重新要了一个酒杯，亲自给夏颜倒满酒：
<br/>
<br/>        “夏总，我们是美丽的小女生，要矜持，虽然他们很狗腿，但是我们一定要配得上我们的地位，女人呐，只有尊重自己的时候别人才能尊重你。”
<br/>
<br/>        宋长生脸上已经是千变万化了，呼延临庆举着酒壶不知道如何是好，顾恩赐一副幸灾乐祸的样子，想笑不敢笑，许弋澄靠着座椅一副置身事外看好戏的模样。

<br/>        “夏姐姐，来，我们三个女人干一杯，女人和女人在一起才是铜墙铁壁，无坚不摧。”陈天天俏皮的拿起饮料和夏颜碰了一下。
<br/>
<br/>        “好，以后你们俩就是我的左膀右臂。”夏颜拿起酒和易倾秋、陈天天碰了一下，轻轻的抿了一口酒，气氛虽然尴尬，她还得掌控大局，这些人必须得掌握在手中。
<br/>
<br/>        “来，大家干了这一杯，我喝不了酒，以后还得需要各位替我挡酒，能挡酒的不是最喜欢的但都是过命的兄弟，希望在座的能和我一起为西省做一份贡献，能替我担当一些。”夏颜不愧是夏颜，一句话又将大家拉到了她的身边。
