　　在楚风眠的身躯之中，一枚玉符突然闪亮了起来，这一枚玉符，正是圣龙宗之中的联络玉符。
<br/>　　楚风眠一股灵力注入其中，便是传递出来了青梦的声音。
<br/>　　“小心圣院宗的人！”
<br/>　　在这青梦的声音之中，楚风眠都是听出了几分焦急的感觉。
<br/>　　圣院宗的人？难道圣院宗的人，对青梦师姐动手了？
<br/>　　楚风眠眼神猛然一变，从青梦声音之中的焦急来看，只怕是青梦遇到了不小的麻烦。
<br/>　　而且看起来，这一切都是圣院宗搞的鬼。
<br/>　　本来楚风眠还以为青梦跟燕皇在一起，两名九劫古帝，应该不会出现在什么危险，看来楚风眠还是将圣院宗对圣龙宗的敌意，想的太简单了。
<br/>　　“青梦师姐跟燕皇，应该还在这藏地之中。”
<br/>　　楚风眠一股灵力再度注入到玉符之中去。
<br/>　　顿时从楚风眠脑海之中，显现出了一副图像来，这玉符之间，可以互相探察对方的位置。
<br/>　　楚风眠观察了一下，就看见青梦跟燕皇，就在这藏地之中，距离楚风眠不过百里的距离。
<br/>　　“过去看看！”
<br/>　　融入人皇罐碎片，不着急在这一时，以青梦的性格，如果不是遭遇到极大的危险，不可能在这个时候拿玉符通知楚风眠。
<br/>　　楚风眠将那一枚人皇罐碎片，收取到了空戒之中，旋即便是化为了一道遁光，在这藏地之中连连穿梭。
<br/>　　随着楚风眠实力的越来越强，在这藏地之中佛门禁制对楚风眠的限制，也越来越低，百里的距离，几乎不足十分钟的时间，楚风眠就赶了过去。
<br/>　　就在百里之外的一处山脉之中，数道身影，在其中连连穿梭，爆发出一阵阵的灵力波动。
<br/>　　显然是有着强者，在其中战斗，这战斗的余波，都是极为强横，许多九劫古帝，看到这一幕，都在旁边围观，却没人敢贸然进入其中。
<br/>　　其中在这战斗之中，其中两名武者的实力，最为耀眼，几乎是以一敌二，以一敌三，在面对这许多古帝的围攻，好不示弱。
<br/>　　但是这两名武者的敌人，却是足足有着数十名古帝，连续不断的追杀之下，也迫使他们二人，只能够是一直逃走避战，无法抗衡。
<br/>　　“青梦，燕皇。”
<br/>　　楚风眠目光一眼看过去，就看见这两名武者，正是青梦跟燕皇。
<br/>　　他们二人如今身上的气息，也要比起进入这藏地之前，强横了不少，几乎都是达到了九劫古帝巅峰的层次，足以是跟小耀王寒夜那等人物，平起平坐了。
<br/>　　尤其是青梦身上的气息，更是要超越九劫古帝巅峰了，算是跟龙摩罗一样，称得上是九劫古帝之中的霸主了。
<br/>　　只要是再进一步，就有着冲击半步天人境的可能了。
<br/>　　青梦也不愧是圣龙宗弟子，绝世古帝，资质逆天无比，只要是得到机缘，实力便是飞涨。
<br/>　　不过双拳难敌四手。
<br/>　　在他们的身后，则是足足数十名古帝，几乎是在一路追杀这他们二人。
<br/>　　青梦跟燕皇，也只能够是边打边退，无法正面抗衡。
<br/>　　“圣院宗的人！”
<br/>　　看到这数十名古帝的装扮，楚风眠就明白了他们的来历，都是圣院宗的圣子。
<br/>　　怪不得青梦会通过玉符告诉楚风眠要小心圣院宗的人，原来她跟燕皇，正在遭受这一群圣院宗圣子的追杀。
<br/>　　论起势力，圣院宗足以算得上是圣龙秘境的第一势力，圣龙宗弟子，虽然逆天，可人数太少，只有天才妖孽，才有资格加入。
<br/>　　大部分圣龙秘境中的天才，都归属于圣院宗了，导致圣院宗的势力强横。
<br/>　　在这数十名古帝之中，九劫古帝就有七位，但是其中一名九劫古帝，他身上散发出来的力量，却是无比强横。
<br/>　　几乎是要比起所有的九劫古帝，都要更强，站在了帝尊境的巅峰，哪怕是龙摩罗这等九劫古帝之中的霸主，都不如此人身上的力量强横。
<br/>　　“好厉害的人物，九劫古帝之中，居然还有这样的强者？”
<br/>　　楚风眠本以为龙摩罗那种，就已经算是站在了九劫古帝巅峰的强者了。
<br/>　　可眼前这一名古帝，论起力量来说，却是比起龙摩罗都要强横不少。
<br/>　　而且此人的年纪，极为年轻，不足百岁，在古帝之中都算是极为年少的一位天才了。
<br/>　　这一位年轻古帝，就是这一群人之中的首领，在他的发号施令下，这些古帝一直在追杀这青梦跟燕皇二人。
<br/>　　“青梦，你不用在挣扎了，圣龙宗的灭亡，已经是注定的了，今日你投靠了本尊，做本尊的女人，本尊可以给你一条活路！”
<br/>　　那一名年轻古帝，实力极为强横，他冷笑一声，旋即大手一挥，灵力凝聚，化为了一道巨大无比的灵力牢笼，将青梦跟燕皇，封锁在了其中。
<br/>　　“你做梦！”
<br/>　　青梦听到这一尊年轻古帝的话，怒喝一声、
<br/>　　转头对着旁边的燕皇说道，旋即凝聚灵力，化为了一道青芒，向着这灵力牢笼斩了过去。
<br/>　　“燕皇，一起出手打出去！”
<br/>　　这一道青芒在青梦的手心之中，如同一道利刃，足以斩杀一切，可斩在这灵力牢笼之上的一刻，这灵力牢笼，却只是震荡的一番，并未被打破。
<br/>　　燕皇同时出手，也是一样的结果，根本无法打破这一道灵力拉拢。
<br/>　　这个结果让青梦跟燕皇的脸色，都变的无比难堪起来。
<br/>　　“不用白费力气了，圣龙宗之中，除了龙千绝那个老家伙，剩下的人，在本尊的面前，不过都是废物罢了。”
<br/>　　年轻男子见到这一幕，冷哼一声道。
<br/>　　“还是说你们打算指望楚风眠那个废物来救你们？那个废物，现在只怕是早就死了，等你们死了，也可以跟他团圆了。”
<br/>　　“既然你们这般顽固不化，那本尊就杀了你们！”
<br/>　　年轻古帝冷喝一声，眼神之中尽是杀意，爆发而出，身形闪烁，冲了过去。
<br/>　　“快退！”
<br/>　　青梦跟燕皇的脸色猛然一变，纷纷凝聚灵力来，化为一大屏障，抵挡在面前，他们的眼神之中都是露出了几分绝望之色。
