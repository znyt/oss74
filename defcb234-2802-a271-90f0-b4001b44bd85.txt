        这把钥匙是金色的，手柄很精致，上面的凹凸匙纹，比普通的钥匙要复杂点。
<br/>
<br/>        童筱颖半信半疑地将钥匙握在手心里，然后跟连华生颔首，道：“那我先回我自己的房间去了。”
<br/>
<br/>        “好！”连华生礼貌性地欠了欠身，“大小姐慢走。”
<br/>
<br/>        见童筱颖离开后，连华生轻轻蹙了蹙眉，漆黑透亮的眼睛里闪过一抹意味不明的情愫。
<br/>
<br/>        一种直觉告诉他，傅夜沉真正爱的女人，不是苏漫雪，而是童筱颖。
<br/>
<br/>        至于苏漫雪和童筱颖为什么会长的一模一样，这期间必定有他连华生不知道的故事。
<br/>
<br/>        傅夜沉很会把自己的真实感情伪装起来，但是，爱一个人时流露出来的真情，是永远无法掩饰的。
<br/>
<br/>        连华生只希望日后，这个童筱颖不要成为阿沉的致命弱点。
<br/>
<br/>        童筱颖拿着金钥匙在大哥傅夜沉的私宅里前前后后逛了一圈，最终决定将金钥匙塞入一个小玻璃瓶中，然后密封，埋在了后花园的玫瑰花丛里。
<br/>
<br/>        她将金钥匙埋好后，又拆下了自己的发圈，绑在了一旁一株玫瑰树上做标记。
<br/>
<br/>        哎——大哥和那个连医生总是神神秘秘的！
<br/>
<br/>        随后，童筱颖被刘婶叫去吃午饭，饭后她才回到自己的房里，坐在床上开始叠千纸鹤。
<br/>
<br/>        这些叠千纸鹤的方纸，其实是马智杰偷拍她的那些相片，她拿去打印店里扫描排版，打印成了一张张方方正正的小彩色照。
<br/>
<br/>        千纸鹤是她叠给马智杰的，希望到时候能随着马智杰一起入土为安。
<br/>
<br/>        他因为她而死，她拿钱安抚马智杰的父母，替马智杰找出杀害他的真凶的证据。
<br/>
<br/>        童筱颖觉得自己能为他做的，也就只有这些了。
<br/>
<br/>        现在，她只坐等林若琴和何明旭落入法网，至于其他的事情，她真的无能为力。
<br/>
<br/>        这一叠，尽在不知不觉中，就是一下午。
<br/>
<br/>        不知道为什么，童筱颖总觉得今天似乎好困，一整天都是脑袋昏昏沉沉，总是在发黑眼晕地想睡觉。
<br/>
<br/>        她叠着叠着，身子软绵绵地趴在了床上。
<br/>
<br/>        合眼前，她看向了卧室门口，房门未关，暮色将至，走廊上的感应灯瞬间灯火通明。只是她这室内，她为开灯。
<br/>
<br/>        所以，当有个人影出现在门口的时候，几乎完全挡住走廊上照射进来的大半光线。
<br/>
<br/>        是昼景吗？
<br/>
<br/>        童筱颖的手中还捏着一只未叠完的千纸鹤，迷迷糊糊地半睁着眼睛，陷入了似睡非睡的状态。
<br/>
<br/>        只见门口，出现了一个穿着件白色T恤，白色轻薄休闲长裤的男人。
<br/>
<br/>        他健硕的胸肌线条在贴身的衣物下隐隐显现出来，身段高大、修长，禁欲感十足。
<br/>
<br/>        不、不是昼景……
<br/>
<br/>        或许是因为常年坐办公室的缘故，昼景的身子骨比较偏瘦。
<br/>
<br/>        所以，他是大哥傅夜沉。
<br/>
<br/>        傅夜沉只身走进来后，童筱颖疲惫得困意来袭，眼皮子再也抬不起了，完全闭上眼睛后，所有的意识在她脑海里消失殆尽。
<br/>
<br/>        听刘婶说，筱颖这一整天几乎都是待在房间里未出来过。
<br/>
<br/>        傅夜沉纳闷童筱颖一个人闷在房间里做什么，于是才来她房间看看。
<br/>
<br/>        童筱颖侧卧在大床上，嘴唇轻阖，眼睑偶尔轻颤一下。
<br/>
<br/>        她身上穿着一件轻盈贴身的白色吊带睡裙，锁骨嶙峋漂亮，香肩圆润白皙，双臂纤瘦光滑。
<br/>
<br/>        再往下，她白色的裙摆下露出一双藕白般修长的美腿，脚趾头纤小白皙，脚背骨线分明。
<br/>
<br/>        童筱颖的睡姿有些曼妙诱人，令人想入非非。
<br/>
<br/>        傅夜沉目不转睛地看着她，只觉有千万道电流通过了他的神经中枢，但表面上他只滑动了一下喉结，清润无暇的瞳仁慢慢地浑浊了些许。
<br/>
<br/>        他盯着她看得越久，思绪就飞得越远。
<br/>
<br/>        蓦地，傅夜沉只觉自己腹下一紧，微微俯身，情不自禁地向童筱颖伸出了手。
<br/>
<br/>        他想触碰她的脸颊，却又在半空中停顿了一下，大手跃过她的身子，拿起她身后的薄毯，轻轻地搭在她的身上。
<br/>
<br/>        正当傅夜沉准备起身离开的时候，深邃的目光，无意间落在了童筱颖裙摆下那白皙的两腿间。
<br/>
<br/>        一丝鲜血顺着她雪白的肌肤，慢慢地滑到了床单上。
<br/>
<br/>        傅夜沉身心一怔，触目惊心地瞪大了眼睛。
<br/>
<br/>        他不假思索地立即将童筱颖从床上捞起，紧张地抱入怀中，疾步离开了房间。
<br/>
<br/>        “痛——”童筱颖一声呢喃，额头上渗出细密的汗珠。
<br/>
<br/>        她本能将双手捂住在肚子上，呼吸急促，心跳的速度恍若已经超过她能承受的极限，腹部一阵阵的绞痛，汩汩热流从两腿间往外涌出。
