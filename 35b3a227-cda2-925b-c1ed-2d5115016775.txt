　　“黄泉之矛！”
<br/>　　巫森轻喝一声，手中黑色的长矛暴刺，如同一道黑色的闪电直击苏莫。
<br/>　　霎时之间，一道巨大的黑色矛影出现了，长达数千丈，霸绝天地，威凌八荒。
<br/>　　这一击威势浩荡九天，势不可挡，强烈的地煞奥义，让得此击地煞之气冲天，威力倍增。
<br/>　　巨大的黑色矛影，洞穿了虚无，将空间平滑的切开，速度快如闪电。
<br/>　　空间甚至来不及崩碎，矛影便一闪而过。
<br/>　　巫森这一击，乃是杀招，就是要一击诛杀苏莫。
<br/>　　一击出，众人皆惊。
<br/>　　不愧是武帝强者，这一招足以秒杀一般的准帝强者了。
<br/>　　整个苍穹世界，或许只有苏莫有希望挡住这一击，当然，只是有希望。
<br/>　　面对巫森强大的一击，苏莫面色凝重，瞬息之间，他体内五枚神丹尽皆催动，他的身体立刻化为了龙象战体，耀眼的金光刺眼夺目。
<br/>　　“杀！”
<br/>　　面对巫森绝杀的一击，苏莫大吼一声，身形不退反进，直接迎着矛影冲了上去。
<br/>　　狂猛的玄力混杂着气力涌入帝皇剑，璀璨的剑光暴涨千丈，剑威无可匹敌。
<br/>　　“斩！”而后，苏莫双手握剑，向着临近到身前的矛影狠狠的一斩。
<br/>　　轰！
<br/>　　帝皇剑斩中矛影，顿时发出一声惊天动地的爆响，声震方圆万里。
<br/>　　轰隆隆！！
<br/>　　矛影被一击斩碎，滔天的冲击波逆卷万里，浩浩荡荡。
<br/>　　呃~
<br/>　　苏莫顿时闷哼一声，身形一震，如同飞驰的流星一般，顿时暴退数百里。
<br/>　　“武帝之境，不过如此！”身形止住，苏莫双眸中精光爆闪。
<br/>　　虽然被击退数百里，但他并没有受伤，只是体内气血被震荡的有些逆流，很快便压制了下来。
<br/>　　但是，从这一击当中，苏莫也能大致的看清了巫森的实力范围。
<br/>　　这个范围，并没有超越苏莫的上限，他心中立刻便有了底气，就算对方还有更强大的绝招，也很难打败他了。
<br/>　　“嗯？”
<br/>　　巫森见苏莫毫发无伤的挡住了他一击，顿时眉头微皱，心中大感意外。
<br/>　　“果然名不虚传！”
<br/>　　巫森冷冷一笑，他早就听闻苏莫是如何如何妖孽，如何如何逆天，本来他还以为传言有些夸大。
<br/>　　但现在一见，传言不仅没有夸大，甚至还有些保守了。
<br/>　　“再接我一招——罗刹之矛！”
<br/>　　毫不犹豫，巫森再次出手了，手中长矛快速刺出。
<br/>　　一道更为巨大的矛影，再次向苏莫袭杀而去。
<br/>　　巨大的矛影之上，有森然的鬼影缭绕，阴森的气息让得此击仿佛从地狱而来，霎是恐怖。
<br/>　　轰隆隆！！
<br/>　　这一击比之前一击威势更加强大，矛影一出，方圆千里之内虚空不断的湮灭。
<br/>　　巨大的矛影裹挟滔天威势，呈排山倒海之势，直击苏莫。
<br/>　　“大苍穹剑术——星辰陨！”
<br/>　　苏莫丝毫无惧，帝皇剑一闪，逆斩而出。
<br/>　　犀利无比的三色剑气，仿佛化为了一颗颗首尾相连的星辰，从天穹之上坠.落，向矛影迎击而去。
<br/>　　刹那之间，星辰陨便和罗刹之矛相击在了一切，仿佛陨石撞击地球。
<br/>　　轰隆隆！
<br/>　　爆响震天动地，罗刹之矛和星辰陨硬撼，纷纷爆炸，化为滔天骇浪，席卷八方。
<br/>　　下方的大地灰飞烟灭，方圆数千里之内，一片混沌。
<br/>　　这一击，两人平分秋色。
<br/>　　所有观战之人，无不震惊的张大了嘴.巴，不可思议的望着这一幕。
<br/>　　苏莫，居然真的能抗衡武帝强者！
<br/>　　这也太惊人了吧！
<br/>　　而且，虽然刚才一击苏莫被击退，但这一击居然丝毫不落下风。
<br/>　　帝释、海明江、闻人天都等人，之前一直面色担忧，但这一刻都不禁露出了一丝笑容。
<br/>　　只要苏莫能抗衡巫森便好，他们也就放心了，他们不求苏莫能打败巫森，只要能保住性命便可。
<br/>　　“这么夸张！”巫森的脸色沉了下来，两击没有打败苏莫，让他也不禁收起了小觑之心。
<br/>　　“巫森，交出宏青璇，我饶你不死！”苏莫面色冷漠如冰，厉声大喝，他现在已经有些把握打败对方了。
<br/>　　“饶我不死？”巫森闻言脸色一黑，他作为苍穹世界唯一的武帝，居然被如此藐视！
<br/>　　而且，藐视之人还是个二十余岁的毛头小子。
<br/>　　“苏莫，你知道武帝和武皇最大的区别是什么吗？”巫森朗声大喝，眸中杀意闪动。
<br/>　　苏莫没有理会对方，心中暗暗思索，如何能斩杀对方。
<br/>　　他的实力并不比对方强多少，只有二十四星宿剑阵，或许有这个希望。
<br/>　　“本帝告诉你，是战魂！”
<br/>　　巫森朗声高喝，言罢，他的背后虚影升腾，立刻出现了一只体长达百丈的妖狼。
<br/>　　此妖狼全身漆黑如墨，身上的毛发闪烁着森冷的寒光，摄人心魄。
<br/>　　此妖狼凶威盖世，模样非常的惊人，赫然长着三颗脑袋。
<br/>　　三颗巨大的脑袋，六只灯笼般大小的眼睛，均是爆射出嗜血的光芒。
<br/>　　苏莫与妖狼的眼眸对视，脑中轰然一震，只感觉的一股寒意，从心底陡然升起，直冲脑门。
<br/>　　“地狱三头狼！”
<br/>　　“天啊！居然是最凶残邪恶的妖兽地狱三头狼！”
<br/>　　“这是战魂，真正的战魂！”
<br/>　　“……”
<br/>　　远处，无数的惊呼之声骤然响起，声音中蕴含着震惊、恐惧、骇然之意。
<br/>　　因为这地狱三头狼，乃是远古异种，是一种非常恐怖的凶兽。
<br/>　　此兽虽然不是神兽，但实力比之一般的神兽，都不差多少。
<br/>　　由此，这种地狱三头狼形成的战魂，威力可想而知。
<br/>　　“战魂！”苏莫低声自喃，眸光紧紧的盯着地狱三头狼，他在此兽的身上，感受到了一丝危险的气息。
<br/>　　武者从武皇境晋升到武帝之境，武魂会脱变为战魂，苏莫以前只听闻过，但还从未见识过真正的战魂。
<br/>　　据闻，战魂极其强大，能让武者的实力大增。
<br/>　　甚至还有传闻，战魂不灭，永生不死。
<br/>　　此言的意思是，只要战魂不被毁灭，武者就不会死亡，哪怕肉身被毁灭成渣，也能存活于世。
<br/>　　这种传闻非常的夸张，但并没有人否认，显然此事很可能为真。
<br/>　　苏莫以前并没有具体了解过战魂，所以也不确定是不是这样。
