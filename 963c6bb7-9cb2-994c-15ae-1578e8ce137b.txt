　　整个苍穹神殿之中，除了苏莫和司空炎、彩芸等几位传人之外，所有的人都是双眼放光，包括海明江和帝释。
<br/>　　帝级中品功法，就算是帝释和海明江这两名准帝强者，也是从未见过，梦寐以求的至宝。
<br/>　　咕咚！
<br/>　　吞咽口水之声不绝于耳，那些天涯海阁和帝玄宫的强者，连帝级下品功法都不曾拥有，更别说帝级中品功法了。
<br/>　　呼！
<br/>　　海明江深深的吸了口气，心中很是意动，帝级中品功法啊！
<br/>　　如果有了此功法，他日后说不定真的有希望晋升武帝之境，实力暴增、寿元大涨。
<br/>　　不过，他心中却是有一道坎，难道为了帝级中品功法，就舍弃整个天涯海阁万年的基业吗？
<br/>　　“不仅是功法，你们每人还可以修炼一门帝级下品武技，甚至是帝级中品武技！”苏莫再次开口，又加了一把火。
<br/>　　其实这些东西，他根本不会保留，就算这些人自愿加入苍穹神宫，他也会传给这些人帝级的武学。
<br/>　　因为这些人的修为，可不是苍穹神宫那些弟子可比，全部都是武皇境武者，而且基本上都是高阶武皇。
<br/>　　对于高阶武皇强者，要想培养他们，也就只有给他们最好的武学了。
<br/>　　“阁主！”
<br/>　　“阁主！”
<br/>　　苏莫话音一落，立刻便有几名天涯海阁的强者忍不住了，轻声呼喊海明江，个个满脸期盼之色。
<br/>　　这几名强者之中，便有炼器堂的胡副堂主，几人眼眸放光，显然是彻底的心动了。
<br/>　　海明江闻言，转头看向几人胡副堂主几人，一眼便看穿了几人的想法。
<br/>　　“你们都愿意加入苍穹神宫？”海明江沉声问道。
<br/>　　胡副堂主等人点头，帝级功法、帝级武技，他们若是舍弃，那真的太愚蠢了。
<br/>　　当然，几人也不是完全经不住诱.惑的人。
<br/>　　他们之所以愿意加入苍穹神宫，其一是因为苍穹神宫现在已经无敌，其二是他们不能反抗苏莫，其三是若真的脱离了苍穹神宫，真的会遭到黄泉魔宗的报复，而他们却无力抵挡。
<br/>　　而加入苍穹神宫，不仅安全无忧，还可以修炼帝级武学。
<br/>　　怎样抉择，根本不需要太多考虑了。
<br/>　　“你们呢？”见胡副堂主几人点头，海明江又看向包括钱副阁主在内的其他人。
<br/>　　其他人亦是全部点头，表示同意。
<br/>　　海明江见此，不由的心中暗叹一声，既然其他人都同意了，他也不必坚持了。
<br/>　　况且，他亦是很心动帝级中品功法。
<br/>　　“老朽愿意带领整个天涯海阁，加入苍穹神宫！”海明江望向苏莫，郑重的说道。
<br/>　　“很好！”
<br/>　　苏莫见此面上露出了灿烂的笑容，只要这些人都愿意加入苍穹神宫，苍穹神宫的实力断层，基本上就能立刻弥补了。
<br/>　　不会再像以前一样，除了他之外，宫中根本没有强者，其他人实力弱的可怜。
<br/>　　“你们所有人，都将成为苍穹神宫的长老，分别按照各自的情况，分配到九大宫！”苏莫目光扫视帝玄宫和天涯海阁众人，朗声宣布。
<br/>　　这些人足有七十余人，自然要打散了，分配到九大宫，才好彻底融入苍穹神宫。
<br/>　　随即，苏莫又命人招来了九名傀儡人，花费了小半个时辰的时间，将众人做好了分配。
<br/>　　帝释被分在了金煌宫，海明江分在了光耀宫、钱副阁主分在了风云宫，胡副堂主这种炼器师，自然分在了火焚宫。
<br/>　　至于苏莫许诺的帝级功法武技，自然也不会反悔，两方势力之人，都可以在各自所在的宫，挑选适合自己的武学。
<br/>　　此事，只要向九宫的守护傀儡人打个招呼就行。
<br/>　　至于天涯海阁并入时宜，苏莫也和海明江讨论了一番，因为天涯海阁在外界还有很多分阁和人手，整合起来颇为麻烦。
<br/>　　最后，苏莫决定，这些分阁保持不变，作为苍穹神宫的店铺，为苍穹神宫赚取财富。
<br/>　　而那些人手，自然也全部成为苍穹神宫之人，
<br/>　　最后，苏莫又和司空炎、闻人天都等人商议了一番招收弟子之事，准备明日正式招收弟子。
<br/>　　因为是九大宫招收弟子，日后需要九大宫培养，所以他征求了一番司空炎等人的意见。
<br/>　　最后几人观点达成了一致，那就是只招收拥有天级武魂的弟子。
<br/>　　这个要求不是一般的高，而是已经高到了极致。
<br/>　　天级武魂的天才，就算是在中洲大地，也是万中无一，可见这要求之高。
<br/>　　如果苍穹神宫按这个标准招收弟子，那真的是十万人、百万人之中才能挑选一两名弟子。
<br/>　　不过，按司空炎的话说，如今苍穹神宫，已经成为了苍穹世界最强的势力，肯定有无数的人想要加入苍穹神宫，就算要求再高，也能招收到很多弟子。
<br/>　　对此，苏莫也表示赞同，毕竟招收弟子，贵精不贵多。
<br/>　　天赋不行的弟子，就算招收千万也不堪大用。
<br/>　　再说了，弟子越多，耗费的资源也就越大。
<br/>　　当然，也有例外，若是有些人武魂天赋不是很高，但有其它方面天赋特别出众，也可以加入苍穹神宫。
<br/>　　比如，属性天赋极其出众，或者是有特殊体质，亦或是精神力特别强大、悟性超绝等等。
<br/>　　这些事情，苏莫和众人商议了一番，确定了具体的考核标准，他便做起了甩手掌柜，离开苍穹神殿，安心回去修养了。
<br/>　　招收弟子之事，则全部由九名傀儡人和司空炎等几位传人去做了。
<br/>　　苏莫回到了自己的住处，便径直进入了练功房之中。
<br/>　　虽然已经过去了数日，但他大三宝术的副作用却依旧存在，他整个人非常的疲惫，实力都下降了不少。
<br/>　　回到练功房，苏莫便屏弃所有杂念，静心打坐调息。
<br/>　　苏莫这一坐，就仿佛是忘却了一切，日升日落，不知时日。
<br/>　　期间，夕儿回来了几次，但都没有打扰苏莫，又悄悄的离开了。
<br/>　　在苏莫静心打坐之时，苍穹神宫却是非常的热闹。
<br/>　　无数的人涌进了苍穹神宫，参加苍穹神宫的入门考核。
<br/>　　现如今，苏莫打败武帝强者巫森之事，早已经以难以想象的速度，传遍了整个苍穹世界。
<br/>　　整个苍穹世界为之沸腾，仿佛是发生了十二级大地震一般。
<br/>　　而后，无数的年轻武者，在各自长辈的带领下，不远亿万里之遥，赶来苍穹神宫。
<br/>　　不过，来到苍穹神宫的人，当得知苍穹神宫招收弟子的标准之后，立刻便被当头浇了一盆冷水。
<br/>　　苍穹神宫，招收弟子的最低武魂标准，居然是天级武魂！
<br/>　　开什么玩笑？
<br/>　　要知道，其他的超级势力招收弟子，就算是地级一阶的武魂天赋，都能通过考核。
<br/>　　而苍穹神宫，居然完全不要地级武魂的弟子，只要天级武魂的弟子。
<br/>　　无数的人哀叹，但也无济于事，最后只能黯然离去。
<br/>　　第一日，苍穹神宫只招收了二十三名弟子。
<br/>　　这二十三名弟子中，还有一大半是散修的老一辈武皇强者，乃是上次在天海城的幸存者。
<br/>　　第二日，招收了二十七人！
<br/>　　第三日，招收了三十一人！
<br/>　　开始的几日，每一日招收的弟子数量，都稀少的可怜。
<br/>　　但半个月后，随着苍穹神宫大开山门、广招弟子的消息传开之后。
<br/>　　来苍穹神宫参加考核的弟子，则是越来越多。
<br/>　　大半个月后，每一日，苍穹神宫都能招收数百名弟子。
<br/>　　一个多月之后，苍穹神宫招收了三万名弟子之后，便关闭了山门，停止招收弟子。
<br/>　　这让后来赶到之人，怨声载道，捶胸顿足，因为错过了这次机会，不知道要等到什么时候，苍穹神宫才会再次招收弟子了。
