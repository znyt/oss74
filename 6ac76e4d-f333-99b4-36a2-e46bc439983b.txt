　　只见一块比人来粗的钢铁砸在杜天易的身上，杜天易的眼中反复无比兴奋起来，体内金属性异象不断显化，这块钢铁已肉眼可见的速度，被吸走了能量。

<br/>　　他很想安慰牛妖王，这已经不是你的别墅，而是我的别墅呀。

<br/>　　不好，涂小安一下子灵光一现，有不好的预感。
<br/>　　整栋别墅塌陷，这动静太大了，而且杜天易全面释放的气息，愈发的恐怖，他这是想引来妖精管理局的人啊。

<br/>　　“不好，杜组长不在，我们恐怕要请示局长了”
<br/>　　工作人员撑着惊骇的脸：“我想不用请示了，我们要去支援，那位三阶生灵散发的气息，仪器数据显化资料，就是杜天易杜组长”
<br/>　　“什么，杜组长在跟三阶妖灵在战斗？”
<br/>　　“我们一组全员出动，将城东郊区一公里内全部封锁，不要有一个普通人，要是有普通人看到了不该看的，直接清除目击者记忆”
<br/>　　“是...”
<br/>　　妖精管理局人影闪动，足足出动了几十位高手，每一位都不下二阶。
<br/>　　...........
<br/>　　别墅废墟内，涂小安看着杜天易吸收了大量的金属物质，从而既然整个人成了金光灿灿的金属人，通体铜皮铁骨，体形也暴涨了一大圈。
<br/>　　“啧啧...不用十分钟，你这里就会被妖精管理局的人包围”
<br/>　　变成金属人的杜天易发出了另类的声音，变得很机械。
<br/>　　“我他吗今天还非要收你做战仆不可了”涂小安看着金属人，眼睛放光，紧接着沉声道：“牛妖王，张天师，这里不需要你们，你们出去，不要让一个妖精管理局的人靠近”
<br/>　　牛妖王道：“主人事情闹得那么大，真的好吗，我们还要不要在温城混啊”
<br/>　　“放心，天塌不下来”涂小安心里有数。
<br/>　　张天师道：“还是我留下来帮你吧，那些来的小喽罗，让牛妖王一人对付就可”
<br/>　　“不用，他成了金属人，刀枪不入，必须用特殊手段才能对付”
<br/>　　咻...咻...
<br/>　　张天师跟牛妖王察觉到很多股气息逼近，也不多BB，直接离开了别墅范围内。
<br/>　　“你好大的口气，一个人对付我，让我看看，你是人还是妖”金属人杜天易通体金光灿灿，铜皮铁骨，让人非常的心惊胆战，而且他的气息也异常的强悍。

<br/>　　砰...

<br/>　　这种金属性能力，远远比想象的要变态。
<br/>　　这样的人，成为妖精管理局的组长一点不过分。
