　　“两辆吉普车正朝教堂的方向过来？什么样的吉普车？”罗昊按着耳麦对布谷问道。
<br/>　　“军用吉普，车门上还有联合政府军的标志。”
<br/>　　布谷打开了狙击步枪的保险，这两辆突然出现的军用吉普让他感觉到了不对劲。
<br/>　　罗昊松开耳麦，对塔尼特问道：“塔尼特小姐，你的父亲今天要来看你吗？”
<br/>　　“不。”塔尼特摇了摇头，问道：“他最近并没有要来这边的计划，一般他来之前都会事先跟我联系。”
<br/>　　“糟了。”
<br/>　　罗昊暗呼糟糕，对着塔尼特说道：“立刻带着孩子到屋子里去！”
<br/>　　“是巴坦伦的人吗？”塔尼特问道。
<br/>　　“如果来的人不是你父亲派来的，那么我有理由相信，是巴坦伦派来的人。”
<br/>　　罗昊拔出了那把使用空尖弹的手枪，拉开套筒确认子弹已经上膛后，按住耳麦对布谷说道：“布谷鸟，拦住他们，别让他们继续前进！”
<br/>　　“收到！”
<br/>　　布谷卸下了XM2010增强狙击步枪上的弹匣，拿起旁边那个装填了穿甲燃烧弹的弹匣装入狙击步枪，拉开枪栓，把子弹推入枪膛。
<br/>　　第一辆军用吉普朝着教堂转完过来的时候，布谷扣动了扳机。
<br/>　　因为在枪口安装了高效消音器的关系，布谷扣动扳机的时候，狙击步枪只发出了一声沉闷，即便是在夜晚都不容易被察觉的枪声，黑色带有红色标记环的弹头朝着第一辆军用吉普的引擎盖飞去。
<br/>　　铿！
<br/>　　钨芯穿甲弹穿透了吉普的引擎盖，击穿了汽车的引擎，引擎盖下立刻冒出了一股黑烟，紧接着有火苗从引擎盖下蹿出来，火势越烧越大，很快引擎盖四周都是有着火焰冒出来。
<br/>　　军用吉普车内的四个联合政府军士兵看到着火燃烧的汽车，大惊失色，拿着AK自动步枪从汽车上跳了下来。
<br/>　　忽！
<br/>　　四个联合政府军士兵刚跳下汽车，一团火球从引擎盖下冲了出来，把引擎盖都是掀了起来，紧接着火焰彻底包裹了吉普车，四个联合政府军士兵拿着枪立刻朝着四周逃开。
<br/>　　轰——！
<br/>　　燃烧的吉普车发生了爆炸。
<br/>　　逃开的四个联合政府军士兵惊魂不定的看着被炸成了废铁，火势依旧在猛烈燃烧的吉普车，他们是被巴坦伦派来“请”塔尼特去巴坦伦秘密别墅做客的。
<br/>　　巴坦伦这个人做事狠辣，他对这次大选势在必得，不容许有任何的差池。瓦斯汀那个软骨头在威逼利诱下，基本已经放弃了大选，现在唯一会对最后大选结果造成影响的就是比洛多。巴坦伦的确是联合政府军的实权者，但是他只掌控了百分之七十的军队，而剩下的百分之三十军队则是掌控在比洛多手中，比洛多手中的军队虽然数量不多，可是他手里这百分之三十的军队却是整个联合政府军中的精锐力量，战斗力非同小可，学习过美国特种部队的作战技巧。
<br/>　　对于这样一支精锐的部队，巴坦伦自然是想要收入到自己手里，可是不管巴坦伦给出什么样的条件，比洛多都不愿意妥协，这让巴坦伦非常恼火。
<br/>　　而且比洛多这个家伙主张全国禁毒，禁止cocaine的种植、提炼和销售，这无疑是对巴坦伦的利益造成了巨大的影响。
<br/>　　瓦斯汀在这次大选的呼声很高，可是巴坦伦并不特别在意，他真正在意的是比洛多，只要比洛多手中还握着那百分之三十的精锐部队就对自己是个巨大的威胁，即便是在大选中胜出，比洛多依旧有可能会对自己的总统宝座造成威胁！
<br/>　　巴坦伦清楚，如果自己想要坐稳总统的位置，就必须要在大选结束前清楚掉比洛多这个威胁。
<br/>　　所以才有了这两辆军用吉普车出现在教堂外，巴坦伦知道在这个教堂中没有武装力量，所以他从自己的警卫队中挑选了八个人出来把塔尼特带走；可是巴坦伦怎么都没有想到，罗昊、齐凡、莱昂、布谷会抢先一步找到塔尼特。
<br/>　　“十环！”
<br/>　　布谷拉动枪栓，退出弹壳，重新推入了一颗子弹，瞄准了从第二辆吉普车中下来的一名警卫队员，手指平稳的扣下了扳机。
<br/>　　咻！
<br/>　　尖锐的破空声中，子弹以四倍音速的速度飞向那名被锁定的警卫队员。
<br/>　　噗嗤！
<br/>　　子弹击穿了警卫队员的脑袋后，依旧保持着超音速飞行，一头扎进了旁边的树干中，将旁边一棵成人手臂粗细的树干击断。
<br/>　　脑袋被打爆的警卫队员向四周喷洒着鲜血，还没完全停止跳动的心脏，每一次收缩扩张都把大量的鲜血挤压出警卫队员的身体。
<br/>　　剩下的七个警卫队员立刻寻找周围就近的掩体躲藏起来，他们意识到情况的不妙。首先在这所教堂中出现除了他们以外的武装力量，就很不寻常，而教堂中未知武装力量的武器装备更是要远远优于他们手中那些AK自动步枪。
<br/>　　消音狙击步枪，穿甲燃烧弹，这绝对不是一般武装力量可以装备得起的。
<br/>　　一名藏在树干背后的警卫队员探头想要找出布谷的位置，可是他的脑袋刚从树干背后探出，一颗子弹就飞了过来，打穿了他的脑袋，被爆头的警卫队员身体摇晃着从树干背后摔倒在地上，子弹的空腔效应使得他的面部扭曲变形，他的左眼球被挤压出眼眶，挂在眼眶外，着实可怖。
<br/>　　“嘿，你的教官难道没有告诉你，遇到狙击手的时候不要藏在树背后吗？树干可抵挡不住狙击步枪的子弹，受到树干阻挡后的子弹，再击中身体，因为弹道轨迹发生了改变，很有可能会对身体造成更大的创伤，加速你的死亡！”
<br/>　　布谷嘴里小声嘀咕着，退出弹壳，再次推入一颗子弹。
<br/>　　教堂外，巴坦伦的警卫队员基本已经猜出了教堂中狙击手的位置。
<br/>　　教堂钟楼，那个地方是视野最开阔的地方；可是即便猜到了布谷的位置，教堂外的六个警卫队员也拿布谷无可奈何。
<br/>　　三颗子弹，打掉了一辆吉普车，收割了两个人头，面对这样的高手，如果没有重火力的掩护，只要露头就是送人头的节奏。
