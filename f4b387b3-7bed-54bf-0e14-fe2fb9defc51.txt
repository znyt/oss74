　　第748章
<br/>　　程大雷正往前行，见到李行哉出现后，下意识放慢脚步。
<br/>　　面对王涛群贼，李行哉呵呵笑着，口中忽然喝了一声：“好话与你说不听，非要老爷我动手么，乖乖地，将骗来的银子分给我一半，我饶你们一命。”
<br/>　　“原来是黑吃黑的，兄弟们，给我动手。”
<br/>　　王涛看模样是个憨直汉子，此刻脸上有些狰狞。他大喝一声，十几个人一拥而上。话说李行哉虽然有些瘸，但移动得竟然颇为灵活，刀未出鞘，只是用刀背砸人。
<br/>　　他习得也是军伍战法，出手狠辣，都是奔着人要害去。另外两个伴当，明显也不是泛泛之辈，虽然没用兵器，但拳脚大开大阖，不多时便撂倒一大片。
<br/>　　程大雷仔细瞧着这一幕，倒是不明白怎么在这里撞见李行哉。
<br/>　　其实不是意外撞见，李行哉本就是过来寻程大雷的。这次藩王入京，李行哉到了，程大雷也到了。程大雷在闲逛，李行哉同样在闲逛。
<br/>　　城隍庙前人山人海，李行哉发现了程大雷，程大雷没发现李行哉。于是便一路跟来过来，却不想恰巧在此地碰见王涛一伙人分赃。
<br/>　　李行哉也是一时兴起，想要戏耍对方。
<br/>　　眼瞅着双方打在一起，李行哉虽然了得，但毕竟双拳难敌四手，渐渐有些失势。
<br/>　　程大雷叹口气：看来还是要自己出手啊。
<br/>　　“呔，贼子休狂，赶快住手。”
<br/>　　程大雷还来不及动作，就有一道炸雷般的声音忽然响起。徐问天不知从什么地方蹦了出来，三两下跳入人群中央。
<br/>　　徐问天之所以出手，不是因为抱不平的义气，而是因为他认出了李行哉。程大雷不肯收留，也不给自己借势出头的机会，倒不如投靠李行哉，未必不是一个机会。
<br/>　　机会眨眼而逝，有些人更善于抓住。只是一闪念的时间，徐问天已经跃出，一拳一脚将几名恶贼打倒。
<br/>　　能教出陈梦这样的徒弟，徐问天的本事也不简单。面对一伙蟊贼，几乎可以说不费吹灰之力，三拳两脚便将他们撂翻在地。
<br/>　　李行哉松了口气，刚才委实有些托大，如果被一伙蟊贼痛揍一顿，传出去也是丢人的一件事。
<br/>　　“多谢好汉仗义出手，不知如何称呼？”
<br/>　　“阁下多礼，某家徐问天，因见这几个蟊贼为非做歹，才打算出手，谈不上谢字。”
<br/>　　这时候王涛一伙狼狈从地上爬起，灰溜溜逃走，李行哉看到这一幕，也并没有阻拦。
<br/>　　他的注意力仍在徐问天身上：“好汉何以为业？”
<br/>　　徐问天面带苦色：“与人看家护院，徒活四十有二。”
<br/>　　“嗳，正是报国立业的年纪，大丈夫怎能操此贱业。”
<br/>　　徐问天无可奈何的摇摇头：“徐某自幼苦练枪棒，也想着为国为民做一些事，奈何空怀良玉，却不逢识宝之人。”
<br/>　　李行哉叹口气，忽然喝了一声：“嗨，你说巧不巧，本王手下正缺人手，不如你来与我做事？”
<br/>　　徐问天表情一怔：“还不知公子尊姓大名？”
<br/>　　“我家主公正是东海逍遥王，歼灭海盗流贼的便是他。”李行哉的手下大。
<br/>　　徐问天眼睛睁大，双手一拍袖口：“徐某有眼无珠，竟然不识得真神，请王爷受我一拜。”
<br/>　　“嗳，多礼了，多礼了。”
<br/>　　李行哉双手捧住徐问天的胳膊，徐问天想拜却拜不下去，二人目光相对，同时开怀大笑。
<br/>　　这次李行哉赴京，一来是为了凑这场热闹，二来就是为了招兵买马。他手下文有文官，武有武将，缺的偏偏就是在战场上杀千屠万的猛将。
<br/>　　看能不能趁着这次藩王会，忽悠几个人去东海落凤城。如徐问天这样的人，他是来者不拒，多多益善。
<br/>　　将徐问天搀起，李行哉目光转了一圈，刚才还恍见程大雷的身影，怎么眨眼就看不着了。
<br/>　　程大雷走在路上，徐神机好奇的问道：“大当家，既然碰着了，怎么不见上一面。”
<br/>　　程大雷负手在身后，慢悠悠道：“你没看到他们两个情投意合，臭味相投，咱们上去瞎凑什么热闹。反正都在长安城，该碰见早晚会碰见的。”
<br/>　　“既然都在长安城，那小公主也该来了吧？”徐神机。
<br/>　　程大雷脚步一滞，哈哈笑了两声：“你问我，我问谁去，算着应该来了吧。”
<br/>　　……
<br/>　　李婉儿出现在皇宫中，仍是她曾经所居的住所，目光所及都是旧物。
<br/>　　李乐天大步从外面进来，挥退了太监宫女。李婉儿屈膝便要行礼。
<br/>　　“免了，免了，父王在世时你也没跪过，何必要跪朕。”李乐天大马金刀的坐下。
<br/>　　“终究是陛下，规矩还是要讲的。”李婉儿还是行了半礼。
<br/>　　李乐天上下打量着李婉儿，眉目中藏不住的溺爱。
<br/>　　“好妹子，你可是瘦了，出落得更加标致了。怎么，六弟又没有欺负你，如果有的话可要与朕说，朕给你做主。”
<br/>　　李婉儿温婉笑笑：“劳皇兄惦记，一切都过得去，六哥可不敢欺负我。”
<br/>　　“怎样，这次回来，就不要走了吧，长安毕竟是长安，其他地方是比不了的。”
<br/>　　“也是住得惯了，不想再颠簸。”
<br/>　　李乐天点点头：“对六弟朕是放心的，这样，多少在宫中住几日，也让为兄尽尽心意。”
<br/>　　“宫中行动不便，我出现在宫中也不合适。”
<br/>　　李乐天再次点头：“可朕总要为你做些什么，这样，你喜欢什么，说与朕听，朕做主赐给你。”
<br/>　　李婉儿笑笑：“别的我也不要了，只是有一方手帕，是我以前绣着玩的，是个念想，便想着带走。”
<br/>　　“这当然没问题，你这屋子朕经常着人收拾，可不敢弄坏了你的东西，那手帕在哪里，当是能找到的。”
<br/>　　二人在屋里，又说了许多话，无非是年少时的胡闹，这些年的辛苦。找到那方手帕后，李婉儿起身告辞，李乐天也没有亲自送，只是让黄门一直送到她在长安城的住处。
<br/>　　兄妹二人相谈甚欢，互诉衷肠，但告别时心底都轻轻叹了口气。
<br/>　　到底是意难平。
