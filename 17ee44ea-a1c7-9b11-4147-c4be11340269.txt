
<br/>　　<br/>　　    彼岸花开两朵，一在彼岸二在忘川河旁，据说彼岸花生长年限十分悠长，几百上千年都能有，在如此漫长的成长岁月中，吸收了忘川河上数不清的阴气和亡魂气息，所以这花有两样功效非常难的，阳间是根本都寻觅不到的。<br/>　　<br/>　　    彼岸花能起到镇压孤魂厉鬼的作用，对生人能稳固三魂七魄，一般人可能没啥感觉，但对道士一系来讲却堪称至宝了，但凡是那些画符，念咒，驱鬼降妖伏魔的道门中人肯定都想要，因为这玩意能坚固他们的三魂七魄，你想啊念几段咒语如果就岔气了，还怎么抓鬼呢？<br/>　　<br/>　　    画符的经常要口吐精血或者咬破指尖来写符箓，血吐多了人受不了咋办？<br/>　　<br/>　　    这彼岸花人用了，就相当于是起到一个加持的作用，延长功效，增强持久力，呃，有点像那个什么神油似的，你们懂吧？<br/>　　<br/>　　    茅小草说道：“你见过我的那盏引魂灯，它专门用来镇压孤魂野鬼，据说当年是张道陵随身之物，不过这灯有一个弊端就是每隔一段时间得需要把一朵彼岸花炼进去，很不巧的是到了我这一代，引魂灯恰好就快要失效了，如果没有彼岸花的话再有几年它就是个摆设了，所以我才想要来阴曹地府一趟，而不是特意想要瞒着你的，认没认识你我这个念头都不会改变的”<br/>　　<br/>　　    王惊蛰说道：“你知道我有可能活不过三十岁的，就是因为我受了天谴的原因，其实这彼岸花对我来讲也未尝一点用没有，天谴就是诅咒，老天咒我活不过三十就浑身溃烂流脓生疮，如果我能让自己的魂魄更加坚韧一点的话，就有可能推迟老天的诅咒，或者慢慢的消磨掉天谴，如果我之前没有受伤的话，可能我也会选择走一趟阴曹地府了”<br/>　　<br/>　　    茅小草叹了口气，说道：“你这么说，是想让我心里好过一点么？”<br/>　　<br/>　　    “不是，是实情！”其实，王惊蛰后面还有句话没有告诉茅小草，就是不从良水河进入阴间，他也有方式能够来阴曹地府一趟。<br/>　　<br/>　　    在朝歌，云梦山后的王家祖坟地，就有一处被埋葬了的鬼门关，王惊蛰也并不一定急于非得在这时候就进阴间，至少他还有好几年的时间可以去等待。<br/>　　<br/>　　    只是这一次，她进来了，他也就跟着来了。<br/>　　<br/>　　    有些事，挂在嘴上就没意思了，你提了反倒是不值钱了。<br/>　　<br/>　　    王惊蛰和茅小草走向忘川河边的时候，陈三岁正皱着眉头一路寻觅着，从那一头开始往这边捋，眼睛始终都盯着忘川河中那些被禁锢住的亡魂厉鬼，他俩之前猜测的没错，对方确实打算要借尸还魂的。<br/>　　<br/>　　    之前，陈三岁曾经花费过大价钱请了个出马仙，于是这出马仙请神上身，将那女人在阴间的魂魄给招了上来，他就知道对方没有进入轮回转世投胎，而是被禁在了忘川河中。<br/>　　<br/>　　    陈三岁这才想要冒险一搏，进入阴曹地府将那女人的魂魄给带出来。<br/>　　<br/>　　    别人想要的是彼岸花，陈三岁要的是借尸还魂。<br/>　　<br/>　　    除了他以外，河岸旁还有不少人正在寻觅着彼岸花，这花其实生长的并不多，有些可能年头太短还没有成型，就是摘了也没有什么用，能长成型的也不多，有一些长年累月下长成的也被采走了，所剩下的可能也就那么几朵而已了。<br/>　　<br/>　　    忘川河里的亡魂厉鬼，看见有生人靠近，水中顿时就翻腾起来，一双双枯瘦的手爪子伸出水面挣扎着，撕心裂肺的吼叫声此起彼伏，这些亡魂的模样多数都很凄惨，很少有保存完整的，生前基本都死于非命或者怨气滔天的。<br/>　　<br/>　　    “狼多肉少啊·····”王惊蛰感慨了一句，说道：“彼岸花估计能采的也就那么几朵而已，这么多人想摘的话哪里够分啊，这不得人脑袋打成狗脑袋，才能抢得到啊”<br/>　　<br/>　　    茅小草说道：“这个世界本来就是残忍的，第一条规则不是相敬如宾而是弱肉强食，你行你就能生存下去，不行就在后面眯着，狼行千里吃肉，狗始终都是吃屎的”<br/>　　<br/>　　    王惊蛰哦了一声，点头说道：“干就完了，别犹豫？”<br/>　　<br/>　　    茅小草傲然一笑：“本来和他们也不熟啊！”<br/>　　<br/>　　    两人的性子都属于那种不惯着类型的，大家要是朋友可以坐下来好好谈，既然不是朋友，那就手下见真章得了，肯定不带惯着你的。<br/>　　<br/>　　    河岸两旁的人从一直往下游捋着找，彼岸花其实很好认，彼岸那一朵是白色的，可以稳固人的三魂七魄，忘川河旁的是红色的，能够镇压亡魂厉鬼。<br/>　　<br/>　　    花很显眼，并不难找。<br/>　　<br/>　　    就在这个时候，河岸旁的人忽然出现了骚动，一道身影快速的跃过几人身旁，径直朝着前方岸边冲了过去，几道眼神全都掠向了那边，离岸三丈远的忘川河里凸起的一块礁石上，一朵白色的小花正迎风招展，花开的很是鲜艳。<br/>　　<br/>　　    “唰”那道身影突然跃起，直接从岸边朝着那块礁石上蹿了过去，这人单脚踩到石头上后，迅速弯腰伸手就要将那朵彼岸花摘走。<br/>　　<br/>　　    但还没等他手碰到花上，一把木剑不知从哪奔着他背后就飞了过去，对方似乎有所察觉，身手利索的用力蹬了下石面，人就腾空而起了，可让人没有想到的一幕发生了。<br/>　　<br/>　　    就在他跃起升空的一瞬间，至少有四五个人同时朝他出手，两道符纸快速掠过，在离他还有一段距离时“噗”的一下就爆了，两团火球朝着人砸了过去。<br/>　　<br/>　　    这时候，那人才脸色大变，额头上都急出了冷汗，人在半空中完全没有借力改变方向的可能了，只能硬生生的挺着了。<br/>　　<br/>　　    “砰”一道符纸化出的火球砸到了他的肩膀上，这人身子一歪顿时就朝着忘川河一头栽了下去，同时还随着一声绝望的嘶吼。
