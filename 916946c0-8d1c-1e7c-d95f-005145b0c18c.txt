　　听闻龙城之言，龙腾心中感到好笑。
<br/>　　不过，他并未表现出来，淡淡的说道：“生死各安天命，他们交手全看本事了！”
<br/>　　“这是当然！”龙城冷笑一声，随即便不再多言，而是看着迷宫中的情况。
<br/>　　此刻，迷宫之中，宽敞的通道之中，司空炎和巫铨正在对峙。
<br/>　　这一幕，不仅是龙城和龙腾在关注，很多龙族之人均是在关注。
<br/>　　毕竟，龙城和龙腾都是万古天骄级别的天才，两人的奴仆交手，可以说是两位天骄的较量。
<br/>　　通道之中，巫铨面色冷漠的望着司空炎，道：“你对主人出言不逊，你的命我收了！”
<br/>　　巫铨心中杀机涌动，既然是龙城的吩咐，他就绝不会让眼前之人逃脱。
<br/>　　司空炎闻言不屑，他知道对方跟着自己进来的，但他根本没有在意。
<br/>　　“你若是挡住我一招，我便饶了你！”司空炎嗤笑一声说道，现如今，他和对方的修为，都被压制在了真神境一品低级。
<br/>　　他还就不信了，一个小小的奴仆，还能和他抗衡不成。
<br/>　　“找死！”
<br/>　　巫铨闻言，眸中杀机一闪，一道雪亮的刀光，在他的手中闪现，而后一刀斩向司空炎。
<br/>　　这一刀凌厉无匹，雪亮的刀光，如同九天闪电，裹挟着无匹的威势，直击司空炎。
<br/>　　“你太弱了！”司空炎见此，冷笑一声，一掌猛然拍出。
<br/>　　这一掌毫无花哨，随着手掌拍出，一股恐怖无比火属性力量爆发而出，让得通道之中，温度暴增数万倍。
<br/>　　一个如同磨盘般大小的火球，从司空炎的掌中拍出，如同一个小型的太阳，裹挟着毁天灭地的火属性之力，瞬间便击中了刀光。
<br/>　　轰！
<br/>　　一声爆响，刀光崩溃湮灭，被火球焚化了虚无，而火球威势不减，直击巫铨。
<br/>　　“什么？”巫铨见此，顿时大惊失色，他完全没有料到，司空炎的实力居然会强大到这种地步。
<br/>　　由于距离太近，火球的速度极快，几乎是瞬息之间，火球便要击中巫铨。
<br/>　　即便火球还未临身，巫铨已经心中大骇，因为，他感觉到自身仿佛置身在熔炉之中，身体都快要融化了。
<br/>　　“给我破！”
<br/>　　无法躲避，电光火石之间，巫铨一声怒喝，手中长刀狠狠的劈了出去。
<br/>　　与此同时，他全身源力沸腾，包裹全身，抵御火焰之力。
<br/>　　轰！
<br/>　　又是一声爆响，长刀斩中了火球，但是火球的威力远超巫铨的想象。
<br/>　　啊！
<br/>　　一声凄厉的惨叫声响起，只见巫铨的长刀崩飞了出去，火球重重的击在了巫铨的身上。
<br/>　　恐怖的火焰之力炸开，让得巫铨如遭雷击，整个人倒飞而出，撞击在五百丈之外通道尽头的墙壁上，让得墙壁剧震。
<br/>　　而此刻的巫铨，极为凄然，整个人漆黑一片，衣裳和头发早已化为了飞灰。
<br/>　　不仅如此，其整个人已经缩小了一圈，身体大量的血肉，已经被焚化。
<br/>　　不过，好在没有伤及根基，虽然外表狼狈，但是还不算受到了重创。
<br/>　　嗖！
<br/>　　巫铨的反应极快，知道自己不是司空炎的对手，身形一转，立刻向另一个方向逃去。
<br/>　　这里通道四通八达，纵横交错，非常适合他逃走。
<br/>　　“居然还能逃走？”司空炎见此，微微惊讶，不过，他也没有太多的意外。
<br/>　　毕竟，他根本就没有用全力，他也没想一招灭杀对方。
<br/>　　他要为苏莫考虑，万一杀了这个巫铨，让龙城恼羞成怒，把事情闹大，他和苏莫的身份很可能暴露。
<br/>　　现如今，苏莫到处都是敌人，天王宫、地王宫、终极神王殿，都是无法抗衡的存在，他不得不谨慎。
<br/>　　不过，即便他没有用全力，一般的天才，也不可能挡住他这一招。
<br/>　　看来这个巫铨，实力还是很强的，能成为龙城的奴仆，果然不一般。
<br/>　　少倾，司空炎也没有逗留，向着另一条通道走去，来这里寻找宝物才是最重要的。
<br/>　　外界。
<br/>　　所有看到司空炎和巫铨交手的人，均是震惊不已，目瞪口呆。
<br/>　　“这……龙腾的这个奴仆这么厉害？”
<br/>　　“一招就击败了巫铨？”
<br/>　　“巫铨的战力可是非常的强，堪比一般的妖孽天才了，居然挡不住一招！”
<br/>　　很多龙族之人，霎时间议论纷纷，龙腾在哪里找到的这么厉害的奴仆？
<br/>　　此时此刻，龙城的脸色阴沉无比，他简直不敢相信，巫铨居然连一招都接不住，那个狗奴才居然这么厉害。
<br/>　　他心中愤怒到了极点，前段时间他败在了龙腾的手中，没想到自己得意的奴仆，居然也败在了龙腾奴仆的手中。
<br/>　　“龙城，你这个奴仆巫铨不错，居然能接住炎空一招不死！”龙腾微微点了点头，对龙城说道，言语之中充满了赞扬。
<br/>　　“你……！”龙城顿时脸色铁青，这是赤果果的打量，一招就被击败，对方居然还表扬！
<br/>　　“龙腾，你这奴仆天赋果然厉害啊，在哪里收服的？”龙洌看着龙腾，沉声问道。
<br/>　　“偶然的机会，我也没想到他这么强！”龙腾笑了笑，并未多言。
<br/>　　司空炎能一招击败巫铨，他没有任何的意外，若是司空炎没有一招击败巫铨，反倒是会让他意外。
<br/>　　若是现在，让他和司空炎同阶一战，他都没有十足的把握战胜司空炎。
<br/>　　“龙腾，你另外那个奴仆怎么样？”
<br/>　　修为最高的龙族女子龙双双，观察了一番迷宫中的苏莫，向龙腾问道。
<br/>　　既然这个叫炎空的奴仆这么厉害，那另外一个奴仆，想必也不一般。
<br/>　　“也还不错吧！”龙腾笑了笑道，并未多言，苏莫的厉害，他就算是说出来，也不可能有人相信。
<br/>　　当然，他不可能说出来。
<br/>　　“想必另外一个奴仆要差点，哪有这么多厉害奴仆！”龙烨笑着说道。
<br/>　　他们的奴仆，都是千挑万选，万里挑一的存在。
<br/>　　龙腾才来龙族多长时间，怎么可能碰巧找到两个天赋都妖孽的奴仆？
<br/>　　“哼，难怪你这个狗奴这么嚣张，原来是天赋妖孽。”龙城冷哼一声，冷冷的说道。
<br/>　　同时，他心中在思索，虽然巫铨败给了那个炎空，但是还有机会。
<br/>　　龙腾的另外一个奴仆，不可能这么厉害，若是被巫铨碰到，必然不会放过。
