　　刘四还记得当时抱她在怀里，逗她发笑的那种喜悦。不管怎么说，总是自己的女儿啊……
<br/>　　“爹，不要丢下我，爹……”
<br/>　　小女孩可怜兮兮地看着他，伸出了两只手，似乎想让他抱抱。
<br/>　　女儿……刘四也伸出了手。
<br/>　　叶少阳看见这一幕，立刻就明白了，大喝一声：“刘四，你不要被蛊惑了！我说过，斩断魂之羁绊，其实是为了她好，让她可以安心去投胎！”
<br/>　　刘四一个激灵，清醒过来，刚要去剪，又听见了那小女孩的呼唤，还有边上那个小男孩，那是自己第三个孩子……
<br/>　　两人一起叫他，泪流满面。刘四虽然理智上知道叶少阳说的是对的，但是面对两个孩子的苦苦哀求，实在不忍心去斩断魂之羁绊，浑身颤抖起来，陷入了剧烈的挣扎之中。
<br/>　　时间一分一秒过去，叶少阳在边上看着，也是干着急，只能不断劝他。突然想到什么，目光在院子里搜索，找到了躲在院子角落的刘四媳妇。
<br/>　　刘四媳妇怀里抱着他们的孩子，在墙角呆滞地看着这边，三个小鬼并没有显形，她看不见他们，叶少阳也故意没有给她开眼，当下冲她大声喊，希望她能劝劝刘四。
<br/>　　“四儿！你快点照叶道长的话去做啊！那些孩子，都已经死了，他们不再是我们的孩子了，我们应该去投胎转世，你送他们走，才是为了他们好啊！我们的孩子小顺在我怀里啊，就算是为了小顺，你也一定要去做啊，啊，小顺醒了，小顺快给你爹叫两声！”
<br/>　　说着用力晃了晃襁褓中的孩子。孩子也很给力地哭了起来。
<br/>　　刘四听见小顺哭，浑身一颤，咬了咬牙，用神识斩断了与小姑娘之间的魂之羁绊，之后又斩断了第三条……
<br/>　　刘四瘫软在地上。
<br/>　　“天地清明，拨云见日，一气东来，魂归无极！”叶少阳念了一遍咒语，将一张灵符贴在了阴阳镜上，大拇指按在上面，大叫一声：“疾！”
<br/>　　那一束将三个小鬼困在半空中的光，突然往回收紧，拉扯着三个小鬼一起被灵符收了进去。
<br/>　　叶少阳一拍腰带，抓住朱砂笔，在灵符上写下了一个“敕”字，用力吐出一口气，手拈灵符，站了起来。
<br/>　　大功告成……
<br/>　　叶少阳起身走到张道长面前，把封印着三个小鬼的灵符递给他。
<br/>　　一切结束的太快。张道长还有点没回过神来，怔怔地看着手中的灵符，一时间呆若木鸡。
<br/>　　他身边那几个弟子，也都是瞠目结舌地望着叶少阳，尤其是那几个之前嘲讽过叶少阳的，在捉鬼的过程中，虽然也配合，但是内心巴不得叶少阳失败，结果……事情居然超乎想象的顺利，他们一个个都觉得脸上火辣辣的疼，仿佛是被人扇了巴掌，只好低下头尽量不去看叶少阳。
<br/>　　不过，他们明显是想的多了：叶少**本就没看他们，更没想过挤兑他们，虽然说起来有点装逼，但实情就是这样：这种级别的法师，根本就没有让他较劲的动力，连打脸的欲望都没有。
<br/>　　别说是他们，就算是他们的师父张道长，叶少阳也根本不放在眼里，他把灵符交给张道长之后，在众目睽睽之下又回到了刘四身边，刘四泪流满面，目光呆滞地望着之前那三个小鬼消失的方向。
<br/>　　“已经没事了。张道长会送他们去投胎转世的，你不用担心。”叶少阳蹲在他的面前安慰道。
<br/>　　刘四逐渐回过神来，看看叶少阳，又看看张道长，说道：“那我家小顺……”
<br/>　　“没事了，他的哥哥姐姐们不会再来纠缠他，你们也不用背井离乡了。”
<br/>　　刘四媳妇听见这话，也抱着儿子走了过来，拉着刘四一起跪在地上，要给叶少阳磕头。
<br/>　　叶少阳一向不愿意受人大礼，急忙侧身避让，指着张道长说道：“你们谢他吧，他待会还要送三个小鬼去阴司呢。”
<br/>　　刘四夫妻只好对着张道长磕头。
<br/>　　张道长感到脸上火辣辣的，急忙上前扶起夫妻二人，冲叶少阳尴尬地赔笑：“叶道长你这样可真是比骂我一顿还难受啊，可是折煞我了。”
<br/>　　叶少阳道：“我认真的，咱俩就算是功劳一人一半吧。行了我走了，这还有两棵槐树，已经成精……功德也送你了。”
<br/>　　“多谢叶道长。”张道长对叶少阳拱了拱手，说道：“叶道长别忙走，我还有两件事要跟你说。”
<br/>　　“什么事？”
<br/>　　张道长招呼几个弟子，去弄些硫磺过来，让他们把两棵树给烧了，自己拉着叶少阳来到刘四媳妇跟前，指着她怀里的婴儿，说道：“叶道长，你看这孩子，天庭饱满，道纹修长，是个修道之才，最重要是，他这么小一个婴儿，被三个恶鬼附身了这么久，依然没事，说明体质特殊，机缘也这么巧合，将来若是修道，或有大造化。”
<br/>　　叶少阳打量了那孩子一会，问张道长：“你有什么打算？”
<br/>　　“贫道是启云门的监院，启云门是终南山分支，等这孩子长大一些，我可以送他去宗门修道……若是真有天赋，是有机会得到内门真传的。”
<br/>　　原来张道长是终南山分支弟子，也算是名门出身。他说的没错，终南山虽然在古时候就四分五裂，风头不如茅山和龙虎山，但也是五大山门之一，底蕴还是有的，在这样的大门派，不怕学不到真本事，就怕是天赋不够，没资格学太深的法术。
<br/>　　“既然张道长有这想法，我也放心的。只是不知道孩子父母愿不愿意。”叶少阳说道。
<br/>　　张道长立刻跟刘四夫妻沟通，面对救命恩人的提携，刘四夫妻当然一口答应下来，刘四媳妇唯一关心的问题就是：当道士能不能娶媳妇。
<br/>　　叶少阳猛然想到了自己。听师父说，他当时要收自己为弟子的时候，自己的母亲关心的也是这个问题。
<br/>　　想到青云子，想到多年没有见过面的母亲，叶少阳的鼻子突然有点发酸。
<br/>　　刘四媳妇在得到确定的回复之后，也就释然了，求张道长和叶少阳给他儿子取个名字。叶少阳哪懂什么取名，于是推给张道长。
