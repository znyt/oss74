　　在楚风眠说话的时刻，他的身形也是动了。
<br/>　　他站在原地，突然伸出手掌，直接化为了战龙之爪，猛然一爪，冲着那九幽锁，直接横扫过去。
<br/>　　这九幽锁，打在战龙之爪上，突然暴起一连串的火花。
<br/>　　“生死境三重，废物一样的修为，也想要跟本尊抗衡？既然你想先死，那本尊就先杀了你！”
<br/>　　锁魂魔尊看到楚风眠出手，脸色阴冷道。

<br/>　　锁魂魔尊再度出手，突然之间，这九幽锁再一次向着楚风眠轰击过来，不过这一次这九幽锁上，却是散发出了一种透人心魄的漆黑。

<br/>　　鲜血魔尊急忙提醒道。
<br/>　　不过他的声音还未完，楚风眠突然出手了，从他的身边，无数金色火焰，突然燃烧起来。

<br/>　　看到九幽锁都被逼退，锁魂魔尊大吃一惊道。
<br/>　　这九幽锁，可是他亲自炼制的一件天级灵器，正是最为适合他的一件灵器，勾人魂魄与无形之中，霸道无比。
<br/>　　可眼前，居然是被这金色火焰，直接逼退。
<br/>　　“龙炎。”
<br/>　　看到这金色火焰，锁魂魔尊很快便是明白过来，这火焰到底是什么。
<br/>　　唯有龙族的龙炎，在足以破开他的九幽锁。
<br/>　　“果然，看来传闻之中，你得到了颜古龙帝洞府的传承，这传闻不假，正好，你手中的祖龙至尊剑，也会被我得到！”
<br/>　　锁魂魔尊看着楚风眠，猛然长啸一声。
<br/>　　“九幽兵道！”
<br/>　　在锁魂魔尊手中的九幽锁，却是再次变化，这一次他并不在攻击楚风眠的魂魄，而是直接轰击这楚风眠的肉身。
<br/>　　九幽锁就算是并非轰击肉身所用的灵器，但是毕竟位列天级，就算是用来轰击肉身，也是有着极大的威力。
<br/>　　楚风眠的境界，不过是生死境三重的境界，在锁魂魔尊看来，就算是他动用力量，跟楚风眠硬拼，楚风眠也不可能是他的对手。
<br/>　　“死！”
<br/>　　九幽锁如同一道鞭子一般，冲着楚风眠轰击过来，那一道铁钩，甚至是要将楚风眠的胸膛，都给直接贯穿。
<br/>　　“以力相博？”
<br/>　　楚风眠眼神中露出几分冷笑。
<br/>　　这锁魂魔尊，如今无法破解楚风眠的龙炎，居然是打算以力量来对付楚风眠。
<br/>　　这可正中楚风眠的下怀，比拼力量，楚风眠可不惧怕任何人。
<br/>　　“战！”
<br/>　　楚风眠吐出一个字，战龙之爪，顿时凝聚出来，跟那九幽锁，横扫而去。
<br/>　　“轰！”
<br/>　　两者交手，这锁魂魔尊顿时感觉到他的双臂，都被直接震麻。
<br/>　　楚风眠的力量，简直是远远超越他的想象，他手中的九幽锁，都是被直接震飞出去。
<br/>　　“这是什么力量？这小子的力量，怎么可能如此浑厚！”
<br/>　　锁魂魔尊吃惊的喊道。
<br/>　　他的境界，已经是达到生死境巅峰，就算是他并非是以力量出名的武者，但是比拼力量，他也想不到，居然会输给一个生死境三重的楚风眠。

<br/>　　楚风眠大笑一声，猛然一掌，直接冲着锁魂魔尊轰杀过去。
<br/>　　楚风眠的背后，一道龙影就在他的背后，疯狂的咆哮着，每一声咆哮，都镇人心魄，令楚风眠的气势节节攀升。

<br/>　　突然之间，在旁边的迷瞳魔尊，突然窜了出来，他的身形猛然出现在了楚风眠的面前，一双眼睛深邃无比，楚风眠对视的一刻，便是感觉到一道巨大无比的魔手，就在这双眼睛中伸了出来，似乎是要控制楚风眠。

<br/>　　楚风眠冷哼一声，他的目光入剑，猛然向着那迷瞳魔尊刺了过去。

<br/>　　这迷瞳魔尊的双眼被直接刺瞎，惨叫一声，迷瞳魔尊的一身实力，都是依靠这一双魔瞳。

<br/>　　他万万料想不到，楚风眠的隐藏，居然是如此之深。
