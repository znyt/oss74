　　此时，帝释天仰天大笑，心情激动到了极点。
<br/>　　想当年，他获得凤血的时候，也是如此开心，可凤血只能让他长生不死，却不能让他长生不老。
<br/>　　然而，凤血的缺陷，能够用龙元来弥补，所以他将目标放在龙元上。
<br/>　　而现在，他的愿望终于能够实现。
<br/>　　屠龙成功，龙元也在自己的手中，只要吞服炼化龙元，他不仅能够长生不老，实力还会暴增，突破神境，不再是梦想。
<br/>　　“这就是龙元，真是好强的生命力。”
<br/>　　望着帝释天手中的龙元，现场没人能够保持平静，目光中满是炙热。
<br/>　　慕清霄心中也难免有些激动，但他并未直接冲上去，因为苍龙，还没有死透。
<br/>　　然而，就在帝释天猖狂大笑之际，所有人都认为已经死去，没有一丝生命力的苍龙，居然动了起来。
<br/>　　“啊啊啊......”
<br/>　　帝释天就站在苍龙的身旁，自然成为它的首要攻击目标，巨大的龙爪，直接将帝释天牢牢抓在空中。
<br/>　　在苍龙强大的力量下，即便帝释天拥有千年修为，甚至不死之身，也是面露痛苦之色，双手一松，龙源直接高高抛起。
<br/>　　这种机会，断浪等人有怎会放过，直接纵身而上，想要夺取龙元。
<br/>　　慕清霄冷笑一声，身影一闪就来到上空，将散发着浓郁生命力的龙元，掌控在手中。
<br/>　　看到龙元落在慕清霄手中，聂风和步惊云总算是松了口气，也并未起身，依旧盘坐在原地，快速调息起来。
<br/>　　苍龙的力量越来越大，帝释天痛不欲生，望着龙元落入慕清霄手中，愤怒道：“骆仙按照计划行事，怀灭快点动手！”
<br/>　　此时，破军，皇影和断浪已经御剑来到慕清霄面前，怀空也快速赶来，开口道：“慕清霄，交出龙元。”
<br/>　　闻言，慕清霄握着龙元，微笑道：“龙元既然落在我手中，看来是和我有缘，你们要是有本事的话，就自己来取吧。”
<br/>　　听到慕清霄的话语，怀灭四人相视一眼，他们都知道后者的厉害，当下就达成协议，联手对付慕清霄，至于龙元的归属，稍后再议。
<br/>　　“慕清霄，交出龙元。”
<br/>　　破军拔出双剑，直接朝慕清霄冲杀而去，怀灭和皇影相视一眼，也同意联盟。
<br/>　　断浪自然不会落后，他可没忘记，当初慕清霄一招重创他的仇恨。
<br/>　　刀光剑影，剑气纵横，怀灭和皇影皆是天境四层，而断浪和破军则是天境二层，可对于慕清霄来说，就是四个小喽啰罢了。
<br/>　　只见他凝气化剑，修为依旧压制在金丹六层，游刃有余的对付四人。
<br/>　　战斗愈演愈烈，五人身影纵横交错，战场上地动山摇，碎石飞射，越是往下战斗，怀灭等人越是心惊。
<br/>　　因为，慕清霄仅靠着一人一剑，还是一柄灵气所凝聚的长剑，轻松自若的对付他们四人。
<br/>　　神将已经动用全力，却也没有直接下死手，他只需等待帝释天从苍龙爪下逃脱，如此一来，不仅能够拖延住慕清霄，还能拖延住断浪等人。
<br/>　　毕竟，他可是清楚的很，断浪几人对龙元的渴望。
<br/>　　战斗渐渐步入白热化阶段，慕清霄望着破碎的长剑虚影，以及渐渐脱离龙爪的帝释天，嘴角勾起一丝邪异的笑容。
<br/>　　望着朝自己袭杀而来的断浪，慕清霄目光瞬间冷冽下来。
<br/>　　霎时间，一股恐怖的剑意自体内爆发而出，一柄长剑瞬间凝聚而成，以迅雷不及掩耳之势，径直朝断浪斩去。
<br/>　　慕清霄的这一剑快若奔雷，当断浪反应过来，长剑已经抵达他的面前，寒芒一闪，断浪的身体突然僵硬在空中，眼中满是不可置信之色。
<br/>　　而下一秒，他的面部就隐隐出现一道血痕，血痕越来越大，逐渐延伸，然后在破军几人骇然的目光中，断浪的身体直接分开，鲜血与内脏全部自空中洒落。
<br/>　　看到慕清霄一剑斩杀断浪，破军吞咽起口水，心中已经萌生退意，毕竟他的实力，与断浪在伯仲之间。
<br/>　　后者能一剑斩杀断浪，也就表示能够轻松的斩杀自己，破军真的没想到，慕清霄的实力居然这么恐怖。
<br/>　　皇影也感受到慕清霄的恐怖，望着他手中的龙元，又看了眼帝释天，咬牙直接御剑离开此处。
<br/>　　毕竟，跟龙元比起来，还是自己的性命比较重要。
<br/>　　皇影率先离开，破军也不敢在多做停留，化作流光，径直离开神龙岛。
<br/>　　否则，等帝释天从龙爪下逃脱，他们都得死，毕竟他们刚刚的行为，可是背叛了天门，而天门不允许有叛徒，叛徒的下场就是死。
<br/>　　怀灭望着皇影和破军离去的身影，神色愤怒无比。
<br/>　　断浪已经死亡，两人又直接逃离，现在就只剩下他一人，他可没有独自一人对付慕清霄的能力。
<br/>　　“呃呃呃......龙元是本座的东西，你们谁也别想夺走！”
<br/>　　这时，帝释天使出全部的力量，硬生生从垂死挣扎的苍龙爪下逃脱，身影化作残影，瞬间就来到慕清霄的面前。
<br/>　　“非常的好，慕使者，你居然帮本座获得龙元，击杀断浪这个叛徒，以后你就是本座的左膀右臂，快将龙元给我。”
<br/>　　慕清霄似笑非笑的摇头道：“帝释天，难道你聋了不成，我刚刚就说过，想要龙元，有本事就自己来取。”
<br/>　　听闻慕清霄讽刺的话语，一旁的怀灭彻底愣住了，宛如自己听错一般，旋即，神色就变得犀利起来，他当初就觉得后者加入天门，肯定有目的。
<br/>　　现在看来，他的直觉并没有错，而让他感到意外的是，慕清霄居然敢当着帝释天的面，顶撞辱骂他，后者的自信究竟是从何而来。
<br/>　　帝释天呆立当场，如若不是戴着寒冰面具，表情肯定非常的精彩。
<br/>　　“呵呵......慕清霄，你是在跟本座开玩笑吧，快将龙元交给本座，念在你天赋过人，又为天门做过不少贡献的份上，本座可以既往不咎。”
<br/>　　“既往不咎，真是笑话，你以为我进入天门是为什么，给你当仆人，别开玩笑了，我还得好好感谢你，若不是你，我也不会如此轻而易举的拿到龙元。”
