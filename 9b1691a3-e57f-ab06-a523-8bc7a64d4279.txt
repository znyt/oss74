        傅氏集团现在每天下午的三点半到四点是员工下午茶时间，一楼员工餐厅里有各种各样的花式点心、咖啡、奶茶，以及茶和果汁应有尽有。
<br/>
<br/>        童筱颖也跟随大众潮流，和同部门的同事们一起去了餐厅享用下午茶，却不料遇到了一同来吃茶点的落以柔，而且还坐在了同一个圆桌前。
<br/>
<br/>        “你以后跟傅总做爱的时候下口轻点，毕竟傅总是要出去谈商务合同的。若是每天脖子上都顶着吻痕，手背上都落着齿印，这像什么样子？”落以柔身着职业装，端坐着身子，夹了一块方糖放入了自己的咖啡杯中，一边拿起茶匙搅动着咖啡，一边像一副御姐范模样的训斥着童筱颖的不是。
<br/>
<br/>        今早，她好心给傅夜沉递了大号创可贴，但是傅夜沉没用，他一脸毫不在意地带着那两排齿印，在各种异样的目光下，开完了各种会议，淡定地会见了各大商务大佬。
<br/>
<br/>        或许，他不觉得这有什么不妥。
<br/>
<br/>        但作为旁观者，落以柔却觉得很伤风雅，很影响集团公司的形象。
<br/>
<br/>        童筱颖听完落以柔这话后，被嘴里的彩虹螺旋酥给噎得够呛，一口气终于喘过来后，她难受地捂着嘴咳了起来。
<br/>
<br/>        好在，这桌只有她和她两个人，不然若是被旁人听到了，岂不是会炸开锅。
<br/>
<br/>        话说回来，她，落、以、柔，是不是住海边的？
<br/>
<br/>        这种私事，她也管？
<br/>
<br/>        虽然，她这话听起来似乎真的是为了集团公司着想，但是，总觉得很别扭！
<br/>
<br/>        童筱颖端起奶茶，大口大口地喝了几口，给自己顺了顺气后，放下杯子，毫不客气地回怼落以柔：“这话，你应该去跟你们的傅大总裁说！”
<br/>
<br/>        “他是我的上司，我不可能去跟他说这种话，所以只能从你这下手了。毕竟，我是为了公司好。童小姐应该不想做那个‘祸国殃民’的‘苏妲己’吧？”落以柔振振有词起来。
<br/>
<br/>        童筱颖淡定地笑了笑：“你这个比喻用得真是太好了！说得你现在每天的工作跟‘助纣为虐’似的。”
<br/>
<br/>        “看来，在童小姐心里，傅总是商纣王。”落以柔脸色极差。
<br/>
<br/>        童筱颖不以为然地挑了挑柳眉：“这话是你说的，我可什么也没说。”
<br/>
<br/>        “我真发现，你是不要脸到了极点。傅总都已经跟你离婚了，还对你下了禁令，你居然变着法子进我们傅氏集团来上班。而且还不要脸地去主动爬傅总的床，我真的是‘墙都不扶就服你’，童筱颖，你算是我见过的所有女人中，最不要脸的那一个了。”落以柔嗤之以鼻，一口气说完，连咖啡都未喝完，就起身甩手离去，丝毫不给童筱颖反驳的余地。
<br/>
<br/>        而童筱颖则被落以柔说得一脸懵。
<br/>
<br/>        落以柔这个女人，也太不知好歹了吧！
<br/>
<br/>        怎么说，她能继续留在傅氏集团总部工作，也是她童筱颖替她说的情吧！
<br/>
<br/>        这女人，不懂知恩图报也就算了，还喜欢自以为是地在她面前嚣张！！
<br/>
<br/>        算了，不跟她计较了。
<br/>
<br/>        童筱颖觉得自己的心很大，不跟落以柔这种小女子一般计较。
<br/>
<br/>        结果，一口气，她连吃了三个彩虹螺旋酥，又去点了杯鲜榨橙汁打包带走。
<br/>
<br/>        好在，她气来的快，去得也快。
<br/>
<br/>        回到办公室里后，童筱颖很快地进入了工作状态。
<br/>
<br/>        虽然，落以柔的话很难听，但是本意还是好的。
<br/>
<br/>        今早她咬大哥那事，真不能怪她。
<br/>
<br/>        她本来睡得好好的，而且还没睡醒，大哥突然入侵她，她出于自卫，才……
<br/>
<br/>        呃——
<br/>
<br/>        好吧！
<br/>
<br/>        她确实也有不对的地方。
<br/>
<br/>        工作告一段落后，她离开了办公室，去了医务室买了个大号的创可贴，找了个工作上的理由，才让前台小姐给她刷了卡，坐电梯去了顶楼的总裁办。
<br/>
<br/>        童筱颖叩响总裁办的大门时，傅夜沉正在和郑忠威谈工作上的事情，而落以柔在一旁做笔录。
<br/>
<br/>        知道自己来得不是时候，童筱颖主动将自己买的创可贴放到了傅夜沉的办公桌上，打算转身离去的时候，却被傅夜沉给叫住：“筱颖，等等。”
<br/>
<br/>        “今天就到这里。”傅夜沉随后对郑忠威说道。
<br/>
<br/>        郑忠威点了点头，起身离开。
<br/>
<br/>        落以柔也收拾了一下笔记本和平板电脑，跟随郑忠威而去。
<br/>
<br/>        离开前，落以柔下意识地回了下头，余光里，他们前脚刚走，傅夜沉后脚就将那个童筱颖给搂入了怀中，这让她看得心里很不是滋味。
<br/>
<br/>        他俩不是已经离婚了吗？
<br/>
<br/>        现在这是又复合呢？
<br/>
<br/>        她心里怎么就这么酸？
