        童筱颖听着季雅珊这话，忍不住笑了起来：“我离婚，怎么会是你害的。”
<br/>
<br/>        “你忘了吗？曾经我就对你说过，我是颗婚姻灾星。谁跟我亲近，谁的婚姻就会不幸。你老公，噢，不，是你的前夫，是因为他出轨了，你才会跟他离婚的吧！”季雅珊垂下眼帘自责道。
<br/>
<br/>        童筱颖一时间不知道说什么是好，傅昼景确实是出轨了，但是应该跟她季雅珊是不是灾星没关系吧！
<br/>
<br/>        可是，季雅珊似乎很把这种事情当真的一样去看。
<br/>
<br/>        童筱颖犹豫了一下，干笑着指着自己的耳朵，说道：“我前夫没有出轨啦！而他家暴！你看，我的听力就是被我前夫给一巴掌打失聪的。”
<br/>
<br/>        “太可恶了！居然打女人！还家暴！”季雅珊一听是这么回事，立即义愤填膺起来。
<br/>
<br/>        童筱颖会心一笑：“所以我离婚了，无依无靠。”
<br/>
<br/>        现在，她要做的，就是安安静静地待在这里，等着大哥派人来寻她。
<br/>
<br/>        只是，她这样算不算是把十爷给出卖呢？
<br/>
<br/>        “筱颖，你不该来这里的。”季雅珊惆怅道。
<br/>
<br/>        童筱颖不以为然地回答：“为什么不能来？”
<br/>
<br/>        “这座古堡，有着不可告人的秘密。进来的人，除了‘爷’，要想出去，除非死。你上次能出去，全靠十爷护着，但这次，十爷不可能再护你了。”季雅珊一脸忧郁地凝视着童筱颖的眼睛。
<br/>
<br/>        童筱颖这才意识到事情的严重性。
<br/>
<br/>        她有那么一会儿心不在焉，随即又回过神来，微笑道：“你放心，我们一定都能一起出去。”
<br/>
<br/>        “我们不说这些了，既然进来了，那就跟我一起干活吧！”季雅珊随之将床头柜上的一叠衣服拿了过来，递给了童筱颖。
<br/>
<br/>        童筱颖接过衣服看了看，发现是黑白灰的女仆裙装。
<br/>
<br/>        “我也要穿这个？”
<br/>
<br/>        “嗯，以后，你就跟着我一起，负责十爷和八爷的生活起居好了，噢，对了，还有四爷。”季雅珊微笑道。
<br/>
<br/>        童筱颖身心一怔：“四爷？”
<br/>
<br/>        大哥也在？！
<br/>
<br/>        这事她完全没料到。
<br/>
<br/>        季雅珊解释道：“他是最近才从国外完成了任务回来的，听说是个冷酷无情的杀手。”
<br/>
<br/>        “夜惑里的十位爷，是不是彼此都不知道彼此长什么样子？”童筱颖纳闷地问道。
<br/>
<br/>        因为，她发现，不管是八爷还是十爷，他们都是戴着假面眼镜出面的。
<br/>
<br/>        季雅珊回答道：“八爷和十爷彼此是知道的，因为，十爷是八爷带入集团的，所以，十爷和八爷比较亲近，但是他们确实不知道其他八位爷的真实面貌。我听说，夜惑里的十位爷，排名不按年龄而是按入集团的先后时间排的，而且全都听命于一个从未露过面的‘冒先生’。一爷和二爷掌管集团的财政，三爷、四爷、五爷、六爷是杀手，七爷和九爷是外交官，而八爷和十爷是高管。”
<br/>
<br/>        “高管？！我没听懂。”童筱颖皱了下眉头。
<br/>
<br/>        季雅珊打趣道：“就是管理集团内务的。”
<br/>
<br/>        “你这是看在我成为了这里的一员的份上，才肯跟我说这些的吗？”童筱颖试探性地问道。
<br/>
<br/>        季雅珊微笑着点了点头，接着说道：“当然。”
<br/>
<br/>        “那四爷为什么会住到这里来？”童筱颖有些纳闷。
<br/>
<br/>        季雅珊接着说道：“听说他杀了一个大人物，被全球通缉了，来这里是为了躲避这段时间的风波。古堡守卫森严，而且上空有屏蔽卫星。任何高科的东西，都扫描不出这块地方来。”
<br/>
<br/>        所以，这就是大哥他们一直找不到这里的原因吗？那她肩头里的那个纳米定位器岂不是没点作用？
<br/>
<br/>        “噢！”童筱颖若有所思地应了声，“看样子，我是真的一辈子都没法出去了。”
<br/>
<br/>        “我觉得待在这里挺好的，不愁吃不愁穿，而且还能照顾阿十的生活起居。”季雅珊甜甜地笑了笑。
<br/>
<br/>        童筱颖看着季雅珊，心里莫名其妙地替她感到忧伤。
<br/>
<br/>        她为了爱情，连自由都可以不要。
<br/>
<br/>        “好了，不说这么多了。你赶紧起来，洗漱完后，换了衣服，跟我一起去把早餐吃了。吃过早餐后，你得负责四爷的生活起居。”季雅珊微笑道。
<br/>
<br/>        童筱颖不解地问：“为什么是我负责？”
<br/>
<br/>        “八爷安排的啊！昨晚，十爷带你回来时，八爷和四爷都在，但是八爷不同意收留你，我说我不可能同时伺候他们三位爷的生活起居，于是四爷开口说要你来伺候他，八爷这才同意你留下来。”季雅珊回答道。
<br/>
<br/>        “八爷和四爷关系很好？”童筱颖诧异。
