　　李然站起身，把手指伸到罗昊面前，说道：“在绊线上涂上了口红，防止绊线在空气中形成反光，有长进啊！”
<br/>　　“这些都是我们玩剩下的！”
<br/>　　罗昊脸上同样挂着不屑，但眼神却变得有些凝重起来，对身边三个人吩咐道：“对方在林子里设置了诡雷，所有人注意自己身周，虽然说他们现在玩的都是我们已经玩剩下的，但是别大意，不要阴沟里翻了船。”
<br/>　　“明白！”
<br/>　　李然、李毅峰、慕容飞三个人沉声回答。
<br/>　　李然蹲下身，用斜口钳剪断了被用作绊线的钢丝，拔出军刀挖出了埋在树根下的白磷弹，对罗昊说道：“西伦科特那混蛋下手还真是够狠，居然用白磷弹，看样子还要试图闯入山顶别墅的人烧得连渣都不剩啊。”
<br/>　　“留着吧，回头留给他们自己用。”
<br/>　　罗昊扫了眼李然拿在手里的白磷弹，压低声音说道：“保持队形，继续前进。狐狸和无常，注意周围可能埋伏的狙击手！”
<br/>　　“明白！”
<br/>　　山林中，四个人继续向山顶别墅推进，在短短前进了一百米的距离，罗昊和李然就先后发现了十二颗诡雷，其中有美军为了在越南战场上最大程度杀伤步兵而研发的“阔刀”地雷，有罗马尼亚生产的苏式POMZ-2式破破片型地雷，还有反步兵跳雷，以及如先前李然所发现的用白磷弹所制作的诡雷。
<br/>　　“种类还挺齐全，简直可以开万国地雷展了。”罗昊把一颗刚才泥土中挖出来的白磷弹塞进自己战术背心的小袋中，将龙牙军刀塞回刀鞘，站起身说道：“继续前进！”
<br/>　　约莫一刻钟后，四个人走出了那片遍布杀机，稍有不慎，很有可能就会被烧得连渣都不剩的山林。罗昊回头望了眼身后的山林，才发觉自己的后背已经满是汗水。
<br/>　　尽管在看到刚才那些布设在山林中的诡雷，罗昊对这些诡雷布设的手法不屑，但是心中却不敢掉以轻心，在丛林中行动，伪装精巧的诡雷无疑是步兵最大的威胁，即便是训练有素的特种兵都有可能会中招，尤其是在晚上，夜色给原本就经过了精心伪装的诡雷增加了一层天然的伪装。
<br/>　　山顶别墅内灯火通明，罗昊、李然、李毅峰、慕容飞四个人不得不关掉多功能战术眼镜的夜视功能，才能看清楚眼前这豪华的别墅。
<br/>　　四个人移动到距离别墅大门差不多两百米左右，有着一块比人还要高的大石头背后，继续对山顶别墅的方向进行观察。
<br/>　　慕容飞眼睛盯着M200狙击步枪上的狙击镜，砸吧着嘴，说道：“这是要卖多少cocaine才能建造出这么豪华的别墅。”
<br/>　　“你应该说，是以多少家庭被cocaine摧毁为代价，才有了这样豪华的别墅。”
<br/>　　李毅峰手中端着M2010狙击步枪，对慕容飞的话进行纠正。慕容飞闻言，只是微微一笑，没有再多说什么，继续对山顶别墅四周的守卫进行观察和记忆。
<br/>　　别墅一共有两栋，一左一右，两栋别墅外是带有高压电网的围墙，正对着龙隐部队四个人的大门口听着四辆汽车，全都是跟机动巡逻队所使用的那种经过了改装，加装了M134转管机枪的大型SUV，四辆SUV周围总共有五名影子部队士兵在警戒。
<br/>　　围墙内，两栋别墅之间是游泳池，别墅前方是花园，院子空地上停放着一辆银白色的跑车，两辆黑色的路虎。不计算别墅内的守卫力量，只是在院子中，罗昊就看到了差不多二十个影子部队士兵在走动巡逻警戒。
<br/>　　这时，泥石山路上汽车在爬坡时所发出的低沉的引擎声，罗昊立刻缩回身体，并且招呼着身边其他三个人隐蔽。
<br/>　　那辆在罗昊、李然、李毅峰、慕容飞四个人刚索降到泥石山路上，从山顶别墅方向开下来的改装SUV又回来了。
<br/>　　改装SUV停在了别墅大门口，然后SUV中的影子部队士兵走了下来，跟门口五个警戒的影子部队士兵完成交接后，原本站在别墅大门口的五个影子部队士兵立刻爬上旁边一辆改装SUV沿着泥石山路往山脚下行驶而去。
<br/>　　李然抬手看了眼手表上的时间，说道：“机动巡逻队来回一次需要至少需要一个小时，这中间的时间足够我们完成对西伦科特的斩首。现在我们需要考虑的是，应该怎么进入那里。”
<br/>　　“要想在不被发现的情况下，潜入进去很难。”慕容飞摇了摇头，说道：“所有能接近别墅的地方都被带有高压电网的围墙给围住了，而别墅背后就是悬崖，不管是爬悬崖还是翻围墙进入别墅都是不可能的，要想进入别墅，只有通过那扇大门。”
<br/>　　罗昊打开了多功能战术眼镜的热成像功能，那辆刚停下的改装SUV引擎盖位置因为发动机还没冷却的关系，呈现出橘红色，通过多功能战术眼镜的热成像功能，除了在别墅院子里能看到的那些影子部队士兵外，在两栋别墅中也同样是能够看到发热体，显然在两栋别墅中也是有着影子部队的存在，唯一不能确定的就是在两栋别墅中到底有多少影子部队队员。
<br/>　　缩回到大石背后，罗昊关闭了多功能战术眼睛的热成像功能，说道：“正如刚才无常所说的，想要在不被发现的情况下潜入别墅，几乎是不可能的事情。既然不能偷偷摸摸，神不知鬼不觉的摸进去，那么我们就……”
<br/>　　“正大光明的杀进去！”李然冷声说道。
<br/>　　“看来我们是想到一起去了。”罗昊微笑道。
<br/>　　“除了这个办法，你还有别的办法能进入这个别墅吗？”李然对罗昊问道。
<br/>　　“并没有。”罗昊耸了耸肩回答道。
<br/>　　“狐狸、无常。”李然对李毅峰和慕容飞两个人，说道：“一会儿我们移动到距离大门一百米左右的位置，采用急速射解决掉门口那五个影子部队士兵，我要求他们在三秒钟内全都变成死尸。所以，我们必须在三秒钟内，干净利索的解决掉他们！”
