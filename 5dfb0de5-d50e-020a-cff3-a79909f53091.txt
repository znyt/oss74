　　“我想，你马上就笑不出来了！”
<br/>　　就在此时，一道霸气无双的声音从天庭之外响起，紧接着，厮杀声由远及近不断响起。不过片刻时间，便见十二道霸气侧露的身影踏入大殿，伴随着的，还有许多其他强者。
<br/>　　“巫族！”
<br/>　　帝俊恨声道，声音寒冷至极，宛若从九幽之下响起，传遍九霄。帝俊现在心情很不好，任谁在结婚当天，被别人杀上自己的家中，心情还会好？
<br/>　　“帝俊小二，你不用叫那么大的声音，我们能够听得见。听说今日是你大喜之日，怎么没有请我们兄弟？哈哈！你太不够意思了，这不，我们兄弟不请自来，都前来为你祝贺呢？哈哈！”
<br/>　　空间祖巫大笑道，笑声回荡在天庭各处，久久不散。
<br/>　　“巫族小二，欺人太甚！”
<br/>　　帝俊寒声道，阴沉着一张脸，仿佛能够滴出水来。
<br/>　　“帝江，你以为我妖族会怕你不成！妖族强者听着，今日屠光巫族，战！”东皇太一大声喊道。
<br/>　　“吼！”
<br/>　　太一话音刚落，十二祖巫便一声巨吼，十二股强大的气势，携带着无穷杀机轰然爆发，直冲云霄。
<br/>　　怒吼声惊天动地，战意冲天。
<br/>　　巫族就是一个十分喜好战斗的种族，欲与天公试比高，伐天自立。
<br/>　　“哈哈！”
<br/>　　帝江大笑，声音传遍九霄，怒声道：
<br/>　　“帝俊、太一小二，打就打，谁怕谁，难道我巫族还会怕你不成。你今天大喜，我们就多杀点妖族为你冲冲喜。哈哈！”
<br/>　　“帝江！多说无益，今日定要灭你巫族！杀！”帝俊不知从何处取出一柄金色长剑，携带着无尽杀意，杀向帝江。
<br/>　　“哈哈！谁怕谁？巫族儿郎们，给我杀！”
<br/>　　帝江大笑，接着挥动着手中拳头向帝俊杀去。自从修炼了《巫神决》后，他们十二祖巫实力大增，肉身强度堪比准圣后期，已经开始开辟法则空间，凝聚法则之海了，对自身血脉之中的法则，运用更加熟练了。
<br/>　　轰！
<br/>　　顿时，无数巫、妖两族的强者战斗在一起，厮杀声震耳欲聋，血雨纷飞。
<br/>　　帝俊大婚被邀请而来的贵宾们，见此情况，也快速向着洪荒飞去，他们可不想参与到两族之中的战斗中，对于他们而言，巫、妖两族都是大族，谁生谁死对自己等人都没有好处，最好是两败俱伤，同归于尽。
<br/>　　“来来来！三脚扁毛雀，上次没有打够，这次再来！”
<br/>　　火之祖巫祝融修炼火之法则，脾气最为火爆，此刻已经饥渴难耐了，对着太一，便挥动着手中的一柄火红色宽刀便向太一杀去。
<br/>　　“哼！祝融小二，给本皇去死！”
<br/>　　太一见此，脾气也上来了，抄起手中的混元剑，无尽杀气汇聚而来，他一步踏出，立刻就与祝融战在一起。
<br/>　　轰！
<br/>　　“噗！”
<br/>　　一声巨响，两人交手只在须臾之间，便见祝融狼狈倒退而回，口中一口鲜血喷出，同时，太一也极为狼狈倒退几步，身上血光乍现。
<br/>　　就目前情况来说，祝融落入下风。
<br/>　　“吼！再来！”
<br/>　　祝融不肯服输，瞪着太一，狂吼一声，挥起手中长刀，继续砍向太一。其祖巫见祝融有些不敌，就又有两个祖巫上前帮忙，与祝融三人一起战太一，一时间，两方势均力敌，争杀不断。
<br/>　　“鱼崽子！我们来战！”
<br/>　　水之祖巫共工看向妖师鲲鹏，顿时眼睛一亮，上一次大战，他还没有战胜鲲鹏，他绝对不会服输，此次一定要分个高下。
<br/>　　“战！”
<br/>　　鲲鹏也没有发话，一个字表明自己的态度，挥舞着大片水光，杀向共工。
<br/>　　“我来助你！”
<br/>　　风之祖巫天吴一声大吼，加入到战场之中，风与水的结合，一起杀向鲲鹏。
<br/>　　后土没有动手，她只是看向高空的女娲圣人。同时，女娲也紧紧盯着后土，也没有出手的意思。
<br/>　　伏羲弹起伏羲琴，一曲战歌浮现，震慑无数巫族灵魂。巫族本来就不修元神，灵魂抵抗力差，如今对上伏羲，正好被克制。不到一会儿，便有许多巫族被伏羲灭杀。其他祖巫见此，顿时大怒。
<br/>　　“伏羲，杀！”
<br/>　　烛九阴带动着时光轮，杀向伏羲，光轮划过的地区，时空扭曲，不到片刻时间，便死伤无数，全部化为飞灰。同时，雨之祖巫玄冥展现自己的领域，领域覆盖之中，大雨倾盆，雨落分生死。无尽悲意回荡在众人心头，他们好似放弃了抵抗，任由大雨将他们的生命夺走。
<br/>　　战！
<br/>　　妖族以前剩下的妖神与如今新晋的妖神，也与其他祖巫战在了一起，一时之间，大战不休，血流成河。巫族的大巫也与一些妖神缠上，大巫天生好战，皮糙肉厚，尤其像刑天、相柳等人，更是能够独当一面的高手。
<br/>　　其中，刑天手持一柄巨斧，上下翻动之间，无数妖族修士被砍杀，他几次冲到战场中，便是浑身血肉，让人看了几欲作呕。
<br/>　　妖族的准备不足，顿时落入了下风，但天庭毕竟是妖族的重地，随着招妖幡一阵摇动，越来越多的妖族修士自三十三重天或者洪荒各处飞来，加入到战斗之中。毕竟妖族人多势众，渐渐地，竟然抵挡住了巫族大军的功伐。
<br/>　　天庭大婚生变，众多洪荒大能纷纷离开天庭，或回到山门，或选择隐匿虚空观看，或者别有用心。
<br/>　　不得不说，这场战争来得既在众人的意料之中，却似乎又出乎众人意料。他们虽然知道巫、妖两族会再次开战，但没想到会来得如此突然，而且一经爆发，双方就是举全族之力，不死不休！
<br/>　　然而，在天庭外打得风生水起之时，却是没人发现，一道身穿黑袍的修士正大摇大摆地进入天庭后面，直奔妖族宝库而去。
<br/>　　他，正是万劫魔尊！
<br/>　　天庭的宝库，建立在三十三天之上，平常把手严密，无数天兵天将来回走动，绝不容许外人轻易靠近，但是，这一次却不一样。
<br/>　　此次巫族大举来袭，妖帝东皇大战巫族，妖族不得不倾巢而出，全力抵挡，这边造成了天庭后方出现短暂空虚，让万劫魔尊有了可乘之机。
<br/>　　不过片刻，他便到了三十三天，此处与天外混沌十分靠近，甚至，抬起头来，便能看见那无尽的混沌虚空。那极高之处，被称之为，天之涯！
