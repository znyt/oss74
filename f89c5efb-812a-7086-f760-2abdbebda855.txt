柳璎珞以柳招娣的身份进入世人的视线，她死后连名字都不被知晓，而她创下的一切都被柳招娣享有着。
<br/>
<br/>        而那些知道柳璎珞的人，很少数那几个人，比如陆茵茵，对柳璎珞却只有恨。
<br/>
<br/>        夏夕绾从心底很心疼柳璎珞这个女人，她本该有绝色风华，无比璀璨的一生，但是她却活成了别人的影子。
<br/>
<br/>        风一吹，什么都散了。
<br/>
<br/>        除了她的儿子陆寒霆，什么都没有剩下。
<br/>
<br/>        夏夕绾心里很疼，她现在更能理解为什么儿时的陆寒霆看着自己妈咪的牌位被烧了会疯狂成那样了。
<br/>
<br/>        夏夕绾再次看向柳招娣，柳招娣第一眼是让人惊艳的美，但是细看之下，她眉眼里深藏着功名与凌厉，让人觉得不太舒服。
<br/>
<br/>        陆寒霆说过，他妈咪柳璎珞跟柳招娣像是一胎双生，容貌九分相似，不知道柳璎珞那样淡然清冷而绝色的女子又是何等容貌，是不是不负帝都第一美人之名！
<br/>
<br/>        “绾绾，这位柳招娣不但是帝都第一美人，而且她出生的时候就有算命的道士说，柳家之女乃是凤后之命，命里贵不可言。”
<br/>
<br/>        “当时的柳家在帝都并不是什么豪门，更别说接触到什么帝都中心圈，但是柳招娣一出世，完美诠释了什么叫柳家有女初长成,养在深闺人未识，天生丽质难自弃,一朝选在君王侧，柳招娣被帝都陆家的嫡长子也就是陆氏财阀的继承人陆司爵看上了！”
<br/>
<br/>        提到陆司爵，叶翎神秘兮兮的眨了一下眼，“绾绾，你知道陆司爵是什么样的人吗？”
<br/>
<br/>        夏夕绾不知道，她只知道陆司爵是陆寒霆的亲生父亲。
<br/>
<br/>        关于这位父亲，陆寒霆从未提过，只字未提。
<br/>
<br/>        叶翎又将一本商业杂志丢了过来，“这可是当年的绝版杂志了，上面是陆司爵这个一代商界帝王的私人采访，现在你不会看到他了，因为他已经成为了各大商业周刊杂志都拍不到的神秘存在了，如果不是这一次合作，这本杂志我也是搞不到手的，绾绾，好好看看吧，让你看一下什么叫真正的极品。”
<br/>
<br/>        夏夕绾低头，看到了杂志封面上的那个男人，陆寒霆的亲生父亲陆司爵。
<br/>
<br/>        封面上的陆司爵坐在很高的商务写字楼上，他穿了一件白色衬衫，周身没有一丝繁华，但他举手投足都弥漫着皇室贵族的高冷倨贵，窗外炫目的阳光镀了过来，洒了他一肩的潋滟瑰丽。
<br/>
<br/>        夏夕绾看着陆司爵那张犹如鬼斧天工般的俊脸，她知道陆寒霆像谁了，陆寒霆真的很像自己的亲生父亲。
<br/>
<br/>        夏夕绾透过陆司爵好像看到了十年后，二十年后的陆寒霆。
<br/>
<br/>        “绾绾，这位陆司爵可是商界奇才，他十几岁进入了金融圈，创下了自己庞大的商界帝国，后来他继承陆氏，将两大帝国进行几次洗礼成为了现在的陆氏财阀，惊艳了一个商界时代，这样一个惊才潋滟又一身禁欲高不可攀的男人，还是被柳招娣给拿下了，都说柳招娣是凤后之命，这话一语成谶！”
<br/>
<br/>        夏夕绾知道，现在叶翎所说的柳招娣其实全部都是柳璎珞，这是柳璎珞的一生。
<br/>
<br/>        而柳招娣从来就是一个见光死的病秧子，她从来就没有出过门。
<br/>
<br/>        帝都第一美人是柳璎珞，Fly创始人是柳璎珞，凤后之命是柳璎珞，被陆司爵看上的，也是柳璎珞！
