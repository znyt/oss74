　　　　可还没等涂小安高兴，另外又出了一件事情。
<br/>　　一百多号中毒晕迷者，八八九九的都苏醒过来了，只有十几位依然不见转醒，呼吸甚至还变得越发的微弱。
<br/>　　陈默检查一翻后，脸色难看道：“不好，这十几个人没有解毒，而且已经有了生命的危险”
<br/>　　众人看去，这十几个人的溃烂伤口附近皮肤都紫黑了起来，而且这种紫黑还在不断的扩散，像是一点点的侵蚀人的生命力。
<br/>　　“陈医生，怎么会这样，都是喝了蛇王水，他们怎么不见效果”何震南疑惑的询问。
<br/>　　如果说苏醒的人是被一条罕见毒蛇咬了，那么这十几个人，就像是被一条更为厉害的毒蛇咬伤，毒素更为致命。
<br/>　　蛇博士陈默思忖一下，然后开口：“应该是这十几个人的毒中的更深，他们的伤口溃烂的也是最为的严重，已经开始病菌感染了，要马上隔离”
<br/>　　“蛇王水都救不了他们，事情麻烦了”
<br/>　　众人心中一凉，好消息伴随着坏消息同时接踵而至。
<br/>　　要是今天哪怕是死一个人，那么事件的性质就完全不一样了。
<br/>　　“那就将这十几个人送往医院在说吧，总不能让他们死在庙内”何震南无奈开口。
<br/>　　陈默看了他一眼：“恐怕要来不及了，他们中的毒太诡异，闻所未闻，送往医院，一时半刻恐怕也查不出来”
<br/>　　起码白镇的医院没有那么好的设施跟专家。
<br/>　　“那么厉害的毒，怎么可能是被蚊子叮咬的”
<br/>　　何震南响起涂小安之前说的话，觉得有点可笑。
<br/>　　其实绝大部分的人对于涂小安这套说法根本也不相信，蚊子叮咬人，能将人整成这种地步，恐怕人类早就灭绝了。
<br/>　　“使者大人，这下怎么办”李顺悄悄的对着涂小安开口。
<br/>　　涂小安也不想解释蚊子的问题，准备暗中召唤系统，看了看商城中有什么宝药能救这十几个人。
<br/>　　这时，陈默忽然对着何震南道：“当初佘山不是出了一条太玄蛇吗，最后太玄蛇落到谁的手中”
<br/>　　“陈医生，为什么突然问这个”
<br/>　　何震南说着像是一下子醍醐灌顶：“莫非太玄蛇能救这十几个人的命”
<br/>　　“太玄蛇为蛇中宝药，它的任何身体器官都是药用，它的蛇血能神奇的缓解人身体的衰老，更可以解百毒，要是有太玄蛇的血，或许还可以试一试”
<br/>　　事态紧急，在陈默现在看来，就算给送往医院，也无济于事了，这十几个人已经濒临死亡。
<br/>　　何震南摇了摇头：“太玄蛇虽然是落到我们白镇人民的手中，但是那条蛇的蛇血早就被人觅走了”
<br/>　　美琳将太玄蛇的器官分开买卖，这件事情何震南是知道的。
<br/>　　况且这事情过了很久，蛇血怎么可能留的住。
<br/>　　太玄蛇的蛇血可以救他们？
<br/>　　涂小安闻言，意识立刻退出了系统商城，要是如此，他到还有一个办法。
<br/>　　一条红白相间迷你精致的小蛇出现在涂小安的手中，他连忙对着李顺说：“将那个装着蛇王水的桶拿来”
<br/>　　李顺立刻照办，将水桶提到了涂小安的面前。
<br/>　　此时，众人皆看向了涂小安手中的这条小蛇，大惊失色。
<br/>　　不管是陈默还是何震南呼吸都凝重起来，一双眼睛烙印在这条小蛇的身上，他们一眼就看出这条蛇的不同凡响。
<br/>　　“好蛇，好蛇”
<br/>　　何震南立刻赞叹起来。
<br/>　　陈默更是诧异不已，深深的看着涂小安，因为他已经看出来了，这条小蛇看着小，但是蛇类气息很浑厚，隐隐间，周围空气都因为这条小蛇的出现，有波动涟漪。
<br/>　　这是一条觉醒蛇啊。
<br/>　　但陈默并没有将这句话说出来，只是对涂小安的身份更为的忌惮。
<br/>　　涂小安也不跟众人解释这是条什么蛇，一把匕首握在手中，语气温柔的看着变异太阴蛇，道：“宝贝，借你点蛇血用用”
<br/>　　如果太玄蛇的蛇血能救命，那么这条由太玄蛇的进化变异蛇种，就更不在话下了。
<br/>　　要不是陈默提醒，涂小安还真将这条太阴蛇给忘记了。
<br/>　　这条蛇的神奇之处，就连一个刚死之人都能救活。
<br/>　　区区不过二十公分的小蛇，红白条纹闪烁光泽，匕首顺着它的蛇鳞片微微一划，第一下居然没有直接破开它的蛇鳞片。
<br/>　　涂小安手指不轻不重的拍了太阴蛇的小脑袋一下，笑道：“乖，就借一点血，到时候补偿你”
<br/>　　这条太阴蛇可是觉醒类蛇种，非常聪明，平白无故的要流血，自然不太愿意。
<br/>　　它要不愿意，普通的匕首根本破不了它的蛇鳞片。
<br/>　　不过主人都开口了，太阴蛇嘤嘤的叫唤了几声，只能认命，只是人性化的显得可怜巴巴。
<br/>　　匕首再次从它晶莹剔透的蛇皮划过，这次蛇皮出现了一条细细不可见的伤口，一抹异常殷虹的血液慢慢溢了出来。
<br/>　　滴答...滴答...滴答...
<br/>　　涂小安最后足足从太阴蛇的体内挤出了十滴血液，滴到了装着蛇王水的水桶中。
<br/>　　顿时间，清澈的蛇王水瞬间被染了红，透着一股宝光流转。
<br/>　　别看才十滴蛇血，但已经可以救死扶伤了。
<br/>　　但凡是觉醒的蛇种，它的蛇血就非常的不同寻常，更况且这条是由太玄蛇进化变异而成的太阴蛇。
<br/>　　“李顺，别愣着，现在将桶里面的蛇王水给那十几位晕迷者灌下去，看看能不能解毒”
<br/>　　事态危急，涂小安也管不了别人异样的眼神，拿出这条太阴蛇救命，总比去系统购买药剂好解释点。
<br/>　　混合了觉醒太阴蛇蛇血的蛇王水再次升级，无论是效果还是价值都不可估量，李顺将一碗映着殷红的水灌入一位深度晕迷者的口中。
<br/>　　接下来，所有人撑着紧张的呼吸紧紧的盯着看，几乎是喝下去没几个呼吸，大家就看到那位中毒晕迷者伤口上的紫黑皮肤已肉眼可见的速度在消退，效果显而易见。
<br/>　　很快，晕迷者苍白的脸色开始出现红晕。
<br/>　　“效果那么好？”
<br/>　　庙内的人都惊呆了，不过只是加了蛇血进去，效果既然变的化腐朽为神奇。
<br/>　　“咳咳...”
<br/>　　晕迷者慢慢的咳嗽起来，他醒了。
<br/>　　“快，让剩下的晕迷者喝这个水”
<br/>　　陈默大喜过望，然后立刻看向那位散发蛇类气息的年轻人，这一看，顿时一惊。
<br/>　　人没了。
<br/>　　此时庙内哪里还有那位清秀男子的身影。
<br/>　　“何兄，那年轻人到底是谁，你认识吗”
<br/>　　陈默立刻的询问何震南。
<br/>　　何震南也是非常的震惊，被他吃喝的年轻人拿出一条蛇，却立刻将陷入危机的局面给挽回了，他功德无量呀，一下子救了十几条人命。
<br/>　　“不认识，也不知道是不是我们白镇的人”
<br/>　　紧接着，陈默跟何震南的目光都投放到了李顺身上，因为这位年轻人是李顺带进来的。
<br/>　　李顺浑身一颤，连忙摇了摇头，眼神不自觉的看向蛇王雕像。
