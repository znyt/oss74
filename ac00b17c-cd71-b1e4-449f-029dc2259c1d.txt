　　毒袍，已然是一具干尸。
<br/>　　生机散尽下，他已就此身死。
<br/>　　萧逸松了口气，但下一秒，眼眸一凛。
<br/>　　毒袍是死了，可毒袍留下来的必杀手段，萧逸却未必能挡下。
<br/>　　六龙六煞，分别是雷煞、木煞、风煞、血煞、毒煞、邪煞。
<br/>　　六煞齐出，狂猛噬来，怕是传奇之境也难以在其中活命。
<br/>　　“易霄分殿主。”远处下方，东方铁云看着六龙六煞从六方噬向萧逸的景象，大惊失色。
<br/>　　“该死，这家伙竟捅出这么大的篓子。”龙啸暴喝一声，但语气惊讶中又带着愤怒。
<br/>　　“木煞，给我回来。”
<br/>　　龙啸双手齐出。
<br/>　　“你要死便自己死，可别连累我六龙地域。”
<br/>　　“一旦六龙狂舞，整个六龙地域都会被夷为平地。”
<br/>　　毒袍不仅要杀萧逸，还要整个六龙地域给他陪葬。
<br/>　　单凭邪道封天阵的自爆，这显然是不够的。
<br/>　　可邪道封天阵上的力量，悉数爆发于六龙六煞的控制中，那便足够了。
<br/>　　毒袍死前的一瞬，夺了六龙六煞的控制权。
<br/>　　死前的一瞬，唯一使出的手段，便是让六龙六煞彻底暴走。
<br/>　　六龙六煞，才是真正能轻松击杀易霄，且将整个六龙地域毁去的东西。
<br/>　　“该死。”龙啸陡然气急败坏。
<br/>　　他全力之下，竟是未能夺回任何一条六煞的控制权。
<br/>　　高空中，远处，六龙六煞，已彻底袭向萧逸。
<br/>　　萧逸咬了咬牙，以六龙六煞的速度，躲是躲不了的；而且以它们的庞大身躯，从六方而来，早已封锁了所有范围。
<br/>　　这些龙形煞气，事实上并无灵智，只是一团庞大的力量。
<br/>　　可毒袍死前留下的手段，却足以让它们暴走。
<br/>　　“来便来吧。”萧逸双眼一眯。
<br/>　　兽手兽脚，早已覆于双手单脚之上。
<br/>　　五色圆环，顷刻凝聚。
<br/>　　再度借用龙炎，凝聚七色圆环，他可不敢。
<br/>　　那种反噬，当真可怕。
<br/>　　冰鸾剑，他也未用。
<br/>　　当然，这二者若有必要，他还是会用。
<br/>　　只是，即便用了，能否挡下六煞齐噬，亦是未知之数。
<br/>　　他能做的，仅仅是拼命一搏。
<br/>　　如果他真挡不住六煞，那便只能使用冰鸾剑和驾驭龙炎。
<br/>　　如果这二者也挡不下六煞齐至，那他萧逸也没办法。
<br/>　　轰…
<br/>　　六煞齐至，在冲撞到他身躯的一瞬间，发出一声惊天轰鸣。
<br/>　　六条巨龙，穿透了他的身躯，随后往六方直噬而去。
<br/>　　这就是毒袍的打算，六煞从六方而至，先是吞噬穿透了萧逸，随后六煞袭向六方。
<br/>　　六龙所过之处，必定无物可挡，一切被冲撞殆尽。
<br/>　　待得六龙各自朝六方冲撞至地域尽头，那么六龙地域，也被毁去八、九成了。
<br/>　　而高空中的萧逸，则是一口腥血喷出。
<br/>　　他没有死，他也没有使用龙炎和冰鸾剑，没有半分反噬。
<br/>　　因为在六重轰在他身上的一瞬间，他便知道，他自己死不了。
<br/>　　同一时间，原本冲撞而出的六龙，忽然环绕而归，再次‘噬’向萧逸。
<br/>　　轰…又是一声爆响。
<br/>　　这一次的萧逸，彻底处于六龙吞噬之下。
<br/>　　另一边，龙啸并未放弃夺回六煞的控制权。
<br/>　　吼…一声怒吼，巨大的木龙煞气，忽然在龙啸的控制下，疾速而回。
<br/>　　“太好了。”龙啸脸色大喜。
<br/>　　不过，远方高空中，另外五煞，仍旧吞噬着萧逸。
<br/>　　一时间，雷电、血气、暴风、剧毒、阴邪，五种不一的气息，五种各异的大作光芒，占据了整个高空。
<br/>　　“龙域主，怎么回事？”东方铁云急急问道。
<br/>　　“六煞的掌控权，不是已经被你夺回了吗？快收走六煞，救下易霄分殿主。”
<br/>　　龙啸摇了摇头，脸色又陡然难看，“不对，邪道封天阵的压制仍旧存在。”
<br/>　　“毒袍那个混蛋死前将大阵之力一瞬间爆发，故六煞的压制极强。”
<br/>　　“我现今能夺回其中一煞的掌控，已是万幸。”
<br/>　　“若要完全夺回六煞，只有等邪道封天阵的大阵之力彻底消散。”
<br/>　　“可是…”东方铁云脸色剧变，“若到那时，易霄分殿主，怕已在五龙煞气之下…”
<br/>　　东方铁云的话，没有说完，但意思却很清楚。
<br/>　　六龙六煞，哪一条不是蕴含滔天之力。
<br/>　　即便被龙啸夺回了一条，可剩下的5条，仍旧足以将易霄轰成齑粉。
<br/>　　“混账，本域主现在可管不了他。”龙啸咬了咬牙。
<br/>　　“高空五煞齐轰，一旦彻底爆发，其力量仍旧足以将我六龙地域毁去7成以上。”
<br/>　　“给我封。”龙啸暴喝一声。
<br/>　　木煞的控制权，他已然夺回。
<br/>　　而他最先想夺回的，也是这一条木煞。
<br/>　　因为他本就修雷之一道与木之一道，雷煞和木煞，也是他最为擅长控制的两条煞气。
<br/>　　相比较雷煞，木煞的防御力更强。
<br/>　　在他身后，可是还有一众龙家武者，那可是他龙家的基石。
<br/>　　若他身后的一众龙家身死，那他龙家可算是完了。
<br/>　　嘭…嘭…嘭…嘭…
<br/>　　无数巨木，拔地而起。
<br/>　　不过片刻，一个巨木世界，再次凝聚。
<br/>　　无数巨木，蜿蜒缠绕，瞬间封锁了高空数万里范围。
<br/>　　“呼。”这时，龙啸才松了口气。
<br/>　　“高空已封锁，即便五煞之力真的爆发，我也能抵挡一二，将后果控制在可承受的范围内。”
<br/>　　龙啸的话语间，底气十足。
<br/>　　东方铁云脸色难看地看着龙啸，“你现今封锁了整个高空，即便易霄分殿主真的勉强挡下一次五煞冲撞，也没机会逃出来。”
<br/>　　龙啸看向东方铁云，“那你的意思是，让本域主放开封锁，眼睁睁看着五龙煞气爆发，波及整个六龙地域？”
<br/>　　“眼睁睁看着六龙地域内生灵涂炭？”
<br/>　　东方铁云冷声道，“本殿主没记错的话，这招巨木封印，是你龙家最强的防御手段。”
<br/>　　“在木煞巨龙的加持下，巨木封印的防御，几无物可破。”
<br/>　　“之前，即便毒袍手中太荒之炎真的降临，龙域主你脚踏木煞，也应有把握拦下，所以我才会叫易霄分殿主速速逃离。”
<br/>　　“可你却让易霄分殿主去拦毒袍。”
<br/>　　“你早便能放出巨木封印，且你早些放出，便不会有此时之事。”
<br/>　　龙啸眼眸冰冷，“如果易霄他能阻拦毒袍，那便不必本域主浪费元力了。”
<br/>　　“施展一次巨木封印，你当本域主容易？本域主将一身实力留于对付毒袍，不是更好？”
<br/>　　“至于本域主何时出手，那是我的事。”
