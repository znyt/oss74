　　第492章
<br/>　　荆州，群山之中有座小山，山中有座山谷，谷中有一片湖泊，临湖搭了一座草庵。
<br/>　　山太小了，也无什么名字，湖泊也无名，这草庵却听月这个名字。
<br/>　　听月庵中住着一白衣比丘尼，无人知道她的来历，在山谷中种几亩闲田，春种秋收，或采山果裹腹，如此倒也可度日。
<br/>　　今夜月明星稀，天上的月落在湖水之中，白衣尼在湖水中洗手，将一颗血淋淋的人头埋入湖边一个挖好的坑中。
<br/>　　“师父。”有一麻布长衫的女子立在湖边。
<br/>　　“隐娘，这是第几个了？”
<br/>　　“第十八个。”聂隐娘答道。
<br/>　　白衣尼点点头，一边填土一边道：“你十岁随我学剑，初时用二尺青锋，学成之日，剑尖磨平三指。后来杀得人越来越多，用得剑也越来越短，学到今日，飞花摘叶皆可杀人。”
<br/>　　“师父，徒儿所杀皆是不义之人。”
<br/>　　“对都是不义该杀之人。如今，你再替我杀一人，便可出师，以后的日子就是你自己的。”
<br/>　　“杀谁？”
<br/>　　白衣尼递给她一副画像，聂隐娘接过来看，上面绘着的人目露凶光，相貌凶恶丑陋，旁边写着三个墨字：程大雷。
<br/>　　“是他？”
<br/>　　“不错，就是他，此贼大闹长安，阻挠和亲，在凉州作威作福，在草原上杀人盈野，乃是天下第一恶人。”说着，老尼在坑上填上最后一捧土，道：“你若能杀了他，便能多救几个苍生百姓。他如今在凉州，身边有重兵保护，想杀他并不容易。”
<br/>　　聂隐娘收起画像，道：“徒儿必提他人头来见师父。”
<br/>　　“多杀几个恶人，也能让这乱世早些结束，好了，你去吧。”
<br/>　　聂隐娘在湖里捧了一把凉水，将脸上的风尘洗净，她冲着白衣泥拜了三拜，负剑向山谷外走去。
<br/>　　离开山谷，目光扫视群山，借着月光辨认凉州方向。月光下，显出她一双凛冽却又干净的眼睛。
<br/>　　……
<br/>　　蛤蟆城的生产运动进行得如火如荼，和珅的忙碌自然不必提，就连秦蛮和关鱼这样的大将，也要下田赶着马耕地。
<br/>　　但在这样忙碌的时候，程大雷却突然宣布他闭关了。对外宣城是闭关悟剑，把自己关在房中，大门不出二门不迈，任何人想见他，都不可带铁器。
<br/>　　这件事怎么看怎么都透着奇怪，不过程大雷做得奇怪的事也不差这一件，大家都已经见怪不怪，既然他想搞这么一处，大家也就随他去吧。
<br/>　　程大雷是自家事自己知道，这件事就算任何人都不理解，他也要这么做。毕竟是性命攸关的事，怎样谨慎也不算过分，程大雷可不敢拿自己的性命开玩笑。
<br/>　　时间一日一日过去，在程大雷『闭关』的这段日子山寨也发生了许多事。
<br/>　　赤练仙子李莫愁已投奔到山贼，刘悲亲自与她进行了面试。据说这赤练仙子出身在一小道观，后来被一公子哥拐跑了，事后才知道自己被忽悠了，原来这公子哥早有了老婆孩子。
<br/>　　李莫愁一怒之下，当着公子哥的面，把他老婆孩子都杀了。然后自己因为怕丢人，也没办法回道观，就在江湖上闯荡，见到薄情忘义之徒就杀，动不动就杀人全家，最后在江湖上闯下赫赫恶名。
<br/>　　不过么，既然这么大名气，程大雷为何以前从未听说过，想也知道这是系统编排的人物背景，程大雷也没有太追究。
<br/>　　程大雷并没有见李莫愁的面，虽然心中对这赤练仙子的相貌还是蛮好奇的，但为了自己的生命安全，还是小心为上。
<br/>　　李莫愁到来后，也没有需要她的工作，先给她套房子养着便是。
<br/>　　在这段时间内，李婉儿如愿以偿的代替奥黛琳娜成为万物教圣女，而万物教的信徒，比如吟游诗人凯恩等人，竟然没有一人反对，还说什么，一看李婉儿长得就像圣女。
<br/>　　奥黛琳娜顺利和刘悲成亲，那一日山寨敲锣打鼓分外热闹。
<br/>　　这两件事，程大雷一件也没有参加，他窝在自己的小屋子里，和苏樱、柳芷、小桃几个逗闷子，日子倒也过得悠哉悠哉。
<br/>　　不过，聂隐娘的事情不处理掉，程大雷一颗心总不能放下来，自己一直这样窝着，总也不是个办法。
<br/>　　也不知道聂隐娘来了凉州没有，说不得她现在就窝在蛤蟆城，正在寻找机会。
<br/>　　想到这里，程大雷一身冷汗，更加不敢出门了。
<br/>　　程大雷虽然大门不出二门不迈，但也并不是什么都不管。喜字队的情报每天都送到房内，在江湖传言的作用下，程大雷通过这些情报了解帝国如今的动向。
<br/>　　过去的这段时间，帝国也发生了很多大事，算起来，每一件都比蛤蟆城这里发生的事情重要。
<br/>　　从并州撤兵后，各路诸侯聚在长安城，经过一番扯皮、口水战之后，最后继承帝位的还是李乐天。
<br/>　　毕竟，他的表现是可圈可点的，军中有大批的支持者，李行哉几乎没可能与他竞争。
<br/>　　当然，李乐天想要顺利登上帝位也不太容易。他纵然有些威信，但靠这威信也压不住天下诸侯的贪念。
<br/>　　这就像分蛋糕，李乐天没有吃掉全部的实力，就需要每个人都分些。最后，他一连封了十八位诸侯王，但凡手中有些兵力，又在这次讨贼行动中露过脸的都成了一地之王。
<br/>　　不过，在这次围剿正义教行动中，出力最多，牺牲也最多的李星，却连个王爷的称号都没有混上。原因是，他看穿了帝国上层人物的龌龊，半途撤了。因此，他不仅没有被封王，还成了口诛笔伐的对象，一时名声臭了大街。
<br/>　　李星自然不干了，根本不认同长安的决定，他在扬州搞了个小朝廷，自己登基称帝，打出的口号是，李乐天那个皇位是假的，明帝生前是把皇位传给了自己。
<br/>　　如此，帝国就有了二帝十八王。李乐天下了圣旨，号召扬州附近的几位诸侯王讨伐李星。
<br/>　　至于李乐天，他自己是不会出兵的，现在傻子也知道是休养生息的时候，而且这一仗也并不好打。
<br/>(本章完)
