　　“砰砰砰……”
<br/>　　星辰世界中，盘古真身与星辰真身像两台巨大的生命收割机，每一次攻击都炸开一片血雾，成片的妖族、巫族、人族或者劫魔倒下。
<br/>　　“吼！”
<br/>　　盘古真身一声怒吼，扭过头来，狰狞地盯着前方的星辰真身，暴戾的双眼血光伸缩不定，浓郁的杀意几乎凝为实质。
<br/>　　“吼！”
<br/>　　巨大的手臂之上乌光闪烁，一道道金色的神纹爬满盘古真身的整条手臂，看起来就像一条条凶残的蜈蚣，狰狞到了极致。
<br/>　　“轰隆隆！”
<br/>　　粗大的手臂朝着星辰巨人狠狠的一拳轰出，数十万里空间轰然崩溃，像是受击的玻璃一般，寸寸粉碎，发出嘎嘎的响声，巨大的拳风卷起无尽灵气，形成一道千万丈的浩瀚长河，隆隆地朝着帝俊撞去。
<br/>　　“砰！”“砰！”“砰！”……
<br/>　　周天星辰大阵中的一尊尊星神手捏玄妙法诀，无数星光洋溢而出，形成一颗颗浩大的星辰，然后向着那浩瀚长河轰然炸开，荡起一波波剧烈的冲击波，炸散长河。
<br/>　　与此同时，帝俊的星辰真身瞬间打出数万拳，一条亿万里长的星光大道从虚空之中显现，这星光大道好似包括无数星辰的玄妙运行轨迹，一个接着一个的星辰空间幻生幻灭，演绎无穷星辰大道至理。
<br/>　　一种种星辰道法在其中显化，单单是看上一眼，便会使人沉浸其中，不可自拔。
<br/>　　“轰隆隆！”
<br/>　　星光大道轰破虚空，仿佛从远古横跨而来，一种无上伟力在其中衍生，携着无穷帝威瞬间朝着盘古真身杀来。
<br/>　　“吼！”
<br/>　　仿佛是自己的尊严受到挑战，盘古真身双腿猛地狠狠一踏。
<br/>　　“轰！”
<br/>　　整个洪荒世界都狠狠的震了一震，周围数十万里的大地直接破碎开来，空间变为粉碎，荡起一层层粉末，整片天空都仿佛变得模糊起来。
<br/>　　就在此时，无数黑色凶煞之气从洪荒各处升腾而起，化为一条条狰狞的黑龙，咆哮着向战场中飙射过来。
<br/>　　未久，盘古真身上空覆盖着一层数千万里的黑云，将整个战场笼罩在其中，日月无光，星辰暗淡，仿佛一片永夜。
<br/>　　“吼！”
<br/>　　盘古真身高高举起用法则煞气汇聚而成的巨大擎天盘古斧，一道道破灭一切的无形杀机瞬间席卷整片战场，一条条巨大的深渊以巨人为源头，向着远处蔓延而去。
<br/>　　“斩！”
<br/>　　盘古真身周身煞气涌动，将盘古斧狠狠地向前一斩，一道道凶煞之气凝聚成的斧刃割裂虚空，仿佛开天辟地一般，毫不留情地斩向星光大道。
<br/>　　“轰隆隆！”
<br/>　　斧刃与星光相撞，发出猛烈的爆炸声，整片空间一阵猛烈摇晃，巨大的余波瞬间绞杀数十亿三族修士，碰撞之处，一个巨大的黑洞缓缓出现，吞噬着周围的一切生机，一道道恐怖的空间风暴席卷而来。
<br/>　　“吼！”
<br/>　　盘古真身一步踏出，跨过无尽虚空，一斧接着一斧，狂暴的劈了过来。此刻的空间，就如同纸糊一般，被劈开一道道巨大的口子。
<br/>　　“杀！”
<br/>　　星辰真身也毫不畏惧，手持星辰之剑与屠巫剑向着盘古真身杀来过来，两者大战在一起。空间风暴不断，灭世般的威力，席卷天下。
<br/>　　大地沉浮，星空破碎，山河碎裂，江河汹涌，一副末世之景，扑面而来。
<br/>　　而与此同时，玄天也与东皇太一大战在一起，大战到了最关键的一步。
<br/>　　玄天的武祖法相顶天立地，全身散发着无量金光，大手向着虚空遮去，神光闪烁，凝聚出无比磅礴的伟力，瞬间，虚空动荡，化为一张遮天罗网，遮蔽虚空，冻结空间，无所不至，把周围一切有形无形的虚空尽数封锁，将东皇太一包围起来。
<br/>　　“天网恢恢，疏而不漏！罗天网！”
<br/>　　“东皇太一，一代妖皇，你也该上路了。”
<br/>　　玄天轻声开口，言语之中，尽是一片淡漠。话音刚落，下一瞬，无边神力迸射而出，伴随着他轻轻一掌退出，一股无可匹敌的力量怒涌而出，霎时间，乾坤震动，宛若十方瀚海在沸腾，磅礴的掌力逆冲九天，贯穿霄汉，直奔东皇太一重击而去，大有吞天噬地的威能。
<br/>　　东皇太一深吸一口气，吞吐天地风雷，周身无尽太阳真火缭绕，狂涌而出，先天神火，燎原而起，宛若滔天火海，覆盖苍穹，却又有着无边星辰天宇，自无边火海之中衍化生成，深邃，看不见底。
<br/>　　“浩荡天地，混元开天！”
<br/>　　面对眼前这人，东皇太一没有丝毫把握，心中并不敢有丝毫大意，口中一声大喝，法力汹涌，运转到极致，无尽威能，被他灌注到混元剑中，化为无边神威，翻天覆地，气冲九霄。
<br/>　　赫见一道冲天剑柱，以东皇太一为中心，无尽锋锐之气，向着八方开裂而去，剑气纵横，充斥着乾坤寰宇，空间破碎。
<br/>　　“斩！”
<br/>　　一声大喝，剑光冲天，天地为之震荡，随即四周一片天崩地裂之声爆发而出，天空之上的剑光一阵巨颤，浩荡掌力，卷起无边风云，御雷执电，铺天盖地一般席卷而来，几乎瞬间，便带着不可计量的伟力，与斩天剑光撞击在一起，发出惊天动地的震耳大响，让人听了忍不住为之一震气血翻滚，心中大感难受。
<br/>　　天上地下，一道道巨大裂缝正在不断蔓延崩溃，纵横交织，向四周扩散，无尽虚空，像是在不断崩溃。罗网所禁锢的虚空，早已破碎，无穷无尽的力量在爆发，如云雾一般，奔腾而起，密布周边空间，震碎天地四极。

<br/>　　东皇太一手持凶剑，踉跄倒退，口中鲜血喷出，胸前还有一道巨大的掌印，全身气息一阵萎靡。显然，他是受伤不轻。
<br/>　　“准圣后期，突破了？”
<br/>　　玄天一阵诧异开口，言语之间，带有几分惊叹，道：
<br/>　　“不愧是绝代妖皇，可惜，遇到了本祖，是你的不幸。今天就算你突破了，也逃脱不了灭亡的命运。”
<br/>　　“帝者，老子第一，老天第二。天上地下，唯我独尊！”
