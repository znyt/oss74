　　“掠夺行为全都是恶，我们从出生那刻起，就不断的进行掠夺。”店长直视着众人，声音平静：“所谓生命，就是不断犯下罪行的过程。”
<br/>　　第四梯队的搜查官们，警惕的看着店长，不知道这名SSS级的喰种，到底是什么意思。
<br/>　　“生命，就等同于恶，我有自觉，我就是恶。”店长用平静的声音，继续说着，他的身体，逐渐被一层如同水晶一般的红色甲质物所包裹。
<br/>　　这是店长的赫者形态，在这一刻，他再也不是那名慈祥的老人，而是令无数人闻风丧胆的，‘枭’。
<br/>　　“而你们也一样，来吧，来杀我吧。”面对着数倍与自己的敌人，枭没有丝毫惧意，仿佛，世界上的任何东西，都不可能威胁到他一般。
<br/>　　完全赫者化的枭，足有两三米高，整个身躯都如同一尊巨兽一般，随着他的动作，每一下都让地面微微颤抖。
<br/>　　面对着如此强大的对手，在场的搜查官们都是不由自主的后退，这股极为强大的压迫感，几乎让他们感到窒息。
<br/>　　而此时，反倒是那些相对实力较弱的特种军人们，站在了搜查官的前方，将手中的枪口，对准了前方的怪物。
<br/>　　“突击！突击！”一名挂着高级军衔的特种军官，举着手中的自动步枪，向店长的方向狠狠扫射着，但这些专门为了猎杀喰种而特制的子弹落在店长身上，却没起到一点效果。
<br/>　　甚至，连阻挡一下的作用都起不到。
<br/>　　慢慢的向前行走着，店长脚下的地面寸寸碎裂，他没有丝毫着急，就连敌人向他发动攻击，都没有让他感到愤怒。
<br/>　　他只是有些可惜，为了逃走的董香和雏实，他不得不杀人了。
<br/>　　想必，四方能够好好照顾好他们吧，他相信四方的能力。
<br/>　　似乎是忏悔，店长血红的瞳目轻轻闭了一下，随后密密麻麻的羽毛，从他背后的赫包中喷射了出来，将围在他身边的特种部队，全部射成了蜂窝。
<br/>　　只是普通人组成的军队，在他这样等阶的喰种眼中，还是不够看啊。
<br/>　　用背后尖刺状的赫子挑起一名特战队员，店长冷漠无情的注视着他，任其在他的赫子上拼命挣扎着。
<br/>　　这些烦人的蚊虫，总是在打扰着他的沉睡。
<br/>　　心中涌上一股久违的嗜血感，店长索性放纵，将特战队员整个人撕裂成了碎块。
<br/>　　“乒！”
<br/>　　一道耀眼的蓝光闪过，四周的建筑群中，突兀出现了几道身穿盔甲的身影，他们手中的武器在店长身上划过，溅起一片火花。
<br/>　　“唔，我记得你们，是上次差点被我杀掉的那些人吗？”店长注视着眼前出现的这些搜查官，赫眼中浮现一抹残忍。
<br/>　　CCG，特等行动小组，先前一直没有出现在战场上的身影，等待的就是这一刻。
<br/>　　全身被覆盖式的盔甲包裹着，众人站在各自的位置，将店长缓缓包围，他们所有人的眼神中，都充斥着无尽的仇恨。
<br/>　　这些人，每一个都是CCG的特等搜查官，对于枭，他们也都有各自的仇恨。
<br/>　　几年前他们败过一次，这次无论如何也不能再失败了，他们，承担不起失败的后果。
<br/>　　……
<br/>　　蹲在建筑群上方的楼顶上，刘哔目光有些兴奋，所有的计划，都在按照他所料想的方向发展，现在唯一需要等待的，就是CCG的特等班将枭杀死，他就可以出手争夺尸体了。
<br/>　　但让他有些担忧的是，这些特等搜查官，似乎并不是店长的对手，虽然两方的火力都相差不多，但是枭的再生能力实在是太过于强大了。
<br/>　　如果说特等队的火力值为五，那枭的再生值大概就为八，也就是说，他们的攻击，对店长几乎造不成什么伤害。
<br/>　　场中的战况陷入了僵局，从刘哔的位置看去，战场上的情况都一目了然。
<br/>　　黑狗那边，已经败了，说起来可笑，那个家伙，失败的原因居然是去救一名普通人类。
<br/>　　失去黑狗的黑色杜宾，逐渐被搜查官们消灭，而魔猿那边，也支撑不了多久了。
<br/>　　金木此时还在青铜树受刑，根本不会像原著中那样，突然出现将他们救走，古间圆儿，落败也是迟早的事情。
<br/>　　有点可惜的是，他根本没有机会将两名喰种的尸体收走，那可是两具SS级的喰种尸体，哪怕是CCG，也将这两具尸体看的无比重要。
<br/>　　这是十分具有研究价值的东西，他们身上的赫包，也能做成威力强劲的库克因。
<br/>　　为了提前引起CCG一方的警惕，刘哔也只能继续蛰伏着，等待店长这边的战斗分出胜负。
<br/>　　僵持之下，场中突然出现了一个纤细的身影，刘哔目光一凝，将视线放到了那道身影上。
<br/>　　他却是没有预料，铃屋什造居然也有资格参与到现在的战斗中来，要知道，什造现在的实力，比特等搜查官还相差了不知道多少。
<br/>　　而场中的局势，也随着什造的加入开始逐渐变化了起来，几乎所有的特等搜查官们，都在围绕着什造，以她为核心的进攻着。
<br/>　　而什造手中的库克因，也换成了一柄刘哔从来没有见过的长刀，而她的每一次进攻，都能在枭身上留下一道深深的伤口。
<br/>　　这个家伙，实力果然没有表面看上去那么简单，刘哔稍稍有些庆幸，自己先前没有贸然出手。
<br/>　　原著中有最强人类之称的什造，就算还没成长起来都如此恐怖，实在难以想象，等其完全成长起来，会发展到什么地步。
<br/>　　场中，充斥着剧烈的爆破声，夹杂着什造有些癫狂的笑声，店长在这场战斗中，开始逐渐露出了颓败的迹象。
<br/>　　苦战良久，天色都已经变的不再阴沉，店长终究支撑不住众多特等搜查官的进攻，被一刀劈碎了面具。
<br/>　　面具下，苍老的脸庞沾满了血迹，他的身上，已经布满了伤痕。
<br/>　　等候了一夜，刘哔也在此时，从建筑群上跳了下来，一根漆黑的龙头拐杖，自枭猩红的眼眶进入，捅穿了店长的头颅。
<br/>　　“真遗憾，我还想再请你喝一杯美味的咖啡呢，刘先生，”嗅到那股特有的气息，店长已经知道了来人是谁，声音平和的仿若不是一个将死之人。
<br/>　　目光微微闪动了一下，刘哔将体内所有的灵气送入店长头颅中，将其内部搅的稀烂。
<br/>　　“可惜，我真的很讨厌喝咖啡。”像是回答店长一般，刘哔轻喃了一声，将店长的尸体装入了空间戒指。
<br/>　　随后，没有理会在场的众人，刘哔挥动背后的黑色羽翼，消失在了茫茫夜色中。
