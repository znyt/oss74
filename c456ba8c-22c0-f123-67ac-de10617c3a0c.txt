　　那金字一共六十四个，在空中飞转，不断分散组合，有风、火、雷、冰，这样的字眼浮现出来，每一个字，代表的都是不同元素的攻击，裹挟着阵法的力量，对他进行一次又一次的冲击……
<br/>　　“你还需要多久。”道风在神念之中，艰难地对上古邪神说道。
<br/>　　“你走吧。”
<br/>　　道风微微一笑，面对阵法之力的绞杀，突然挣扎着站起来，双眼决然地望着清长风，眼中流露出决然之色。
<br/>　　嗯？
<br/>　　清长风心中疑惑，不知道他要干嘛。
<br/>　　道风将打神鞭和番天印扔到上古邪神脚下，盘膝坐下，一股诡异而恐怖的气息，从他身上散发出来，立刻敌住了阵法之力，互相消耗起来。
<br/>　　怎么会这样？
<br/>　　清长风大惊，一步退到石碑边上，一只手按在上面，用自己的修为，不断催发着石碑的灵力，加强着法阵的强度。
<br/>　　突然间，道风额头的皮肤裂开了，一道黑红色的血流淌出来，又迅速蒸发成了一股血雾，在身边挥发，加强着结界的防御。
<br/>　　皮肤不断裂开，更多的血流出来，挥发成血雾，不一会功夫，他浑身上下的皮肤都褪去了，浑身红彤彤的，看上去就像一个人被剥了皮，两个眼珠子死死地瞪着清长风，浑身上下不断地往外冒血，形成一圈圈的血雾，画面触目惊心。
<br/>　　清长风的确异常震惊，他虽然不知道这是什么邪术，但是万变不离其宗，任何生灵的肉身、血肉骨骼，还有自己的魂魄之中，都蕴含着强大的力量，这也是生灵一身修为的源头，平时修炼积攒的灵力，一大部分都被肉身骨血吸收，方才能够不断强大起来。
<br/>　　这就像是一个能量库，一旦爆发出来，能极大地提升灵力的续航能力，只是……这个能量库是一次性的，肉身，骨血，魂魄，对生灵来说，基本上都是不能再生的，就算再生，灵力也要重新一点点地积累，是一个极为漫长的过程。
<br/>　　一旦有生灵融化自己的血肉，那一定是到了绝境，要么是想靠着肉身融化产生的强大力量，来保全魂魄，要么就是为了跟对方同归于尽。
<br/>　　还有一种可能，就是牺牲自己，保护别人。
<br/>　　道风，真的已经被自己逼到绝境了？
<br/>　　就算真是绝境，他为什么不先尝试突围，难道他是想牺牲一部分血肉，来保全魂魄，寻找突围的机会？
<br/>　　一定是这样！
<br/>　　确定了这一点，清长风立刻打起精神，减弱了对法阵对道风的攻势，而提升了防御，一旦道风暴走，他便可汇集法阵的力量，先发制人，让他无处可逃。
<br/>　　“道风，你够狠，也很果断，只是……既然这样，你终究无路可逃！”
<br/>　　清长风面露胜利的微笑，骤然之间，他想到一个非常严重的问题：
<br/>　　道风的肉身，之前不是已经毁了吗？
<br/>　　纵然他已经斩却两尸，精于化形，能随便幻化出肉身，这肉身也能作法，能拿得起法器，但毕竟不是真的，这样的一具肉身，在燃烧血肉的时候，为何会有如此源源不断的真元溢出？
<br/>　　难不成……自己上当了，之前被自己斩杀的肉身不是真的身外化身，眼前这具才是？
<br/>　　清长风有点晕。
<br/>　　但是事已至此，也只能继续刚下去，站在道风的立场，清长风把局面分析了一下，实在想不到他除了等死还能做什么。
<br/>　　纵然他真有什么阴谋，也不可能有翻身的机会。
<br/>　　那些藏在远处山里观战的诸位大佬们，也都屏息凝神，静静地等待着结局，他们也不明白道风为什么要这么做，跟清长风一样，他们也认为道风没有一丝机会。
<br/>　　时间一点点过去，道风的血肉逐渐化尽，剩下一副骷髅，上面还挂着一些没化尽的血肉，看上去真的惨不忍睹。
<br/>　　他的骨头，也在缓缓融化着。
<br/>　　清长风的眼睛越发明亮起来，平素古井无波的内心，也忍不住开始有点激动。
<br/>　　等到他一身骨头化尽，那真的就全完了！
<br/>　　曾经的人间道神、这个三界闻名的杀神，法术界所谓数百年一遇的人才……就会死在自己手上。
<br/>　　在这发生之前，他断定道风一定会做点什么，不过，他做什么又有什么用呢？
<br/>　　清长风朝着对面的山峰挥了挥手，若干人影从山野间走出来，在山口方向站成一排，也都做好了进攻的准备。
<br/>　　这些都是他门下最得意的几个弟子，一共八个，其中一个还是他关门弟子，也是将来门派的继承人。
<br/>　　以他们的实力，自然不是道风的对手，不过他们站的位置，暗合八阵图的方位，将山口牢牢堵死，假如道风真的试图逃走，至少能挡他一下。
<br/>　　当然，这一手其实也没什么必要，无非是多加一道保险。
<br/>　　道风，这次你插翅难逃！
<br/>　　清长风双眼直直的盯着道风，等待他的骨头也完全化掉。
<br/>　　就在这时，一道人影从山谷外飞奔而来。
<br/>　　清长风第一反应是救兵，定睛一看，却是自己门下一个弟子，这弟子一路飞奔到山口处，倒头拜道：“师父，大事不妙！”
<br/>　　“说！”清长风心中一惊。
<br/>　　“山门，山门被人攻打了！就要攻到山顶了，师兄弟们拼死抵抗，但根本抵挡不住！”
<br/>　　什么！
<br/>　　清长风大惊，“什么人攻打山门。”
<br/>　　“道风，就是那个叫道风的！”
<br/>　　清长风怔了几秒钟，斥道：“胡言乱语，道风不是在这吗！”
<br/>　　那弟子这才看到被困在阵中的道风，一下子呆住了，表情相当有意思。“这……师父，闯山的真的是，就是他，道风！”
<br/>　　清长风错乱了，就在这时，突然有人“哇”的一声大叫，吸引了所有人的目光。
<br/>　　清长风也定睛看去。
<br/>　　是山山禅师！
<br/>　　山山禅师一只手捂着胸口，喷出了一口血，一只手指着对面的上古邪神，厉声喝道：“你是道风，你是道风！”
<br/>　　道风！
<br/>　　到底有几个道风？
<br/>　　“上古邪神”猛然起身，一伸手，“道风”脚下的番天印和打神鞭立刻飞到他手中，他人影一闪，手持打神鞭朝山山禅师砸了过去。
