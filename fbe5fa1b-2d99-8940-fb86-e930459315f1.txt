　　小炎帝？
<br/>　　听到这个名字，楚风眠猜想了一下，只怕是这小炎帝，也是像是小叶帝，小绝帝一般，圣龙秘境之中，得到过力量传承的一位天才了。
<br/>　　小叶帝，小绝帝都是圣院宗的天才，五灵宗也同样是身为圣龙秘境的四宗之一，自然也不会逊色太多。
<br/>　　显然是也同样是有着曾经得到过力量传承的天才。
<br/>　　不过在名气上，不如小叶帝，小绝帝二人。
<br/>　　得到过力量传承的天才，几乎是同境界无敌的存在，甚至是足以跨越境界战斗。
<br/>　　像是楚风眠之前跟小叶帝交手之时，小叶帝的境界也不过是准帝境界，但是却比起寻常八劫古帝还要厉害的多。
<br/>　　一旦是小叶帝踏入古帝境界，只怕是会跟楚风眠一样，足以是跟九劫古帝平起平坐，甚至是比起寻常古帝，还要强横。
<br/>　　“这小炎帝是什么境界？”
<br/>　　楚风眠不动声色的向着炎撼世询问道。
<br/>　　“小炎帝，是八劫古帝境界，而且是一步从准帝，踏入到七劫古帝，成就绝世古帝的天才。”
<br/>　　炎撼世看了楚风眠一眼，缓缓的开口道。
<br/>　　“小炎帝，只是一直在圣地之中修行，没有在外闯出威名来，论起资质来说，不亚于小叶帝，小绝帝。”
<br/>　　“八劫古帝……”
<br/>　　楚风眠目光闪烁，一步七劫古帝，绝世古帝，这样的资质，不亚于青梦。
<br/>　　而且小炎帝，更是得到过力量传承，有着曾经一尊半步天人境强者的武道领悟。
<br/>　　在同境界之中，更是所向无敌。
<br/>　　现在看来，楚风眠还是有些太小看了天下天才。
<br/>　　而且在此之前，楚风眠都没有听说过小炎帝这一号人物，可居然就是一尊绝世古帝的存在。
<br/>　　看来在各个势力之中，还有着许多从未出世过的天才，楚风眠都未曾听说过。
<br/>　　这一次九域剧变，将这些天才都吸引了过来。
<br/>　　八劫古帝的境界，还得到过力量传承，这小炎帝只怕是要比起九劫古帝，还要难以对付。
<br/>　　不过纵然是九劫古帝，楚风眠都曾经轰杀过，自然也不会惧怕了这小炎帝。
<br/>　　而且楚风眠跟五灵宗的关系不错，楚风眠只是想要打听一下这佛寺之中的情报而来，并无恶意。
<br/>　　炎撼世带着楚风眠，向着山谷之内走去。
<br/>　　在这山谷之中，密密麻麻布置出了无数的禁制，天罗地网一般，把这一片山谷，经营成了一座坚固无比的老巢。
<br/>　　其中的许多禁制，玄妙无比，纵然是九劫古帝都难以布置出来，其中散发这火系灵力的残留，看来便是出自那小炎帝的手笔。
<br/>　　楚风眠想要强行打破这些禁制，冲入其中，都需要耗费不少的手段。
<br/>　　这一片山谷之中的禁制联合起来，就算是碰到九劫古帝强攻，都足以抵挡一段的时间了。
<br/>　　楚风眠跟随在炎撼世的身后，一路步入到了其中，再穿过最后一道禁制之后，便是真正的来到了山谷的内部。
<br/>　　这山谷的内部，密密麻麻有着不少的山洞，看样子显然是被人以力量开辟而出的。
<br/>　　每一座山洞之中，都有着一道气息从存在，几乎都是古帝境界之上的武者。
<br/>　　无一例外，全部都是五灵宗的圣子，其中不乏有着一些五灵宗圣子的实力，已经是达到了八劫古帝，比起炎撼世都要强大几分。
<br/>　　在这一群人中，炎撼世的实力，也只是在中游左右。
<br/>　　“这就是五灵宗的底蕴。”
<br/>　　楚风眠扫视了一眼，心中便是感叹一句，圣龙秘境不愧是大罗天星域的四大势力第一，底蕴深厚。
<br/>　　这五灵宗之中的圣子，便是就有着如此之多，这些圣子之中，许多都有着机会可以冲击九劫古帝的。
<br/>　　不过最终能够冲击到九劫古帝境界的圣子，却是寥寥无几，毕竟想要踏入九劫古帝。
<br/>　　不止是需要逆天的资质，更是需要海量的财富，资源。
<br/>　　没有得到逆天的奇遇，想要踏入九级古帝，几乎是难上加难。
<br/>　　“炎撼世，我不是让你出去寻找情报吗？怎么回来了？还有你身边带回来的人是谁？”
<br/>　　从一座最大的山洞之中，突然间走出来了一道身影，此人一身红发，身上散发着恐怖的火灵力。
<br/>　　如同一尊真正的火神一般，一旦发怒，便是足以将一切敌人，焚烧殆尽。
<br/>　　“此人的实力，不亚于五行刀祖！”
<br/>　　楚风眠打量了一眼这红发男子，便是看出了此人的实力。
<br/>　　此人的境界，只是八劫古帝，但是一身实力气息，几乎是跟九劫古帝，没有丝毫的区别。
<br/>　　这样的人物，应该就是炎撼世口中的，五灵宗天才，小炎帝了。
<br/>　　跟随小炎帝出来的，还有几名五灵宗圣子。
<br/>　　能够成为圣龙秘境圣子的，几乎都是千万里挑一的绝世天才，一块大陆之中都难以诞生一位。
<br/>　　可这些五灵宗圣子，跟随在小炎帝背后，却是无比的恭敬，将这小炎帝当做是他们的首领。
<br/>　　小炎帝，也是五灵宗圣子之中的领袖，也怪不得此人可以统领无数桀骜不驯的天才。
<br/>　　这种实力，比起木麟还要强横的多，木麟的实力也就是八劫古帝巅峰。
<br/>　　而这小炎帝的实力，足足是可以媲美九劫古帝，差距了无数倍。
<br/>　　“小炎帝师兄，我刚想离开山谷，正巧是碰到了楚兄，所以带来给小炎帝师兄引荐一番。”
<br/>　　炎撼世听到小炎帝的质问，急忙的解释道。
<br/>　　“这位就是我说过的，曾经救过我们一命的楚兄，圣龙宗的圣子楚风眠。”
<br/>　　“圣龙宗，楚风眠？”
<br/>　　那小炎帝听到炎撼世的话，打量了楚风眠一眼，目光之中旋即露出了几分轻视。
<br/>　　“这就是圣龙宗，号称横空出世，不亚于云尊的天才，，我看倒是没什么神奇之处，也就是圣龙宗的人，故弄玄虚出来的小家伙罢了。”
<br/>　　这小炎帝的话中，讥讽之意毫不掩饰。
<br/>　　楚风眠听见都是眉头一皱，他也看的出来，这小炎帝只怕是无法无天之人，嚣张蛮横，看到楚风眠身上气息并不强横，所以不将楚风眠放在眼中。
